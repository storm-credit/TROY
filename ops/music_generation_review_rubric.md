# music generation review rubric

## 1. purpose

- 이 문서는 music generation 결과를 오케스트라가 어떤 기준으로 pass / revise / reject 할지 잠근다

## 2. core rubric

- truth:
  - 회차의 감정 기능이 맞는가
- voice:
  - 아린 authored voice가 살아 있는가
- restraint:
  - 과장된 해피엔드 / 과장된 비장미로 흐르지 않는가
- memorability:
  - 후렴이 남는가
- non-imitation:
  - 실존 유명곡 모사처럼 들리지 않는가

## 3. verdicts

- pass:
  - 그대로 다음 레이어로 이동 가능
- revise:
  - 핵심은 맞지만 prompt나 arrangement wording 조정 필요
- reject:
  - 곡의 역할 자체가 어긋남

## 4. warning signs

- 과한 belting
- finale glow
- ost underscore처럼만 들림
- 남성 시점 설명곡화
- mystery / miracle drift

## 5. director note

- 좋은 시도보다 `맞는 시도`가 우선이다
- bigger is not better
