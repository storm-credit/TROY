## Rationale (post-script, not pasted to Suno)

### First-line hook design
"신호가 바뀌기 전에 네가 한 발 먼저 섰어" — 도입부터 "두 사람의 박자 차이"가 한 줄에 가시화. 보편 도시 풍경(신호등) 안에 미세한 timing offset 만 있고, 누구도 누구를 끌지 않는다. bossa-folk 의 restrained groove 와 호흡이 맞는 단순 사건. 워밍업 0, 화자가 이미 상대를 보고 있는 상태에서 시작.

### 5 standout lines
1. "신호가 바뀌기 전에 네가 한 발 먼저 섰어" — first-line hook, 박자 어긋남이 오히려 동행의 증거가 되는 도입.
2. "어깨가 닿을 듯 닿지 않은 그 거리가 / 오늘은 무겁지 않은 게 이상했어" — Phase 2 register 의 핵: 거리가 거리로만 남는 가벼움. melodrama 회피.
3. "없는 사람이 있는 사람보다 또렷할 때가 있다더니 / 오늘은 옆에 있는 사람이 처음으로 또렷해" — v6 standout 라인의 보편 인용 + 뒤집기. waiting register 에서 walking-together register 로의 전환이 가사 안에서 일어남.
4. "발이 먼저 박자를 알아 버렸어 / 머리보다 먼저 한쪽으로 기울어" — chorus lock. 인지보다 몸이 먼저 알아챈 정서. bossa-folk 의 sway 와 호환.
5. "어떤 오후는 사건도 없이 기억에 남는다 / 무슨 말을 했는지보다 / 어떤 보폭으로 걸었는지가 남는다" — bridge 의 subversion thesis. 사건 없음을 사건으로 끌어올림.

### Bridge subversion
v6 의 bridge ("없는 시간이 더 정직할 때가 있어") 는 absence 안에서의 정직을 말했음. v8 bridge 는 정반대 — 사건도 대화도 없이 보폭만 남는 오후를 "기억의 정직"으로 뒤집음. 줄거리 tension 을 brief moment 의 layer 로 깎아내는 v8 스타일. "도착할 곳이 중요하지 않고 / 신호가 몇 번 바뀌었는지가 중요하지 않고 / 그냥 같이 걷는 동안이 다였어" 의 3-부정 + 1-긍정 구조로 lift 직전의 호흡 정리.

### Final-1-variant lift
chorus 마지막 행만 final chorus 에서 변형:
- chorus 1·2: "나도 모르게 조금 웃고 있었어"
- final chorus: "나는 조금 더 오래 웃고 있었어"
"모르게 → 의식 / 조금 → 조금 더 오래" 의 미세한 시간 확장이 lift. bossa-folk 의 restrained 안에서 melodrama 폭발 없이 "이 오후가 더 길게 남았다"는 종결. variant 1회만 사용 (룰 준수).

### Universal-imagery audit / Phase 2 / canon
작품 신 직접 묘사 0건 확인:
- v6 잔재(빈자리/창문/시계/책상/복도/화면) 전부 제거 — waiting register 와 walking register 는 양립 불가.
- 캠퍼스/강의실/카페/실내 무대 → 제거. 신호등/횡단보도/편의점 유리 = 도시 보편 시어, 특정 장소 아님.
- 공감각·청취 능력·sensory origin → 제거. "발이 먼저 박자를 알아" 는 metaphor 차원에서 listening 이 아닌 body-rhythm — `Self-Hearing Prohibition` motif 위반 없음.
- 운명/영원/없으면 못 살아/꽃/봄/비/우산/너의 향기 → 0건.
- "또렷해" 는 v6 의 핵심 어휘를 의도적으로 inversion 으로 흡수 (없는 사람 → 옆에 있는 사람).
- Phase 2 register 유지: early closeness 만 표현, declaration / longing climax 없음. 발음 위험 자모 0, 영어/숫자 0, 반말 자기독백 일관. 45-50 line 룰 안 (49 라인).

## Removed from v6
- "오후의 빛이 책상 위로 한 칸씩 물러나고 있었어" / "복도 끝에서 발소리" / "시계 초침" / "텅 빈 의자" / "보내지 못한 한 줄" — 전부 absence/waiting motif. v8 thesis 와 충돌하므로 전량 제거.
- "빈자리가 먼저 와서 / 네 이름 없는 시간을 채웠어" (v6 chorus lock) → "발이 먼저 박자를 알아 버렸어 / 머리보다 먼저 한쪽으로 기울어"로 교체. "먼저" 라는 부사만 motif 로 보존 (몸이 인지보다 먼저).
- v6 의 acoustic-ballad BPM 64 → bossa-folk BPM 84 로 라인 호흡 단축, 행 끝 "-었어" 유지하되 라인당 음절수 감소.

## Unanswered for Conductor
1. **chorus lock 음절 길이** — "머리보다 먼저 한쪽으로 기울어" 가 BPM 84 bossa-folk groove 에서 한 호흡 안에 들어가는지 Suno 청취 후 판정 필요. 길면 "한쪽으로 기울어" 만 후렴 마디 끝에 두고 앞을 짧게 ("머리보다 먼저 기울어") 로 회귀 가능.
2. **여성 intimate alto 적합성** — "어깨가 닿을 듯 닿지 않은 그 거리" 류 신체 거리 라인은 female intimate alto 에 잘 맞춤. male vocal 회귀 시 일부 라인 ("처음으로 또렷해" 의 감탄어조)이 약간 어색해질 수 있음 — female lock 유지 추천.
3. **bossa-folk 색조** — restrained groove 룰 안에서 final chorus 의 "조금 더 오래" lift 가 lounge 화 위험 있음. Style prompt 측에서 "no lounge swing, dry acoustic, restrained" 명시 필요.
4. **v6 standout 보편 인용 처리** — "없는 사람이 있는 사람보다 또렷할 때가 있다더니" 라는 인용형(-다더니)은 화자가 그 명제를 빌려와 뒤집는 구조. 너무 메타하게 들리면 "없는 사람이 또렷할 때도 있지만 / 오늘은 옆에 있는 네가 더 또렷해" 로 직설화 가능.
