# Manuscript Length Plan: Episodes 001-020

## 1. Purpose

이 문서는 원고 집필 시 회차별 호흡과 총량을 통제하기 위한 글자 수 기준표다.

핵심 원칙:
- 기본 회차는 `4,000-5,000자`
- 감정 전환점이나 핵심 사건 화는 `5,500-7,000자`
- 브리지나 여운 정리 화는 `3,200-4,000자`

## 2. Series-Level Length Policy

추천 총량:
- baseline average: `4,500자`
- core emotional episodes: `6,000자` 전후
- bridge episodes: `3,500자` 전후

예상 총량:
- compact run: 약 `48만 자`
- target run: 약 `54만 자`
- generous run: 약 `60만-65만 자`

총괄 기준:
- 본 프로젝트의 기본 목표 총량은 `약 54만 자`
- 지나치게 짧은 요약 회차를 줄이고, 중요한 회차만 선택적으로 길게 가져간다
- 노래와 MV가 함께 가는 프로젝트이므로, 소설이 모든 감정을 혼자 다 떠안지 않게 한다

## 3. Movement 1 Length Logic

Phase 1 / Chapter 1-4는 다음 리듬을 따른다.

- CH01 (E001-E005): 규칙, 고립, 예외의 첫 흔들림
- CH02 (E006-E010): 관찰에서 접속으로 이동
- CH03 (E011-E015): 재회, 수업, 과제, 공연으로 관계의 언어 형성
- CH04 (E016-E020): 몸의 거리와 감정의 거리 동시 축소

길이 규칙:
- 설명 회차보다 장면 회차를 우선 길게 쓴다
- 첫 만남, 첫 공연, 손이 닿는 순간, 뒤풀이 같은 회차는 감정 잔향을 위해 여유를 준다
- 초반부는 과도한 설정 설명으로 글자 수를 채우지 않는다

## 6. Control Rule

- 1차 초고는 목표치에서 `+-15%`까지 허용한다
- 2차 문장 보정에서 장면 밀도에 따라 `+-10%`로 다시 맞춘다
- 감정 전환점 회차는 글자 수를 줄이지 말고, 설명 회차에서 절감한다
- 글자 수가 부족할 때는 설정 설명을 늘리지 말고 장면의 촉감, 침묵, 동선을 보강한다
