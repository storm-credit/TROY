## Rationale (post-script, not pasted to Suno)

### First-line hook design
"잘 지내냐는 말에 잘 지낸다고 답했어" — 한국 청자가 즉시 자기 경험으로 환원할 수 있는 보편 거짓말. 시작부터 "감정 사건"이 이미 발생한 상태이고, 표면 평온과 내면 균열이 한 줄에 동시에 들어감 (검정치마 「강아지」 의 첫 줄 패턴 = 평범한 발화 안에 정서 균열). 워밍업 0.

### Chorus emotion line placement
Locked phrase "내 귀부터 닫고 살았어" 는 chorus 2행에 verbatim 2회 (chorus 1·2), final chorus 의 마지막 lift 라인은 자연 변형 "나는 내 귀부터 닫아 두었어" 로 종결 — 같은 동사 어간을 미세하게 과거완료적으로 닫아 음악적 안착(restraint 안에서의 lift). 변형 1회만 허용된 룰 안에서 사용.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스/강의실/식당/창가 → 제거 (보편 "방", "거울", "창문", "같은 자리"로 치환)
- 이어폰 / 음향 차단 도구 → 제거 (chorus 의 "귀부터 닫고" 추상 라인만 유지)
- 공감각·청취 능력·sensory origin → 제거 (motif_audit 두 건 모두 OK)
- 봄/계절 직접 선언 → "계절이 나만 빼고 지나가는 것 같아" 한 줄만, 보편 상실감 표현
- 클리셰 (비/새벽/우산/너의 향기/빈자리) → 0건
- borderline: "창문은 열려 있는데 바람은 안 들어와" — 「창문/바람」은 일반 시어지만 클리셰는 아님, 정서 작동 OK

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "괜찮다 / 괜찮지" 는 의도적 반복 (정서 anchor).
- "닿지 / 닫고 / 닫아" 디귿계열 의도적 반복 — 청취 시 motif 형성, 발음은 명확.
- "이불을 한 번 더 끌어안고 누웠어" — 한 호흡 안에 들어감 확인. "끌어안고" 부분이 약간 길지만 BPM 70-75에서 안정.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백).

### Removed from v6
- "귀를 막아도 들려와 / 누군가의 한숨 같은 봄" → 1행 1행이 시적이지만 첫 hook 폭발력 부족. "잘 지내냐는 말에 잘 지낸다고 답했어"로 교체 (즉발 hook).
- "강의실 맨 뒷자리 창가 / 유리에 비친 내 얼굴이 더 멀고" → 캠퍼스 신 직접 묘사. "거울 속 얼굴이 한 박자 늦게 웃어"로 교체 (보편).
- "이어폰 줄을 다시 감으면" → 작품 motif 직접 호출. "가까이 오는 건 다 무거운 것 같아"로 교체.
- "한 칸씩 뒤에 서 있어" / "사람들이 너무 많아" → 사용자 지시대로 "마음들이 너무 많아서"로 통합.
- "이어폰만 다시 끼웠다" (Bridge 마지막) → 도구 직접. Bridge 전체를 "다정인지 도망인지" 자기 의문으로 재작성.
- v6 Outro "들리지 않아 — 그래서 괜찮아" → final chorus 안에 흡수, 별도 outro 제거 (50 라인 안에 압축).

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 마지막 "나는 내 귀부터 닫아 두었어"** 가 음악적 lift로 작동하는지는 Suno 생성 후 청취 판단 필요. lift가 약하면 verbatim 3회 ("내 귀부터 닫고 살았어") 로 회귀 가능.
2. **여성 보컬 적합성** — verse 2 "거울 속 얼굴이 한 박자 늦게 웃어" 류 자기관찰 라인은 female intimate withdrawn 톤에 잘 맞춤. male vocal 회귀 시 일부 이미지 (이불, 거울)가 약간 페미닌하게 들릴 수 있음 — female lock 유지 추천.
3. **Bridge 길이 (6행)** — 8-12행 권고 하한이지만, 의도적으로 짧게 가져가서 자기 의문이 마무리 짓지 않은 채 final chorus 로 떨어지게 함. 더 길게 원하시면 2행 추가 가능 ("내가 무서운 건 사람이 아니라 / 내 안의 이 조용한 공기였을지도").
4. **Outro 제거 결정** — v6 의 "들리지 않아 — 그래서 괜찮아" outro 는 final chorus 안 "조금은 괜찮아질 줄 알았어"에 의도적으로 흡수 (괜찮음의 부정형 — 더 정직). outro 별도 행 필요하면 fade 라인 추가 가능.
