## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘은 인사를 미리 연습하지 않기로 했어" — 보편 청자가 즉시 자기 경험으로 환원할 수 있는 라인. 시작부터 이별 전 자리의 결정이 이미 내려진 상태이고, "연습하지 않기로 했어" 안에 그동안 연습해 왔다는 사실이 동시에 들어감 (인식 전환을 통한 hook). 워밍업 0, 1행이 thesis 압축.

### 5 standout lines (정렬)
1. "오늘은 인사를 미리 연습하지 않기로 했어" (verse 1, line 1) — first-line hook, 보편 인식 전환.
2. "큰 약속은 자주 무너졌고 / 작은 약속은 천천히 남았어서" (chorus head) — v6 standout 보존, chorus 진입 anchor.
3. "잘 지내라는 말은 너무 가볍고 / 행복하라는 말은 너무 무거워서" (verse 2) — 인사의 무게 양극에서 가운데를 찾는 보편 정서.
4. "얼굴을 똑바로 보는 일이 어려웠는데 / 오늘은 그게 가장 쉬운 일이었어" (verse 2) — 마지막 자리에서만 가능한 역전.
5. "끝이라는 말은 끝까지 안 했어 / 끝이라는 말을 하지 않아도 / 끝은 알아서 다가오고 있었으니까" (bridge) — 침묵의 완성 register, "끝" 4회 의도적 반복.

### Bridge subversion
v6 bridge는 "이건 결말이 아니야 / 시작도 아니야 / 중간에 들어가는 한 줄이야" — confession 프레임의 *연속성 주장*. v8 bridge는 정반대로 *끝의 정중한 인정*: "끝이라는 말은 끝까지 안 했어 / ... / 끝의 입구에 잠깐 같이 앉아 있기로 했어". Phase 6 의 "침묵의 완성" 룰을 직접 음악에 끌어들임. "끝" 단어를 절대 회피하지 않으면서도 입에 올리지는 않는 자기모순을 6행 안에 넣어 final chorus 로 떨어뜨림.

### Final chorus 1-variant lift
chorus 1·2 는 verbatim 유지. final chorus 의 마지막 두 라인만 변형:
- chorus 1·2: "작은 말 하나만 두고 가기로 했어 / ... / 가장 늦게 꺼내는 인사야" (현재형 결심)
- final chorus: "작은 말 하나만 두고 갈게 / ... / 가장 늦게 꺼낸 인사였어" (반말 약속 → 과거 회상으로의 이행)

같은 동사 어간 ("두고 가-", "꺼내-")을 미세하게 시제 이동시켜 *결심이 이미 일어난 일이 되어 닫히는* 음악적 lift. 변형 1회 룰 안에서 사용. 시제만 바뀌고 의미·운율 코어는 유지.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 카페 달 / 카페 조명 → 제거 (보편 "같은 자리", "같은 컵", "시계", "얼굴"로 치환)
- 공감각·청취 능력·sensory origin → 0건 (motif_audit_seojun_sensory_origin / heard_vs_said / silence_deep_sea / arin_song_thread / contact_equals_danger / arin_exception 모두 OK)
- 윤서준/지아린 이름·고유명 → 0건. POV 는 1인칭 자기독백, 보편 화자.
- 클리셰 (운명/완성된 사랑/마지막 키스/빈 의자/우산/비/창가) → 0건
- 계절 직접 선언 ("늦가을") → 제거. 대신 "시계는 평소보다 느리게 갔는데" 같은 시간 감각으로 phase 6 의 정적을 우회 표현.
- borderline: "컵을 두 손으로 감쌌어" — v6 의 "두 손으로 감싼 컵" motif 를 *완전히 같은 동작*으로 가져왔지만, 카페·조명 맥락 제거로 **보편 동작**으로 전환됨. motif_audit 통과.
- forbidden language ("운명", "완성", "결말", "victory", "perfect timing", "closure") → 0건. "끝" 은 사용 (bridge 4회) — 그러나 brief 의 "closure glow"가 아니라 *입에 올리지 않는 끝의 인정* 으로 정반대 방향 사용.

### Korean pronunciation pass (BPM 70)
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- 디귿계열 의도 반복: "두고 / 두 손 / 들어 / 늦은" — 청취 시 motif 형성, 발음 명확.
- "큰 약속 / 작은 약속" 비대칭 대구 — 90s ballad 의 호흡 단위에 정확히 맞음 (한 호흡 1행).
- "끝이라는 / 끝까지 / 끝은 / 끝의" ㄲ-ㅌ 클러스터 4회 bridge 집중 — 의도된 motif. BPM 70 에서 ㅌ 받침 명료.
- "조금만 더 있어도 / 조금만 더 있으면" pre-chorus 대구 — 동일 5음절 호흡, sing-along 안정.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백, female intimate alto).
- 한 호흡 안 들어가는 가장 긴 라인 ("같은 자리에 앉아서 같은 컵을 두 손으로 감쌌어") — BPM 70 에서 안정. measured ballad pace 의 sweet spot.
- "행복하라는 / 잘 지내라는" Sino+Native 혼용 위험 → 둘 다 일상 한국어, 클리셰 회피.

### Removed from v6 (frame shift: confession → final-greeting)
v6 의 thesis 였던 "고백은 알아내는 것이 아니라 책임지는 것이다" 는 v8 에서 폐기. 이유: E112 의 episode genre assignment 에서 정의된 emotionalAnchor 는 "마지막 인사 직전의 가장 고요한 자리" 이며, 이는 v6 song_brief 의 "confession of responsibility" 프레임과 **다른 좌표**. v8 은 genre assignment 의 좌표를 thesis 로 채택함.

- v6 chorus lock "모른 채로도 책임지겠다는 말이야" → v8 chorus head "큰 약속은 자주 무너졌고 / 작은 약속은 천천히 남았어서" (v6 의 standout 라인을 chorus 진입으로 승격)
- v6 verse 1 의 "다 알아내고 하는 고백" 시퀀스 → v8 의 "오늘은 인사를 미리 연습하지 않기로 했어" (인사 좌표로 이동)
- v6 의 "사랑이라는 말 / 책임이라는 말" 메타-언어 분석 → v8 의 "잘 지내라는 말 / 행복하라는 말" 로 보편화. 같은 *언어 자체를 다루는 자기관찰* 구조 유지하되 confession motif 제거.
- v6 bridge 의 "결말이 아니야 / 시작도 아니야" → v8 bridge 의 "끝이라는 말은 끝까지 안 했어" 로 정반대. v6 는 결말 부정, v8 은 결말 정중 인정.
- v6 outro 없음 → v8 도 outro 없음 (final chorus 안에 흡수).

## Unanswered for Conductor

1. **Frame shift 승인 필요** — v8 은 v6 song_brief 의 "고백/책임" 좌표가 아니라 120_episode_genre_assignment.json 의 "마지막 인사 직전의 가장 고요한 자리" 좌표를 thesis 로 채택했음. 두 소스가 *같은 에피소드*에 대해 다른 정서 좌표를 지정하고 있음 — Conductor 가 어느 쪽을 canon 으로 잠글지 결정 필요. v8 은 후자를 택했고, song_brief 의 confession motif 는 의도적으로 폐기됨.

2. **변형 라인 1회 사용 — final chorus 마지막 두 라인 시제 이동** ("두고 가기로 했어 → 두고 갈게", "꺼내는 인사야 → 꺼낸 인사였어") 가 음악적 lift 로 작동하는지는 Suno 생성 후 청취 판단 필요. lift 가 약하면 verbatim 3회 회귀 가능. (단 Phase 6 "침묵의 완성" 좌표상 *과거형으로 닫는 것*이 emotionally 더 정확하다고 판단.)

3. **여성 보컬 적합성** — "얼굴을 똑바로 보는 일", "잘 지내라는 말 / 행복하라는 말" 류의 *마지막 자리에서의 정중함* 이 female intimate alto 에 잘 맞춤. Phase 6 register 와 lock.

4. **Bridge 길이 (6행)** — "끝" 4회 반복이 의도. 더 늘리면 motif 가 산만해짐. 더 줄이면 bridge subversion 의 정중함이 약해짐. 현행 6행이 90s-vintage-ballad bridge 의 표준 호흡.

5. **총 라인 수** — 47행 (45-50 범위 안). chorus 3회 lock 유지, bridge 6행, verse 8행씩, pre-chorus 4행.
