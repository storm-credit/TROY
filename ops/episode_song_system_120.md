# Episode Song System: 120 Episodes

## 1. Core Rule

이 작품은 `1화 = 1곡` 체계를 공식 채택한다.

- total songs: 120
- one episode, one theme song
- one song, one emotional function

## 2. Why This Rule

- 회차별 정체성이 선명해진다
- 소설, Suno, MV가 같은 episode id로 묶인다
- 장기 연재에서 감정 아카이브가 쌓인다
- 독자는 이야기를 `읽는 경험`과 `듣는 기억`으로 동시에 기억하게 된다

## 3. Song Function Types

각 화의 곡은 아래 네 가지 중 하나로만 분류한다.

### A. Seojun Inner Monologue

- 서준의 내면을 압축
- 가장 많이 쓰이는 기본형
- 독백과 잔향 중심

### B. Relationship Theme

- 두 사람 사이의 감정 변화와 거리
- 고백, 데이트, 갈등, 이별, 재회 같은 전환점에 적합

### C. Arin In-World Song

- 아린이 실제로 만들거나 부르는 곡
- 너무 자주 쓰지 않는다
- 중요한 복선/회수 포인트에서만 사용

### D. Ending Reflection

- 화 끝의 감정을 정리하는 회고형
- 후반부, 특히 악장 말미에 자주 배치

## 4. Allocation Rule

권장 비율:

- A. inner monologue: 45-55%
- B. relationship theme: 25-35%
- C. in-world song: 10-15%
- D. ending reflection: 10-15%

주의:
- 모든 화를 러브송으로 만들지 않는다
- 모든 화를 서준 독백으로만 밀지 않는다
- 아린 창작곡은 희소해야 힘이 생긴다

## 5. Per-Episode Template

각 화의 곡 문서는 아래 항목을 가진다.

- episode id
- episode title
- movement / phase
- song function type
- emotional thesis
- key image
- lyrical taboo
- chorus emotion line
- sound direction
- MV carry-over note

## 6. Naming Rule

- 파일명은 episode id 우선
- 제목은 바뀔 수 있어도 id는 고정

example:
- `E001_theme_song.md`
- `E054_theme_song.md`
- `E120_theme_song.md`

## 7. Final Arc Rule

마지막 악장에서는 곡의 역할이 점점 바뀐다.

- 111-115: 선택과 이해의 축적
- 116-119: 떠남과 승인
- 120: 최종곡은 재결합이 아니라 사랑의 승인

## 8. Output Principle

- 본편이 원본이다
- 곡은 본편의 감정을 압축한다
- MV는 곡과 본편 사이의 시각적 번역이다
- 셋 모두 같은 episode id를 공유한다
