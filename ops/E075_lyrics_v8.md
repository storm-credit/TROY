## Rationale (post-script, not pasted to Suno)

### First-line hook design
"손끝이 멈춘 자리에 한 박자가 떨어졌어" — 즉발 hook 으로 워밍업 0. "손끝/멈춘/한 박자" 세 키워드가 thesis (subtle edge between try-once-more-and-stop) 를 한 줄에 압축. 손가락이 보낼지 말지 망설이는 보편 경험 (메시지·이메일·전화 — 어느 것에도 환원 가능) 을 청자에게 즉시 떠올리게 함. dream-pop BPM 71 의 첫 박자가 "한 박자가 떨어졌어" 라는 가사 자체와 일치하는 메타 hook.

### 5 standout lines
v8 craft 요건의 5 standout 라인 의도 배치:
1. "손끝이 멈춘 자리에 한 박자가 떨어졌어" (verse 1, line 1) — 즉발 hook.
2. "지우면 가벼울까 봐 더 가벼울까 봐 무서웠어" (verse 1, line 4) — "가벼움" 의 양면성 (해방과 상실) 을 한 줄에 압축, 의도적 반복 ("가벼울까 봐 더 가벼울까 봐") 으로 망설임의 반복 구조 형성.
3. "한 발은 앞에 두고 한 발은 뒤에 둔 채로" (pre-chorus) — 미세 경계의 신체화. thesis 그 자체.
4. "멈춘 빛이 가장 따뜻한 빛인 적도 있었어" (verse 2) — Phase 4 register 의 통증 안에서 미세한 위로. dream-pop 의 sonic palette 와 직접 일치 (멈춘 빛 = pitch-blurred memory).
5. "망설이는 일도 결국 결정이라는 걸 늦게 알았어" (verse 2 마지막) — 자기 인식 painful but incomplete. 해결 아닌 인식.

### Chorus emotion line placement
Locked phrase "한 번 더 해 볼까 / 그만 둘까 사이에서" 는 chorus 1·2행에 verbatim 3회 (chorus 1, chorus 2, final chorus). 마지막 final chorus 의 6행만 "나는 그 사이에 오래 머물렀어" → "나는 그 사이에 조금 더 머물기로 했어" 로 변형 1회 — 과거형이 현재 결정형으로 미세 이동. 해결이 아닌 "선택의 유보 자체를 선택" 하는 Phase 4 incomplete recognition. v8 final 1-variant lift 룰 안에서 사용.

### Bridge subversion
v8 craft 요건의 bridge subversion: verse·chorus 가 모두 "내가 결정해야 한다" 는 1인칭 능동 frame 이었던 것을, bridge 에서 처음으로 "답이 오기를 기다리고 있었던 건지도 몰라 / 누가 대신 정해 주면 따라가려고" 로 frame 을 뒤집음. 망설임의 정체가 결정 회피가 아니라 외부 결정 위탁이었다는 자기 폭로. 그러나 "오늘은 아무도 오지 않았고 / 나만 그 자리에 남아 있었어" 로 회피의 실패까지만 보여 주고 closure 거부 — Phase 4 painful but incomplete 가드 준수.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스/강의실/식당/창가 → 0건
- 능력 (공감각·청취) / sensory origin → 0건
- 메시지·앱·알림 직접 명사 → 제거 ("한 글자", "글자" 의 보편 시어로 추상화)
- 아린 / 서준 / 인물 호명 → 0건
- 클리셰 (비/새벽/우산/너의 향기/빈자리/별/달) → 0건
- 영어/숫자/외래어 → 0건

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "한 번 더 / 그만 둘까" hook 은 평탄한 모음·자음 분포 (`ㅎ ㄴ ㅂ ㄴ ㄷ ㄱ ㅁ ㄴ ㄷ ㅋ`) — dream-pop close vocal 에서 발음 안정.
- "가벼울까 봐 더 가벼울까 봐" — 의도적 반복, "ㄱㅂ" 군집 있으나 BPM 71 에서 한 호흡 안에 안정.
- "망설였어 / 망설이는" 디귿계열 motif 의도 반복 — 청취 시 motif 형성, 발음 명확.
- "조금만 더 / 조금만 멀어지면" 대구 — pre-chorus 의 양가성 강조, 발음 평이.
- 영어/숫자/외래어 0건. jamo emoticon 0건. stage direction 0건.
- 한 호흡 라인 점검: "보낼 듯 말 듯 한 글자를 오래 쥐고 있었어" — 호흡 안정 (3+3+5+3 음절 분할).
- "한쪽 손이 모자란 사람처럼 나는 자꾸 망설였어" — 길지만 BPM 71 에서 안정, 의도적 비유 강도 유지.

- v6 의 "지나친 물건 / 화면 가장자리 / 늦게 알아챈" 시각 motif → 제거 (E075 새 brief 가 "missed detail" frame 이 아닌 "edge-between-try-and-stop" frame).

- v6 의 7행 chorus → 6행으로 압축, lock 라인 (1-2행) 후 4행 supporting 으로 균형.
- v6 outro 없음 → v8 도 별도 outro 없음. final chorus 의 "조금 더 머물기로 했어" 1-variant lift 가 outro 역할 흡수 (50 라인 안 압축).
