# Change Control

## 1. Purpose

이 문서는 프로젝트 진행 중 생기는 중간 아이디어가 전체 캐논과 제작 흐름을 흔들지 않도록 관리하기 위한 규칙이다.

핵심 원칙:
- 아이디어는 환영한다
- 하지만 모든 아이디어가 즉시 본체를 바꾸지는 않는다
- 총괄 오케스트라가 `잠금 / 보류 / 실험`으로 분류한다

## 2. Three States

### Locked

이미 잠긴 결정이다.

예:
- 120화
- 아름다운 이별 결말
- 청취 소실 -> 침묵의 완성
- 1화 = 1곡
- 6악장 구조

규칙:
- 사용자 아이디어가 떠올라도 즉시 변경하지 않는다
- 총괄이 전체 연쇄 영향 검토 후에만 바꾼다

### Parked

좋지만 지금 당장 본체를 바꾸면 손해가 큰 아이디어다.

예:
- 특정 서브캐릭터 확장
- 외전 아이디어
- 삽화 스타일 변형안
- 추가 세계관 장치

규칙:
- 문서에 저장은 한다
- 현재 집필과 핵심 구조는 건드리지 않는다

### Experimental

실험 가치가 있고, 작은 범위에서 테스트 가능한 아이디어다.

예:
- 1화 MV 컷 스타일
- Suno 보컬 거리감 실험
- 문체 비유 변주

규칙:
- 본체를 바꾸지 않고 샘플로만 시험한다
- 결과가 좋으면 다음 라운드에서 채택 여부를 판단한다

## 3. Decision Rule

새 아이디어가 들어오면 총괄은 아래 순서로 판단한다.

1. 이 아이디어가 결말에 영향을 주는가
2. 120화 구조를 다시 흔드는가
3. 복선 회수 계획을 깨는가
4. 음악/MV/소설의 공통 문법을 어지럽히는가
5. 지금 바꾸는 이득이 나중 수정 비용보다 큰가

하나라도 위험도가 높으면 즉시 `Parked`로 보낸다.

## 4. What The User Can Expect

- 사용자의 새로운 생각은 버리지 않는다
- 다만 총괄이 바로 `본체 반영`하지 않고 먼저 등급을 매긴다
- 이미 잠근 사항은 쉽게 흔들지 않는다
- 대신 좋은 아이디어는 `보류함`이나 `실험안`으로 반드시 남긴다

## 5. Current Locked Set

- format: 120 episodes
- structure: 24 chapters x 5 episodes
- presentation: 6 movements
- ending: beautiful separation
- ending language: listening loss -> completion of silence
- media rule: one episode, one song

## 6. Current Producer Rule

총괄 오케스트라는 다음 우선순위로 움직인다.

1. 잠긴 결정 보호
2. 현재 작업 끝까지 밀기
3. 새 아이디어는 저장 후 분류
4. 실험은 샘플 단위로만 진행
5. 충분히 쌓인 뒤 한 번에 체계적으로 반영
