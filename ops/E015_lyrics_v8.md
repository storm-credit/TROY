## Rationale (post-script, not pasted to Suno)

### First-line hook design
"세는 걸 그만둔 날부터 자꾸 셈해지는 게 있어" — 한국 청자가 즉시 자기 경험으로 환원할 수 있는 보편 자가-모순(셈을 멈추니 더 셈해지는). 시작부터 "축적의 사건"이 이미 발생한 상태이고, 인정 회피와 알아챔이 한 줄에 동시에 들어감 (E001의 "잘 지내냐" 패턴 = 평범한 발화 안에 정서 균열). 워밍업 0. Ambient-piano BPM 61 의 가장 늘어진 호흡에 한 호흡으로 안착.

### Chorus emotion line placement
Locked phrase "하루의 빈칸에 네가 조용히 들어왔어" 는 chorus 1행에 verbatim 2회 (chorus 1·2), final chorus 의 마지막 lift 라인은 자연 변형 "하루의 빈칸에 네가 천천히 머물러 있어" 로 종결 — 같은 주어구를 미세하게 현재완료적으로 닫아 음악적 안착(restraint 안에서의 lift, "들어왔어"의 단발 사건 → "머물러 있어"의 지속 상태). 변형 1회만 허용된 룰 안에서 사용. Phase 1 closer 가 아니므로 폭발적 lift 회피, 머무름의 감각만.

### 5 standout lines audit
1. "세는 걸 그만둔 날부터 자꾸 셈해지는 게 있어" — first-line hook, self-paradox.
2. "지운 자리가 더 선명해져서 손을 멈췄어" — verse 1 closing, 지움이 강조가 되는 이미지의 전환.
3. "오늘은 너를 한 번도 마주치지 않았는데 / 오늘이 너로 가득했다는 게 이상해" — verse 2 의 부재-편재 페어 (한국 발라드에서 strong standout).
4. "이렇게 조용히 한 칸씩 채워지는 일인 줄 몰라서" — bridge 정의 갱신 라인, 가까워짐의 재정의.
5. "내일도 같은 자리가 따뜻할 것 같아" — final chorus 안의 미래 예감, 확정하지 않은 채 남기는 안전감.

### Bridge subversion
v6 의 bridge ("이게 가까워지는 거라면 / 무섭지 않을 것 같아")는 가정형 / 조건형. v8 은 인지의 전환으로 subvert — "가까워진다는 게 뭔지 사실 잘 몰랐어" 로 시작해 화자가 자신의 정의를 갱신한다. "한 번에 닿는 거" → "한 칸씩 채워지는 일" 로 가까워짐의 문법 자체가 다시 쓰이는 순간. "무서워야 할 것 같은데 무섭지가 않아" 는 v6 의 가정형을 직설로 바꾼 것 — 두려움의 부재가 두려움보다 먼저 도착함. Phase 1 의 마지막 닫힘이 아닌 알아챔의 정직.

### Final chorus 1-variant lift
변형 라인 1회 허용 사용 — final chorus 의 마지막 라인을 "하루의 빈칸에 네가 천천히 머물러 있어" 로 처리. 이유:
- 동사 시제 변형 ("들어왔어" = 완료적 단발 → "머물러 있어" = 진행적 지속) 으로 사건이 상태로 굳어진 시간을 표시.
- "조용히" → "천천히" 부사 미세 교체 — 침묵에서 시간성으로의 전환.
- restraint 안에서의 lift, ambient-piano 61 BPM 에 적합한 비-폭발적 종결.
- 이 line 이 약하다고 판단되면 verbatim 3회 ("하루의 빈칸에 네가 조용히 들어왔어")로 회귀 가능.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 복도/강의실/캠퍼스/벤치/카페/타임테이블/종이컵 → 제거 (브리프 도메인 image family 도 보편화: "하루/자리/길/한 칸").
- 노트북/가방끈 → 제거.
- 공감각·청취 능력·sensory origin → 제거 (motif_audit 둘 다 OK).
- 아린 exception 직접 호출 → 제거 (motif_audit 도 destiny language 회피 권고).
- 봄/계절 직접 선언 → 0건 (Phase 1 의 봄빛은 음악/사운드 layer 가 담당).
- 클리셰 (운명/완성된/완벽한 연결/소울메이트) → brief 의 forbidden 단어 0건.
- borderline: "한쪽이 따뜻해 / 따뜻할 것 같아" — 일반 시어, 클리셰 아님, 정서 작동 OK.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "세는 / 셈해지는 / 세 번째" 시읏 계열 의도적 반복 — motif 형성, 발음 명확.
- "채워져 / 채워지는 / 채워졌어" 차은 계열 의도적 반복 — chorus payoff 강화.
- "지웠어 / 지운 자리" 한 호흡 안에 들어감 확인. BPM 61 의 늘어진 호흡에 안정.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백).
- "하루의 빈칸" 의 비읍-키읔 인접 발음 클리어, chorus 첫 음 안착 OK.

### Removed from v6
- v6 verse 1 "큰 사건이 있었던 건 아니야" → pre-chorus 로 흡수 ("큰 사건은 없는데 하루가 자꾸 무거워져"). first hook 자리에는 "세는 걸 그만둔 날부터…" 의 자가-모순 라인 배치.
- v6 verse 1 "한 자리만 어딘가 채워져 있었어 / 누가 채운 건지 잘 모르겠어" → "어제와 똑같이 걸었는데 한 자리만 따뜻해 / 누가 거기 있었는지 묻기엔 너무 조용해서" 로 재작성, "조용함" 의 침묵 텍스처 강화.
- v6 pre-chorus "같은 길을 걸어도 한 자리가 달라" → "큰 사건은 없는데 / 큰 변화는 없는데" 의 부정형 페어로 압축, chorus 진입 전 정서 가속 강화.
- v6 verse 2 "인정하지 않은 채로 자리만 비워 두었는데" → "오늘은 너를 한 번도 마주치지 않았는데 / 오늘이 너로 가득했다는 게 이상해" 로 교체 — 부재-편재 페어 (5 standout).
- v6 bridge → 인지 전환형으로 subvert (위 §Bridge subversion).
- v6 final chorus "내일도 비슷하게 흘러갈 것 같아" → "내일도 같은 자리가 따뜻할 것 같아" 로 — "비슷함" 의 추상에서 "따뜻함" 의 감각으로 lift.
- 행수 22 → 47 라인 (45-50 권고 안). v8 craft 길이 락 충족.

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 마지막 "하루의 빈칸에 네가 천천히 머물러 있어"** 가 음악적 lift로 작동하는지는 Suno 생성 후 청취 판단 필요. lift가 약하면 verbatim 3회로 회귀 가능.
2. **여성 보컬 적합성** — verse 2 "오늘이 너로 가득했다는 게 이상해" 류 자기관찰 라인은 female intimate alto 톤에 잘 맞춤. female lock 유지 추천.
3. **Bridge 길이 (6행)** — 8-12행 권고 하한이지만, 의도적으로 짧게 가져가서 인지 전환이 마무리 짓지 않은 채 final chorus 로 떨어지게 함. 더 길게 원하시면 2행 추가 가능 ("이미 와 있는 걸 알아 버린 다음엔 / 아닌 척하는 게 더 어려워졌어").
4. **Outro 제거 결정** — final chorus 의 마지막 변형 라인이 outro 역할을 흡수. 별도 outro 행 필요하면 fade 라인 추가 가능 ("하루의 빈칸에 — 네가 조용히").
5. **Phase 1 closure vs anticipation** — E015 는 Phase 1 closer (E020) 가 아니므로 의도적으로 닫지 않음. 미래 예감 ("내일도 같은 자리가 따뜻할 것 같아") 만 남겨 Phase 2 진입의 정서적 토대 확보.
