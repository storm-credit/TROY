## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘은 문 하나가 작은 소리로 열렸어" — v6 standout 라인을 v8 에서도 유지. 이 라인은 이미 Phase 3 opener 정서의 thesis 역할을 완벽히 수행하고 있고 (첫 사건 발생 + 조용한 임계 통과 + "오늘은" 일상 anchor), 검증된 hook 을 굳이 교체할 이유가 없음. v8 craft 룰에서 first-line hook 은 "즉발 정서 사건"을 요구하는데, 이 라인은 "열렸어" 과거형으로 이미 사건이 발생한 상태에서 시작 — 워밍업 0. Phase 3 의 "containment within warmth" 톤 (anthemic 금지)과도 정합.

### Chorus emotion line placement
canon anchor "오늘은 문 하나가 작은 소리로 열렸어" 는 chorus 1·2 verbatim 2회 (chorus lock ≥2x 충족), final chorus 의 마지막 lift 라인은 자연 변형 "오늘은 문 하나가 작은 소리로 열려 있어" 로 종결 — 과거형 "열렸어" 를 현재진행형 "열려 있어" 로 미세 변형해, 사건 (시작) 이 상태 (지속) 로 안착하는 음악적 lift 를 만듦. 변형 1회만 허용된 룰 안에서 사용. Phase 3 의 "warm but contained" 락 안에서 anthemic build 없이 자연 정착.

### Five standout lines (v8 craft target)
1. "오늘은 문 하나가 작은 소리로 열렸어" — first-line hook, 임계 통과의 정서.
2. "환한 게 들어오면 그늘도 같이 들어와서 / 나는 두 가지를 같이 데리고 들어가기로 했어" — Phase 3 의 빛/그늘 thesis 락. 두 줄 한 호흡.
3. "잃을 수 있는 자리에 마음을 두는 일이 / 사랑이라는 걸 처음 인정해 보았어" — 보편 "사랑" 정의 라인. Phase 3 dailiness-overlap 의 핵심 정서.
4. "의자 두 개를 나란히 두는 일에 가까워" — bridge subversion. "박수 같은 소리" (anthemic 시작) 를 "의자 두 개" (가구 단위 일상) 로 다운스케일. v8 craft 의 bridge subversion 룰 정확 충족.
5. "오늘 네가 마신 자리에 내일 내가 앉아도 / 어색하지 않은 그런 하루가 되었으면" — bridge 닫는 행. "겹쳐진 시간표" (E036 image seed) 를 "같은 자리, 다른 날"로 시간차 표현. 문이 열린 후의 "함께 사는 시간 단위"가 처음 등장.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스/강의실/카페/공식 발표 신 → 제거 (보편 "문/공기/자리/식탁/의자/컵"으로 치환)
- "공식 관계 시작", "친구들의 인지", "손 가까이" 등 harness scene rhythm → 직접 묘사 0건
- 공감각·청취 능력·sensory origin → 0건 (motif_audit `Seojun Sensory Origin` watch 항목 보호)
- 여름/계절 직접 선언 → 제거 (v6 의 "여름빛과 그림자" 도 v8 에선 "환한 게 / 그늘"로 보편화. song_brief 의 "여름" 은 sonic palette 에만 반영, 가사에는 미투입 — 보편 청자 환원성 우선)
- 클리셰 (영원/운명/완벽/해피엔딩) → 0건 (song_brief words to avoid 전부 회피)
- borderline: "이름을 붙인 자리" — 「관계 명명」을 직접 부르지 않고 추상화. 작품 캐논 "공식 관계" 와 보편 "이름 붙은 마음" 이 동시 작동.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "열렸어 / 열려 있어" ㄹ-ㅇ 흐름, BPM 70 acoustic-ballad 안정.
- "데리고 / 두기로 / 두었어" 디귿계열 의도적 반복 — 청취 motif 형성.
- "의자 두 개를 나란히 두는 일에 가까워" — 한 호흡 안에 들어감. "나란히" 비음 안정.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백). 여성 alto intimate 톤에 정합.
- 라인 카운트: 47행 (45-50 라인 룰 충족). chorus 5행 × 3 = 15, verse 8행 × 2 = 16, pre-chorus 4행, bridge 6행, final chorus 6행.

### Bridge subversion (v8 craft target)
v6 bridge ("어떤 시작은 박수로 시작되지 않아 / 조용히 자리를 옮겨 앉는 일에 가까워") 의 thesis 를 유지하되, v8 에선 한 단계 더 구체적으로 다운스케일: "박수 같은 소리" 를 부정한 자리에 "의자 두 개를 나란히 두는 일" 이라는 가구 단위 이미지를 박아 넣음. 이어 "같은 식탁 / 같은 컵 / 네가 마신 자리에 내일 내가 앉아도" 로 "겹쳐진 일상의 시간차" 를 처음 도입 — Phase 3 OPENER 의 정확한 정서 ("두 사람의 일상이 처음 겹쳐지는 시작"). bridge 가 chorus 의 "문" 이미지를 안 쓰고 새로운 "의자/식탁/컵" 가구 패밀리로 갈아탔다가, final chorus 에서 다시 "문" 으로 돌아오게 해 subversion → return 구조 완성. anthemic 금지 락 유지.

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 첫 행 "열려 있어"** 가 음악적 lift 로 작동하는지는 Suno 생성 후 청취 판단. 약하면 verbatim 3회 ("오늘은 문 하나가 작은 소리로 열렸어") 로 회귀 가능.
2. **여름 키워드 미투입 결정** — song_brief 는 "early summer threshold" 를 명시하지만, v8 보편 라인 룰을 위해 가사에선 "환한 게" / "그늘" 추상으로만 처리. 여름 명시 원하면 verse 1 마지막 행을 "환한 게 들어오면 그늘도 같이 들어와서" → "여름이 들어오면 그늘도 같이 들어와서" 로 1단어 교체 가능 (보편성 약간 감소).
3. **Phase 3 OPENER vs 중반 차별화** — E036 은 Phase 3 시작점이라 "겹침" 이 처음 발생하는 정서가 핵심. v8 은 bridge 에 "같은 식탁 / 같은 컵" 의 시간차 겹침 이미지를 deliberately 넣어 phase 3 후속 곡들 (E037-E044) 과의 anchor 구분. 후속곡이 더 깊은 겹침을 다룰 때 v8 의 "처음 인정해 보았어" 라인이 시간 anchor 로 작동.
4. **Bridge 6행 유지** — 8-12행 권고 하한이지만, "anthemic build 금지" Phase 3 락 보호 위해 의도적으로 짧게 유지. 더 길게 원하시면 "의자 사이에 흐르는 짧은 침묵도 / 같은 침묵이 되어가는 게 좋아" 2행 추가 가능.
5. **chorus 5행 구조 (4행 표준 대비 +1)** — v6 에서 검증된 5행 구조 유지 ("처음이라는 말이 무겁지 않았어" 닫는 행이 정서 anchor). 4행으로 압축하면 phase 3 의 "처음" 무게를 제대로 못 실음.
