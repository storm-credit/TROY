## Rationale (post-script, not pasted to Suno)

### First-line hook design
"같은 자리에 앉았는데 다른 박자로 숨을 쉬었어" — Phase 4 opener thesis ("같은 자리에 있는데 서로 다르게 들리는 첫 어긋남") 를 첫 줄에 즉발 압축. "같은 자리"는 공간 동일성, "다른 박자로 숨"은 보편 신체 어긋남 (호흡 = 가장 기본적인 청취 가능 신호). 캠퍼스/복도 직접 묘사 0, 능력/공감각 0. v6의 "같은 길에 서 있었는데 한 걸음 뒤로 물러섰어" 대비 — "물러섬" 동작어를 뺐고, "박자 어긋남"이라는 음악적/정서적 thesis 가 더 직접 들림. 워밍업 0.

### Chorus emotion line placement
Locked phrase "멀어진 건 우리가 아니라 / 먼저 부를 수 있던 목소리였어" 는 chorus 1·2 verbatim 2회 (chorus lock ≥2x 충족), final chorus 마지막 라인에 자연 변형 1회 ("멀어진 건 우리가 아니라 — 내 목소리였어") 로 lift. 변형은 "우리"의 외부 시선에서 "내" 1인칭 자각으로 좁히는 단축형 — restraint 안에서의 final 1-variant lift. 변형 1회 룰 준수.

### Five standout lines (v8 craft target)
1. "같은 자리에 앉았는데 다른 박자로 숨을 쉬었어" — opener hook, thesis verbatim.
2. "익숙해지면 안 되는 종류의 익숙함이었어" — 자기 모니터링 라인, 도덕적 자각.
3. "부를 수 있던 목소리가 먼저 사라졌어" — pre-chorus 종결, chorus emotion line 의 사전 echo (목소리 = canon anchor 의 키 명사).
4. "같은 공기를 마시면서도 / 다른 계절을 사는 사람처럼 앉아 있었어" — verse 2 종결, Phase 4 의 "유리 너머의 너" 보편화 (계절 어휘는 phase preset 안에서 작동).
5. "멀어진 건 우리가 아니라 — 내 목소리였어" — final lift, 외부귀속을 자기귀속으로 회수.

### Bridge subversion
v6 bridge ("물러나는 건 도망이 아니야 / 물러나는 건 사랑이 아니야") 는 두 부정문으로 닫혀 있었음. v8 은 이를 의문문으로 열어 두고 ("물러나는 게 도망인지 / 물러나는 게 다정인지") 그 사이 멈춤을 인정 → 마지막 2행에서 처음으로 "듣고 싶었다"는 욕구를 (E001 v8 의 bridge 구조처럼) 자기 인정한다. Phase 4 답게 해결 없이 final chorus 로 떨어짐.

### Universal-imagery audit
- 캠퍼스/강의실/복도/카페 직접 묘사 0건 (v6 의 "복도 끝" 도 제거, "같은 자리/의자/유리"로 보편화).
- 능력/공감각/sensory origin 0건 (motif_audit_seojun_sensory_origin OK).
- self-hearing 능력 가시화 0건 (motif_audit_self_hearing_prohibition OK — "듣고 싶었다"는 욕구 인정만, 능력 작동 묘사 없음).
- 결별/이별/안녕/영영 어휘 0건 (song brief words-to-avoid 준수).
- 비-눈물 클리셰 0건 ("젖은 유리"는 phase 4 motif 인정 범위, "비/눈물" 직접 어휘 없음).
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백).
- Phase 4 minor-key tilt 유지: 어긋남·번짐·늦음·물러남 동사군 위주, 결별 단정어 회피.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "멀어진 / 멀어 / 멀어지는" ㅁ-ㄹ 계열 의도적 반복 — chorus motif, 부드러운 비음.
- "부르려다 / 부르고 / 부를 수 / 부르는" ㅂ-ㄹ 계열 의도적 반복 — chorus emotion line 키동사.
- "익숙해지면 안 되는 종류의 익숙함" — 한 호흡 안 들어감 확인. BPM 73에서 안정.
- "젖은 유리에 손을 대면 흐려지는 윤곽처럼" — "흐려지는" 약간 길지만 long-vowel 중심, 발음 명확.
- 영어/숫자/외래어 0건. 반말 자기독백 일관.
- jamo 이모티콘 0건. stage direction 0건.

## Removed/changed from v6

- v6 "같은 길에 서 있었는데 한 걸음 뒤로 물러섰어" → "같은 자리에 앉았는데 다른 박자로 숨을 쉬었어" (Phase 4 opener thesis 직타).
- v6 "복도 끝에서 너를 봤는데 / 나는 다른 길로 돌아갔어" (캠퍼스 신 직접) → "같은 자리에 앉아 있는데 / 다른 길로 돌아가고 있었어" (보편 자리 + 진행형으로 cumulative retreat 표현).
- v6 bridge 두 부정문 → 두 의문문 + "듣고 싶었던 건지도 몰라" 자기 인정 (E001 v8 bridge 구조 호환).
- v6 final chorus 평탄 종결 → "멀어진 건 우리가 아니라 — 내 목소리였어" 1-variant lift 추가.
- v6 outro 없음 유지 (final chorus 안에 lift 흡수).
- chorus emotion line 자체는 v6 그대로 lock 유지 (canon anchor).

## Unanswered for Conductor

1. **Final lift 변형 ("내 목소리였어")** — 외부귀속 → 자기귀속 회수가 Phase 4 의 "어긋남" 정서를 강화하지만, female intimate alto 가창에서 "내" 단음절 emphasis 가 약하면 verbatim 3회 ("먼저 부를 수 있던 목소리였어")로 회귀 가능.
2. **Bridge 길이 (7행)** — v6 의 5행에서 2행 늘림 (욕구 인정 라인 추가). 8-12행 권고 하한 근접. 더 길게 가야 할 경우 "그 한 걸음 너머의 너" 다음에 "내가 닫아 둔 건 너의 거리가 아니라 / 내 목소리의 거리였을지도" 추가 가능.
3. **"젖은 유리"** — Phase 4 motif set 의 "젖은 창문"을 "유리"로 살짝 추상화함 (창문보다 일반). song brief 의 "rain-tears cliche 회피"와 충돌 없음 확인.
4. **"계절"** — verse 2 마지막 "다른 계절을 사는 사람처럼" 한 줄만 사용. Phase 4 lyric image seed ("유리 너머의 너")와 호환. 직접 계절 선언 (가을/봄) 회피.
5. **여성 보컬 적합성** — 자기관찰형 라인 ("익숙해지면 안 되는 종류", "부르는 법을 잠시 잊은 사람처럼") 다수 → female intimate withdrawn 톤에 적합. minor-key tilt + 결별 회피 register 유지.
