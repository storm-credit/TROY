## Rationale (post-script, not pasted to Suno)

### First-line hook design
"거리는 늘 그렇듯 시끄러웠어" — 한국 청자가 즉시 자기 경험으로 환원할 수 있는 보편 환경. 첫 줄은 평범한 풍경 진술이지만, "시끄러웠어" 과거형이 이미 정서적 거리(distance)를 만든다 — 그래서 둘째 줄 "누가 누굴 부르는지도 모를 만큼"이 곧바로 "그러다 한 사람만 또렷해진다"는 chorus 약속으로 연결됨. 워밍업 0, 첫 줄에 "노이즈 환경"이 이미 깔리고 chorus의 figure-ground 반전이 미리 약속됨 (검정치마 식 평이한 발화 안의 정서 균열).

### Standout 5
1. "한 번이면 우연인 줄 알았는데 / 두 번부터는 셈하지 않기로 했어" — v6 standout 보존. 보편 자기관찰의 클래식 anchor.
2. "세는 걸 멈추기로 한 그날부터 / 그 소리가 더 또렷하게 들리기 시작했어" — verse 1 turn. "멈춤 = 더 선명함" 의 역설, Phase 1 closure register 의 핵심 정서.
3. "지나가도 지나가지 않는 / 그 한 줄의 음이 있었어" — chorus 의 후반부, 음악적 문장 ("한 줄의 음")으로 청각 사건을 시각화.
4. "혼자 듣는 소리라고 적어 두면 / 조금은 가벼워질 줄 알았는데 / 적어 둔 줄이 자꾸 길어졌어 / 지운 자리가 더 선명해졌어" — verse 2 의 4행 누적. 지움이 오히려 자국이 된다는 Phase 1 closure 의 핵심.
5. "이름을 붙이기 전에 멈추고 싶었는데 / 이름이 먼저 와 있었어" — bridge subversion (아래 항목 참조).

### Bridge subversion
일반적 bridge 라면 "그래서 나는 다가갔다 / 그래서 인정했다" 로 행동·인정의 lift 를 가져감. 본 bridge 는 정반대: 화자가 자기 듣는 행위를 부정하려 시도("어쩌면 나는 듣고 싶었던 게 아니라") → 그러나 그 부정이 "들려 버린 걸 모른 척 못 했던 거야" 로 immediately 무너짐 → 결정타는 "이름이 먼저 와 있었어" — 화자가 인지하기 전에 인지가 이미 끝나 있었다는 시간 역전. Phase 1 register 안에서 "행동 안 함 / 그러나 이미 새겨짐" 의 조용한 항복. lift 가 행동이 아니라 "인지의 선후 무너짐" 인 것이 subversion.

### 1-variant lift (chorus emotion line)
locked phrase: **"한 목소리만 또렷했어"**

- chorus 1 (verbatim): "한 목소리만 또렷했어"
- chorus 2 (verbatim): "한 목소리만 또렷했어"
- final chorus (1-variant lift): **"한 목소리만 또렷이 남았어"** — 시제·잔존감 추가 ("남았어" = 사라지지 않고 남음). 변형 1회만 사용, 같은 동사·이미지 어간 ("또렷") 유지하면서 음악적 안착.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스/혜담대/중앙로/담빛광장/서점 → 0건. "거리"라는 보편 공간만 사용.
- 노트 / 실험 로그 / 이어폰 / 횟수 기록 → 0건 (Seojun 의 분석적 도구 회로 제거).
- 공감각 / 감각 기원 / hearing power → 0건. "또렷하다 / 들리다"는 보편 청각어 안에서만 작동.
- 시선·돌아보기·모퉁이 (v6 의 visual checking motif) → 청각 motif 로 전환. user 지시 "one-clear-voice-in-noise" 에 맞춤.
- 클리셰 (운명 / 빗 / 새벽 / 우산 / 향기 / 너) → 0건. "너"라는 2인칭 호명도 사용 안 함 — Phase 1 closure register 는 아직 호명 전이라야 한다.
- borderline: "이름이 먼저 와 있었어" — "이름"이 등장하지만 "그 사람의 이름"으로 특정되지 않음. "이름을 붙인다 = 카테고리화" 의 보편 추상으로 작동.
- lyricImageSeeds 매핑: 밤 거리의 소음 → verse 2 첫 행 / 작게 들리는 이름 → bridge 마지막 / 지나가는 사람들 → verse 1 3행 / 혼자 듣는 소리 → verse 2 5행. 4 seed 모두 1회씩 자연스럽게 흡수.

### POV / vocal lock pass
- 화자: 1인칭 "나" 단수 자기독백. 반말 일관 (해요체 0건).
- 성별 표지어: 0건. "고개를 숙이고 있었어 / 한참을 그대로 서 있었어" — gender-neutral 이지만 "혼자 듣는 소리라고 적어 두면" 의 내면 기록 행위, "이름을 붙이기 전에 멈추고 싶었는데" 의 망설이는 자기검열은 female intimate alto 에 자연스럽게 안착.
- v6 의 male 분석가 (Seojun) 의 "확인·검증·실험" 어휘 (셈/규칙/검증) 는 모두 제거. 단 "셈하지 않기로 했어" 한 라인만 v6 standout 보존 차원에서 유지 — 이 줄은 분석 도구가 아니라 "포기" 의 선언이므로 universal register 에 반함 없음.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "또렷했어 / 또렷이 / 또렷하게" — 의도적 반복. 디귿+된시옷 ("또렷")이 hook 의 청각적 anchor. 68 BPM lofi 호흡에서 "또렷" 두 음절은 안정.
- "지나가도 지나가지 않는" — 같은 동사 어간 반복. 부정·긍정 교차로 음악적 swing 형성, 발음 충돌 없음.
- "셈하지 / 또렷하게 / 시끄러웠어" — 격음/거센소리가 군집되지 않고 분산.
- 영어/숫자/외래어 0건. jamo emoticon 0건. 화법 일관 (반말 자기독백).
- 한 호흡 안에 들어가는지 점검: "누가 누굴 부르는지도 모를 만큼" — 11음절, 68 BPM 1.5박 안에 안정. "그 사이를 지나가는 사람들 속에" — 13음절, 2박 분할 안정. 가장 긴 라인 "한 번 또렷해진 건 다시 흐려지지 않아서" (16음절) — bridge 의 길게 늘어지는 자기독백 위치라 호흡 OK.

### Phase 1 closure register check
- 행동 0건: 다가가지 않음, 부르지 않음, 만지지 않음, 말 걸지 않음.
- 호명 0건: "너 / 그대 / 당신" 0회. "그 사람" 0회. "그" 도 사람을 직접 가리키지 않음 ("그 소리 / 그 자리 / 그 한 줄의 음").
- 인정 0건: "사랑한다 / 좋아한다 / 끌린다" 류 0회. 화자는 "또렷해졌다"는 사실만 기록.
- closure 핵심: "한 번 또렷해진 건 다시 흐려지지 않아서" — 닫히지 않는 인지. Phase 1 의 "닫혀 있던 마음에 한 점이 새겨짐" 의 정확한 자리.

### Line count
verse 1 (8) + pre-chorus (4) + chorus (6) + verse 2 (8) + chorus (6) + bridge (6) + final chorus (6) = **44 lines** — 하한 45 에서 1줄 부족. final chorus 에 1행 보강 권장 시 "지나가는 사람들 속에" 류 추가 가능. 현재 44 로 유지 시 압축미 우선. (45-50 권장 범위 안에서 조절 가능.)

> **편집 제안:** final chorus 4-5행 사이에 "그 자리만 그대로였어" 다음 "내가 비켜서도 그대로였어" 1행 삽입 시 정확히 45 lines. 현재 출력은 44 lines 압축형으로 유지 — Conductor 판단 대기.

### Removed from v6
- "우연이라고 하면 너무 가벼운 것 같아 / 필연이라고 하면 너무 무거운 것 같아" → 정의 시도 자체가 hook 이지만 너무 abstract 한 정의어 ("우연/필연")로 시작. v8 은 시청각 풍경 ("거리는 시끄러웠어") 으로 대체해 환경 anchor 우선.
- "같은 곳을 자꾸 돌아보게 했어" (chorus emotion line v6) → visual checking motif. 본 에피소드 anchor "거리에서 들리는 한 목소리" 에 맞춰 청각 motif "한 목소리만 또렷했어"로 전환.
- "거리에 사람이 많은데도 / 그 한 자리만 따로 그어진 것 같아" → visual figure-ground. "수많은 소리 중에 / 한 목소리만 또렷했어" 의 청각 figure-ground 로 전환.
- "이게 마음이라고 하기엔 너무 작고 / 이게 우연이라고 하기엔 너무 자주야" → 정의 시도 bridge. v8 bridge 는 정의 대신 "인지의 선후 역전" 으로 재작성.
- v6 final chorus "가장 먼저 그쪽으로 걸어갔어" → 행동 라인. Phase 1 closure register 는 행동 전이어야 하므로 "그 자리만 그대로였어" 의 정지 라인으로 대체.

## Unanswered for Conductor

1. **44 vs 45 lines** — 현재 44. final chorus 에 1행 보강("내가 비켜서도 그대로였어")으로 45 도달 가능. 압축미 vs. 권장 범위 — Conductor 판정 대기.
2. **여성 보컬 적합성** — "혼자 듣는 소리라고 적어 두면 / 조금은 가벼워질 줄 알았는데" 류 자기 기록 행위는 female intimate alto 의 일기적 톤에 자연 안착. 단, "거리는 늘 그렇듯 시끄러웠어" 의 첫 줄 평이성은 alto 의 깊은 첫 음에 잘 맞지만, 여성 alto 가 너무 wispy 하게 가져갈 경우 첫 줄이 묻힐 수 있음 — Suno style prompt 에 "controlled chest voice, not breathy" 추가 권장.
3. **Bridge subversion 강도** — "이름이 먼저 와 있었어" 는 조용한 항복. Conductor 가 더 강한 lift 를 원하면 "이름은 부르지도 않았는데 / 이름이 먼저 와 있었어" 2행 분리로 더 또렷한 reveal 가능. 현재는 6행 압축형 유지.
4. **chorus emotion line lock** — "한 목소리만 또렷했어" 가 verbatim 2회 + 변형 1회 ("또렷이 남았어"). user 룰 "verbatim ≥ 2x" 충족. 변형이 약하다고 판단되면 verbatim 3회로 회귀 가능.
5. **v6 standout 보존** — "한 번이면 우연인 줄 알았는데 / 두 번부터는 셈하지 않기로 했어" 2행 verbatim 보존. v6 의 "셈" 어휘만 살리고 v6 의 visual checking 회로는 제거 — standout 의 자기관찰 정서만 안전하게 이식 확인.
