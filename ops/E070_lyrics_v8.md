## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘은 너에게 연락하지 않기로 했어" — 한국 청자가 즉시 자기 경험으로 환원할 수 있는 보편 결정. 다툼·결별·소원함이 아니라 "오늘 하루 따로 보내기"라는 작은 선택이 첫 줄에 즉발 hook 으로 들어감. 워밍업 0, 감정 사건 이미 발생 상태 (검정치마 「강아지」 1행 패턴 — 평범한 발화에 정서 균열). v6 의 "오래 듣던 멜로디가 어디선가 다시 흘러나왔어" 보다 한국 청취자에게 더 즉각적이고 보편적.

### 5 standout lines (v8 craft lock)
1. "비어 있는 게 어떤 무게인지 처음 알았어" — pre-chorus 진입 직전 감정 명명. 직설적이지만 진부하지 않은 정서 정리.
2. "멀리 있는 게 아니라 따로 있는 거였어" — pre-chorus 1행. "멀리 / 따로" 의 미세 의미 차를 한국어 감각으로 잡아냄. v8 standout 후보 #1.
3. "혼자 있는 법은 알고 있었는데 / 혼자 있는 법을 다시 배워야 했어" — chorus 후반 lift. "함께 있어 본 사람의 혼자"가 처음 혼자와 다르다는 thesis 가 한 호흡에 정리됨.
4. "한 사람을 빼고 보내는 저녁의 길이를 / 오늘 처음으로 끝까지 재어 보았어" — verse 2 closer. "재어 보았어" 라는 동사 선택이 보편 시어 안에서 이 곡만의 색을 만듦. v8 standout 후보 #2.
5. "비워 둔 자리도 같이 앉는 자리라는 걸 / 오늘 처음 알았어" — final chorus interior lift. 부재가 곧 함께함의 한 형태라는 phase 4 정직 — pre-loss instability 안에서 따뜻함이 살아 있음.

### Bridge subversion
Bridge 는 verse·chorus 의 "처음 알았어" 패턴을 잠시 멈추고, 결정/억제/금욕적 선택의 톤으로 전환:
"보고 싶다는 말은 너무 큰 말이라 / 오늘은 그 말 대신 다른 말을 골랐어" — 그동안 노래가 따라간 "발견" 모드를 "선택" 모드로 뒤집음. 마지막 "혼자가 외로운 게 아니라 / 혼자가 조금 더 분명해진 거였어" 로 phase 4 의 정직 — 외로움의 부정이 아니라 외로움의 재정의 — 를 한 줄로 굳혀 final chorus 로 떨어뜨림. v6 bridge ("잃지 않은 채로 잃어가는 일이 있어") 의 손실 어휘를 의도적으로 뒤집음.

### Final chorus — 1-variant lift
Locked phrase "같은 자리에 있어도 너 없는 자리는 처음이었어" 는 chorus 1·2 verbatim 2회. final chorus 의 마지막 lift 라인은 1회 변형 허용 룰로:
"같은 자리에 있어도 너 없는 자리에 앉아 있었어" — 동사를 "처음이었어 → 앉아 있었어" 로 한 차례 미세 변형하여 정서가 발견(perception)에서 거주(occupancy)로 안착. 변형 1회만 사용. lift 약하면 verbatim 3회로 회귀 가능.

### Universal-imagery audit
- 작품 신 직접 묘사 0건: 캠퍼스/강의실/이어폰/멜로디 모티프 → 모두 제거. v6 의 "오래 듣던 멜로디 / 번진 노트 / 문틀이 가르는 프레임 / 청회색 저녁" 직접 호출 0건.

- 클리셰 (비/새벽/우산/너의 향기/빈자리·빈 의자 직접 명명) 0건. "비워 둔 자리"는 클리셰 "빈자리"와 의도적으로 다름 — 능동형 ("비워 둔") 으로 작중 화자의 선택성 유지.
- 결별 어휘 (헤어졌다 / 끝났다 / 마지막 / 안녕) 0건 — 사용 금지 룰 준수. Phase 4 텐션은 "처음" 의 반복으로 유지.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "처음" 반복은 의도적 정서 anchor (5회) — 청취 시 motif 형성, 발음 명확.
- "같은 자리 / 같은 의자 / 같은 컵 / 같은 저녁 / 같은 시계" — "같은 X" 시리즈 의도 반복. BPM 76에서 한 호흡 안 안정.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백, female intimate alto 톤).
- 라인당 한 호흡 OK. 가장 긴 라인 ("한 사람을 빼고 보내는 저녁의 길이를") 도 BPM 76 기준 안정.
- chorus 락: "같은 자리에 있어도 / 너 없는 자리는 처음이었어" — 2행 분할. 디귿/지읒계열 무겁지 않음, 알토 톤에 침착하게 안착.

### Removed from v6
- "같은 선율이 돌아와도 우리는 같은 자리로 돌아가지 못했어" → 사용자 지시 thesis 변경에 따라 chorus anchor 자체 교체. "선율 / 음 / 박자" 음악 어휘는 작품 신 직접 묘사 위험으로 전면 제거.
- "타일 사이 빗물 / 번진 노트 / 문틀이 가르는 프레임 / 청회색 저녁" → brief image family 였으나 v8 universal register 에 맞춰 모두 제거. 대신 "의자 / 컵 / 저녁 / 창밖 / 시계 / 메시지" 보편 시어로 치환.
- "헤어진 건 아닌데 / 이전의 우리로는 더 이상 갈 수 없었어" → "결별 직전" 톤이 너무 강함. "오늘은 너에게 연락하지 않기로 했어" 의 작고 의도적인 하루 단위 선택으로 다운스케일.
- "잃지 않은 채로 잃어가는 일이 있어" bridge → 손실 어휘를 "선택 어휘" 로 뒤집음 ("보고 싶다는 말은 너무 큰 말이라").
- 50라인 안 압축 (총 47라인): outro 별도 행 제거, final chorus 안 lift 흡수.

## Unanswered for Conductor

1. **Thesis 시프트 확인** — 사용자 v8 지시 "처음 따로 보내는 시간의 무게" 가 v6 "돌이킬 수 없는 금" 보다 한 단계 더 부드러운 phase 4 톤. brief 의 "irreversible crack" 보다는 "first-empty-evening" 에 가까움. E070 의 에피소드 기능을 "금"에서 "처음 따로"로 살짝 부드럽게 가져가는 결정이 맞는지 conductor 확인 요망. 만약 "금" 선언적 톤이 필요하면 v6 chorus anchor 회귀 가능.
2. **변형 라인 1회 사용** — final chorus 마지막 "같은 자리에 있어도 너 없는 자리에 앉아 있었어" 가 음악적 lift 로 작동하는지는 Suno 생성 후 청취 판단. 약하면 "같은 자리에 있어도 너 없는 자리는 처음이었어" verbatim 3회 회귀.
3. **여성 보컬 적합성** — verse 2 "읽지 않은 메시지 칸을 한참 들여다봤어" / "오늘만큼은 내 하루를 내 안에서 끝내 보고 싶어서" 류 자기관찰 라인은 female intimate alto withdrawn 톤에 잘 맞춤. male vocal 회귀 시 일부 라인 (메시지 칸 들여다보기) 가 애상 쪽으로 기울 위험.
4. **Bridge 길이 (8행)** — 권고 8-12행 하한. 더 길게 원하시면 2행 추가 가능: "너에게 가지 않는 게 너에게 가는 또 다른 방법일지도 / 오늘은 그 가능성에 조용히 기대어 봤어".
5. **Outro 제거 결정** — 별도 outro 없이 final chorus 안 lift 라인으로 종결. fade-out 별도 라인 필요시 "같은 하늘 아래에서 / 나는 오늘도 너의 자리를 비워 두었어" 추가 가능.
