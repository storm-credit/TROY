## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘은 일찍 자려고 했었는데" — 누구나 즉시 자기 경험으로 환원할 수 있는 보편 발화. 시작부터 "오늘을 끝내려는 의지" 와 "끝나지 않은 한 칸" 의 균열이 첫 두 줄 안에 박힘 (E001 v8 의 "잘 지내냐는 말에 잘 지낸다고 답했어" 패턴 = 평범한 자기독백 안에 정서 사건 동시 발생). 워밍업 0. 봄/내일/희망 같은 thesis-단어를 첫 줄에 풀지 않음으로써 closure register 유지.

### Chorus emotion line derivation (carefully)
brief 의 raw anchor ("상처를 기억하는 밤인데도, 나는 자꾸 내일 쪽을 바라봤다") 는 두 절로 길고 1인칭이 노출됨 → 음악적으로 무거움. v6 의 "자꾸 내일 쪽을 바라봤어" 는 좋지만 살짝 직설적·시각 동사 의존. v8 에서는 "오늘은 내일이 먼저 깨어 있었어" 로 derive — (a) 주체를 '내일' 로 옮겨 화자의 의지 부정 ("내가 기다린 게 아니라 내일이 먼저 와 있었다"), (b) "깨어 있다" = 잠 못 듦 + 작은 빛, 두 의미 동시 작동, (c) closure register 안에서 first-time anticipation 을 우회적으로 표현 (희망 단어 0건). chorus 1·2 verbatim 2회 lock, final chorus 마지막 라인만 1-variant lift ("오늘은 내일이 나를 먼저 깨워 두었어") — 같은 구문 골격에 "나를 깨워 두었다" 로 미세하게 능동/수동 전환, 화자가 결국 그 깨움을 받아들였음을 암시하면서도 "기다림" 단어는 끝까지 회피.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 이어폰 / 베드사이드 램프 / 커튼 가장자리 / 휴대폰 화면 → 작품 motif 직접 호출 모두 제거
- 캠퍼스 밤길 회상 → 제거 (보편 "방", "이불", "창문 한 칸")
- "유리 뒤" / "지나간 봄을 다 기억" → 제거 (v6 의 직설 라인 후퇴)
- 공감각·청취 능력·sensory origin → 0건 (motif_audit_seojun_sensory_origin / arin_song_thread 모두 OK)
- 봄/내일/희망 직접 단어 → "내일" 1회만 chorus 에 anchor 로 사용, "봄"·"희망"·"설렘" 직접 단언 0건 (pre-chorus "설렌다는 말은 가벼운 것 같아" 로 거리두기 형식만 사용)
- 클리셰 (비/새벽/우산/너의 향기/빈자리/별빛) → 0건
- borderline: "한 칸이 켜져 있었어" — '칸' 은 시적 추상 명사, 특정 motif (커튼 한 칸 / 창문 한 칸) 환기 가능하지만 보편 시어 범위. 화자가 의지하는 작은 단위로 작동 OK.

### Korean pronunciation pass (female intimate alto, BPM 73)
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "깨어 / 켜져 / 깨워" ㅋ/ㄲ 계열 의도적 반복 — chorus 에서 한 음절 단위 명료 lift, 발음 명확.
- "한 칸 / 한 번 더 / 한쪽" 한 자 단어 반복 — 작은 단위가 모티프화, alto 호흡 안에서 안정.
- "이불 끝을 자꾸 다시 매만지게 됐어" — 한 호흡 안에 들어감 확인. "매만지게 됐어" 부분이 약간 길지만 BPM 73 에서 안정.
- "잠이 오는 자리를 더듬다가 / 오지 않는 쪽으로 손이 먼저 갔어" — '오는/오지 않는' 동사 거울 구조, intimate alto 의 자기관찰 라인에 적합.
- 영어/숫자/외래어/jamo emoticon 0건. 화법 일관 (반말 자기독백).
- final chorus lift "나를 먼저 깨워 두었어" 의 'ㅁㄴ' 받침 흐름 안정, 끝맺음 어미 "두었어" 가 chorus 1·2 의 "있었어" 와 운 호환.

### Female intimate alto 적합성
- brief 는 "low intimate male vocal" 이지만 hard lock 은 female intimate alto. 두 톤 모두 작동하도록 자기관찰 라인 ("이불 끝", "한 칸", "이름을 붙이지 못한 마음") 으로 통일 — 페미닌 한쪽으로 기울지 않음.
- "괜찮아진 건 아닌데 / 괜찮아도 되는 건지 처음 물어봤어" (bridge) — 여성 alto 의 조심스러운 자기허락 정서에 잘 맞음. closure phase 의 첫 균열을 "물어봄" 으로 처리하여 uplift 회피.

### Phase 1 closure register · no uplift 검증
- thesis 가 "내일을 기다리는 마음이 처음 생긴 봄날" 이지만, 다음 절제 룰 적용:
  - "기다림" 직접 동사 0건 → "이름을 붙이지 못한 마음 / 한 칸이 켜져 있었어" 로 우회
  - "설렘 / 봄 / 희망 / 다정 / 따뜻" 단어 0건 → bridge 에서 "밝은 건 아니고 다정한 것도 아니야" 로 명시적 부정
  - chorus 멜로디 폭발 회피 → chorus 1·2 마지막 라인을 첫 라인의 verbatim 반복 ("오늘은 내일이 먼저 깨어 있었어") 으로 닫아 lift 대신 안착
  - bridge 의 자기 허락 ("괜찮아도 되는 건지 처음 물어봤어") 도 단정 아닌 의문형으로 종결

### Removed from v6
- "오래 닫아 두었던 창문 앞에 서 봤어" → 첫 hook 으로는 동작 직접 묘사가 약함. "오늘은 일찍 자려고 했었는데" 로 교체 (즉발 자기독백 hook).
- "자꾸 내일 쪽을 바라봤어" → 시각 동사·1인칭 의지 노출. "오늘은 내일이 먼저 깨어 있었어" 로 derive (주체 전환 + 시청각 우회).
- "상처를 다 기억하는 밤인데" / "상처는 그대로인데" → '상처' 라는 단어가 closure register 에서 살짝 무겁고 narrative 환기. "이름을 붙이지 못한 마음 하나가" 로 추상화.
- "오래 잊고 있던 자리에서 새는 빛 같아" → '빛' 직접 호출이 hopeful uplift 로 미세하게 흐름. bridge 를 "밝은 건 아니고 다정한 것도 아니야 / 그냥 어둡지 않은 어떤 색이" 로 재작성 (이중 부정으로 lift 차단).
- v6 final chorus "오늘 밤만 한쪽이 부드러워졌어" → '부드러워졌어' 가 결과 단언. "한 칸이 켜져 있었어" 의 verbatim 회귀로 변경, 마지막 라인만 1-variant lift.

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 마지막 "오늘은 내일이 나를 먼저 깨워 두었어"** 가 음악적 lift 로 작동하는지는 Suno 생성 후 청취 판단 필요. lift 가 약하면 verbatim 5회 ("오늘은 내일이 먼저 깨어 있었어") 로 회귀 가능.
2. **Vocal lock 충돌** — song_brief 는 "low intimate male vocal" 이지만 user lock 은 "female intimate alto." v8 라인은 두 톤 모두 작동하도록 중립 작성했으나, 최종 캐스팅 결정은 Conductor 확인 필요. 인-월드 룰상 아린이 부르면 female alto 가 정합.
3. **"내일" 단어 사용량 (chorus 5회, final chorus lift 1회 추가 = 총 6회)** — closure register 안에서 thesis 키워드 노출이 과한지 검토. 줄이려면 chorus 마지막 verbatim 라인을 "한 칸이 끝까지 켜져 있었어" 로 치환 가능 (단, hook 약화 위험).
4. **Bridge 길이 (6행)** — v6 (4행) 보다 2행 늘어남, "괜찮아진 건 아닌데 / 괜찮아도 되는 건지 처음 물어봤어" 의 자기허락 의문이 thesis "처음 기다리는 마음" 의 핵심을 짊어짐. 더 짧게 원하시면 마지막 2행만 남기고 첫 4행 압축 가능.
5. **Outro 부재** — closure phase 답게 final chorus 의 1-variant lift 에 그대로 잠기게 처리. fade 라인 필요시 "잠들기 전에 한 번 더 / 같은 쪽을 바라봤어" 2행 추가 가능 (단, 50라인 budget 내 47라인이라 여유 있음).
