## Rationale (post-script, not pasted to Suno)

### First-line hook design
"시계 소리가 방보다 먼저 자리에 앉았어" — "시계 소리"라는 일상 청각 사물이 "방"이라는 공간보다 먼저 의인화되어 자리에 앉음으로써, 청자는 즉시 "정적이 사물처럼 와 있는 시간"이라는 정서 좌표 안에 들어선다. 워밍업 0. 첫 행 안에 "시간 인지의 둔화"와 "마지막 한 시간이 이미 시작됐다"가 동시에 작동. ambient-piano BPM 60 의 첫 박과 정확히 합치하는 라인 길이.

### 5 standout lines (v8 신규 craft 락)
1. "시계 소리가 방보다 먼저 자리에 앉았어" — opening hook, 사물이 시간보다 먼저 도착함
2. "할 말이 없는 게 아니라 / 할 말을 다 모아 두기로 했어서" — 부재의 정직 선언, "모아 두다"의 성숙 동사
3. "들이마시는 일과 내쉬는 일 사이에 / 오늘이라는 한 시간이 들어와 있었어" — 호흡과 시간의 등가 metaphor, Phase 6 침묵 완성 모티프 보편화
4. "잡지 않은 손이 더 따뜻할 수도 있다는 걸" — bridge subversion 의 핵심 라인, 작별의 통념 ("잡고 싶다") 을 정직하게 뒤집음
5. "이 한 시간을 다 듣고 일어서기로 했어" — final chorus 의 1-variant lift, "머물기로 했어"가 "다 듣고 일어서기로 했어"로 미세 변형되어 종지 직전의 의지 라인이 됨

### Bridge subversion
관습적 작별곡 bridge = "더 잡고 싶었어 / 보내기 싫었어" 의 강화된 회한.
v8 bridge subversion: "잡지 않은 손이 더 따뜻할 수도 있다는 걸 / 이 한 시간 안에서 처음 알게 됐어" — 작별의 통념 (closeness=holding-on) 을 한 줄로 뒤집고, "잡지 않음 = 더 따뜻함" 이라는 비-회한 정직으로 착지. "나는 떠나려는 게 아니라 / 정확히 여기에 한 번 더 있었던 거야" 로 떠남의 통상 동사를 "정확히 머묾"으로 재정의. E118 의 dignified-parting 으로 자연 연결.

### Final chorus 1-variant lift
chorus 의 마지막 행 "오늘만은 아무 데도 가지 않기로 했어" 가 final chorus 에서 "이 한 시간을 다 듣고 일어서기로 했어" 로 변형. 변형 1회 룰 안에서 사용. "가지 않기로 했어" (정지 의지) 가 "다 듣고 일어서기로 했어" (완수 의지) 로 lift — 종지 직전의 미세한 능동성 회복. "듣다" 동사로 ambient-piano 의 청취감과 의도적으로 합치. verbatim chorus는 chorus1, chorus2 에서 2회 반복으로 chorus lock ≥2x 충족.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 한강 / 카페 달 / 카페 달 방향 → 제거 (보편 "방", "창밖", "거울", "물 한 잔"으로 치환)
- 옷깃을 정리하는 바람 / 길 위의 걸음 → 제거 (실내 "한 시간"의 정적 frame 으로 통합)
- "종지", "예의", "재시작 vs 종지" 어휘 → 제거 (보편 "조용한 한 시간 / 정확하게 머물기"로 치환)
- 공감각·청취 능력·sensory origin (Stage 5 threshold) → 직접 명명 0건 (대신 "가장 조용한 한 시간", "들이마시는 일과 내쉬는 일 사이" 의 보편 청각·호흡 frame 만 작동)
- 클리셰 (이별/눈물/우산/마지막 인사/너의 이름) → 0건
- 계절 직접 선언 → 0건 (늦가을 색감은 "창밖은 어두워졌고", "방은 어둡지 않았어"의 빛 contrast 로만 함의)
- forbidden cliches (nostalgia bloom / restart cue / destiny turn) → 0건 검증 완료
- borderline: "거울" 한 회 등장 — E001 v8 과 motif 중복이지만 "거울 쪽으로 두지 않았어" 의 부정 사용으로 자기-감시 회피의 정서 차별화. OK.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "조용한 / 정확하게 / 정확히" ㅈ 계열 의도적 반복 — Phase 6 정적 motif 형성, 발음 명확.
- "들이마시는 / 내쉬는 / 들어와" ㄷ-계열 / 호흡 동사 의도적 반복 — 청취 시 "호흡 = 시간"의 motif 강화.
- BPM 60 ambient-piano 에서 호흡 fit 검증: "할 말을 다 모아 두기로 했어서" — 한 호흡 가능. "들이마시는 일과 내쉬는 일 사이에" — 한 호흡 안에 들어감 (12음절 안정).
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백, 여성 intimate alto 적합).
- 받침 무게: "와 있었어 / 앉아 있었어 / 들어와 있었어" 의 과거진행형 종결 통일 — Phase 6 의 "이미 와 있는 정적" 시제 lock.
- jamo emoticon (ㅎㅎ/ㅋㅋ/ㅠㅠ) 0건. stage direction parenthetical 0건.

### Removed/changed from v6
- "오늘의 목적은 종지였어" chorus → 한자어 "종지"는 보편 청자 즉발성 약함 + Phase 6 의 가장 깊은 정적 anchor 와 어긋남. "가장 조용한 한 시간이 와 있었어" 로 교체 (보편·즉발).
- "오늘의 목적을 가는 길 내내 한 번 더 적어 두었어" verse 1 opening → "길 / 가는 길" 작품 motif (한강 쪽 길) 직접 호출 위험. "시계 소리가 방보다 먼저 자리에 앉았어" 로 교체 (보편 실내 정적 hook).
- "한강 쪽 길은 늘 같은 자리에 있었지만" verse 2 → 작품 지명 직접. "창밖은 어두워졌고 방은 어둡지 않았어" 로 교체 (보편 빛 contrast).
- "끝을 정확하게 두는 일이 가장 큰 예의여서" → "예의" 라는 한자어 abstract 가 v6 의 강한 명제였지만 v8 universal register 에서 청자 즉발성 약함. "오늘만은 정확하게 머물기로 했어" 로 치환 — "정확하게" 부사를 "머묾"이라는 행동에 붙여 보편화.
- "끝을 회피하지 않는 게 사랑일 수 있다는 말이야" bridge → "사랑/예의/끝" 의 명제적 단언이 v8 의 정직 register 와 충돌. "잡지 않은 손이 더 따뜻할 수도 있다는 걸 / 이 한 시간 안에서 처음 알게 됐어" 로 교체 — 명제 대신 발견의 동사.
- v6 "잡지 않은 채로도 정직할 수 있어서" final chorus → bridge 의 "잡지 않은 손" 으로 흡수, final chorus 는 lift 라인 "이 한 시간을 다 듣고 일어서기로 했어" 로 종결.
- 라인 수: v6 = 35행 → v8 = 47행 (45-50 라인 권고 안 충족, ambient-piano BPM 60 의 호흡 여백 확보).

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 마지막 "이 한 시간을 다 듣고 일어서기로 했어"** 가 음악적 lift 로 작동하는지는 Suno 생성 후 청취 판단 필요. lift 가 약하면 verbatim 3회 ("오늘만은 아무 데도 가지 않기로 했어") 로 회귀 가능.
2. **chorus emotion line 변경** — v6 의 "오늘의 목적은 종지였다" 에서 v8 의 "가장 조용한 한 시간이 와 있었어" 로 derived-anchor 교체. genre_assignment.json 의 chorusEmotionLine 은 null 이므로 director derive 정당하지만, 시리즈 anchor 보드에 등록 시 디렉터 최종 lock 필요.
3. **Phase 6 시제 통일** — 모든 종결을 "와 있었어 / 있었어" 의 과거진행형 으로 통일. E118 의 90s-vintage-ballad 가 같은 시제 보드를 받을지, 아니면 동일 Phase 안 register 차별화로 다른 시제 (예: 회상형) 를 받을지 사전 결정 권장.
4. **"방 / 거울 / 창문" 모티프 E001 중복** — E001 v8 universal pass 와 v8 E117 universal pass 사이에 실내 정물 motif 가 의도적으로 겹침 (Phase 1 첫 위축 ↔ Phase 6 마지막 정적 의 회귀 호) — 시리즈 universal 보드 차원에서 의도된 회귀로 lock 권장. Conductor 미승인 시 E117 의 "거울 쪽으로 두지 않았어" 1행 삭제 가능.
5. **Bridge 길이 (6행)** — 8-12행 권고 하한이지만 ambient-piano BPM 60 + Phase 6 정적 register 에서 의도적 단축. 더 길게 원하시면 "끝을 향해 가는 게 아니라 / 끝을 같이 듣고 있는 거야" 2행 추가 가능.
