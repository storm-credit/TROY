## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘 네가 한 말이 자꾸 다시 켜져" — 보편 즉발 hook. "켜지다" 라는 동사가 잠 든 방의 불·머릿속의 한 문장·다시 시작되는 오늘을 동시에 켠다 (1줄에 3겹). 검정치마/잔나비 식 평범한 발화 안 정서 균열 패턴. 워밍업 0, 누구나 1초 만에 자기 경험으로 환원되는 "잠 들기 전 다시 떠오르는 한 마디". Phase 2 의 "조심스럽게 닿기 시작한 자리" 의 미세한 자기 검열을 그대로 드러냄.

### Chorus emotion line placement
이 에피소드는 JSON 에 `chorusEmotionLine: null` — 새 hook 을 자체 설계했다. Locked phrase "그 한 문장이 / 나를 깨워 두었어" 를 chorus 1·2 verbatim 2회 (lock ≥2x 충족), final chorus 마지막 lift 라인은 1변형 허용으로 "한 문장이 나를 재워 주었어" 로 종결 — 같은 동사 어간 "재우다/깨우다" 의 의미 반전으로 lift 만들어냄 (밤이 답을 결정해 준 안착). 변형 1회만 사용.

### Universal-imagery audit (v6 → v8 변경 표)
- 작품 신 직접 묘사 0건 확인:
  - "횡단보도 / 신호등 / 골목 / 그림자" (v6 의 거리 풍경) → 제거. lofi-bedroom 레인 + Phase 2 lyricImageSeeds ("자기 직전의 한 문장", "되감는 오늘", "다시 떠오르는 표정", "잠 못 드는 이유") 에 더 정확히 정렬되도록 "방·이불·천장·머리맡 시계" 의 보편 침실 이미지로 전치.
  - 캠퍼스/카페/공연장 등 작품 무대 직접 명명 0건.
  - "운율 / 듣는 능력 / 공감각" sensory motif 직접 호출 0건. "한 문장" 이라는 추상 단위만 사용 (motif_audit_seojun_sensory_origin · self_hearing_prohibition 양 audit OK).
- v6 standout 보존: "보폭을 맞춘다는 건 / 같아지는 게 아니라 기다리는 거라는 걸" 을 pre-chorus 에 그대로 보존 — 이 곡의 thesis 운반체. 단, v6 의 "걷기" 메타포에서 "대답하기" 메타포로 재맥락화 (= "빠른 대답이 정답이 아닐 수도 있다는 걸") 해서 "잠 들기 전" 프레임 안으로 통합.
- 클리셰 (비/새벽/우산/너의 향기/빈자리/창가) 0건. borderline: "머리맡의 시계가 한 칸 더 갔어" — 시계 모티프이지만 클리셰가 아니라 한 문장의 무게를 외부 객체로 객관화하는 정직한 영상이라 채택.

### Five standout lines (v8 craft check)
1. "오늘 네가 한 말이 자꾸 다시 켜져" — first-line hook. 잠 든 방의 불·다시 시작되는 오늘·머릿속 문장 3겹.
2. "대답을 너무 빨리 한 것 같아서 / 대답을 너무 늦게 한 것 같아서" — Phase 2 의 자기 검열 양가성. 같은 구문 끝만 바꾸어 잠 못 드는 이유의 양면을 한 호흡에 압축.
3. "보폭을 맞춘다는 건 / 같아지는 게 아니라 기다리는 거라는 걸" — v6 에서 보존한 thesis 운반체. 이 곡의 가장 큰 정서 무게를 pre-chorus 에 두어 chorus hook 으로 폭발하기 직전의 압력 형성.
4. "잠이 오는 걸 잠이 든 거라고 믿어 보려다가 / 믿어지지 않는 게 더 정직해서 그만뒀어" — v8 의 정직성 록. 자기위로의 거짓을 스스로 거두는 alto-register 자기 응시.
5. "한 문장이 나를 재워 주었어" — final lift. chorus 의 "깨워 두었어" 를 1변형으로 뒤집어 음악적 안착. 잠을 빼앗은 그 한 문장이 결국 잠을 결정해 주는, 답을 미루기로 한 자기 결정의 부드러운 종결.

### Bridge subversion
v6 bridge 는 "다른 속도가 틀린 속도가 아니라는 걸" 의 자기 설득이었음. v8 bridge 는 한 단계 더 들어가서 — "빨리 답하는 게 마음을 쓰는 일이라고 믿었는데 / 오늘은 천천히 답하는 게 더 마음을 쓰는 일 같아" — 화자의 기존 가치관 자체를 뒤집는다. Phase 2 의 "조율" 학습이 "속도" 가 아니라 "타이밍" 의 문제라는 미세한 재정의. 자기 설득 → 자기 갱신.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음 (lofi-bedroom 67 BPM 에 안전).
- "켜져 / 깨워 / 재워" — chorus 동사 anchor 가 ㅋ/ㄲ/ㅈ 계열로 가볍고 청취 명확.
- "대답 / 다시 / 다정 / 닿다" 디귿계열 의도적 반복 — verse 1·chorus 모두 cross-section motif 형성.
- "이불 끝을 한 번 더 끌어 올렸어" — 한 호흡 안에 들어감 확인. "끌어 올렸어" 부분이 약간 길지만 67 BPM 에서 안정.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백). 자모 emoticon (ㅎㅎ/ㅋㅋ/ㅠㅠ) 0건.
- "한 박자 / 한 칸 / 한 줄 / 한 문장 / 한 번" — 단위 명사 "한-" 의 의도적 반복 = "한 문장의 무게" thesis 의 음운 강화. 7회 등장하지만 위치 분산되어 monotony 회피.

### Removed from v6
- "횡단보도 앞에서 한 발 먼저 내려놓고 나서야" → 거리 풍경 직접. "오늘 네가 한 말이 자꾸 다시 켜져" 로 교체 (즉발 hook + 침실 prefiguring).
- "조급한 건 너의 발이 아니라 / 나의 마음이었어" (v6 chorus) → "다정인지 미안인지 / 밤이 한참 길어졌어" 로 교체 — Phase 2 의 자기 분류 어려움이 "발/마음" 의 신체 비유보다 더 정직하게 들어감 (E001 v8 의 "다정인지 도망인지" bridge 와 같은 자기 의문 패턴).
- "오후의 그림자 / 골목 / 시간을 두는" (verse 2) → 침실 시계로 교체. lofi-bedroom 레인 + emotionalAnchor "잠 들기 전 떠오르는 한 문장의 무게" 에 정확 정렬.
- v6 final chorus 의 "오늘은 다음 신호를 같이 기다릴게" → "내일 아침에 한 박자만 늦게 / 같은 말을 다시 해 볼게" — 약속의 단위가 "신호" (외부 거리 객체) 가 아니라 "내일 아침" (시간) 으로 바뀌어 침실 프레임 안에 머문다.
- v6 의 "빠르다는 건 늘 정답인 줄 알았는데" 는 의미 보존하되 chorus 직전 압력 라인 "빠른 대답이 정답이 아닐 수도 있다는 걸" 로 압축 변환.
