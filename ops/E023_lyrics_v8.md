## Rationale (post-script, not pasted to Suno)

### First-line hook design
"잘 지내냐고 쓰다가 한 번 지웠어" — 한국 청자가 즉시 자기 경험으로 환원할 수 있는 보편 "메시지 작성 망설임" 행위. 시작부터 "안부를 쓰는 중인데 보내지 못함" 이라는 사건이 이미 일어난 상태. E001 v8 의 "잘 지내냐는 말에 잘 지낸다고 답했어" 와 의도적 거울 — E001 은 받은 안부에 거짓 응답, E023 은 보내려는 안부의 망설임. Phase 2 등록증.

### 5 standout lines (v8 lift candidates)
1. "호칭 한 줄을 비워 둔 채로 한참 앉아 있었어" — v6 "이름 없는 마음" 계승, 행위로 외화.
2. "보내지 않은 말이 가장 오래 남는 걸" — pre-chorus 정서 좌표.
3. "한 번도 불러본 적 없는 이름을 / 오늘은 한 번 부르고 싶었어" — chorus 결구, 보편 "첫 호칭" 욕망 직격.
4. "마침표를 찍었다가 물음표로 바꿨어 / 끝내는 게 무서운 건지 묻고 싶은 건지 몰라서" — verse 2 핵심, 문장부호로 정서 외화.
5. "안부는 묻는 게 아니라 / 묻고 싶다는 마음을 들키는 건지도 몰라" — bridge subversion line, 안부 정의 자체를 뒤집음.

### Bridge subversion
v6 bridge 는 "확인 vs 놓치고 싶지 않음" (질투 자기위장) 이었음. v8 bridge 는 안부 행위의 정의를 뒤집어 — "안부 = 묻는 행위" 가 아니라 "안부 = 들키는 행위" 로 재정의. 즉 chorus 의 "부르고 싶었어" 욕망을 bridge 가 "이미 들키고 있다" 로 한 단계 끌어올림. Phase 2 의 "가까워지는 숨" 정서적 핵심 (감추려 하지만 이미 새는 마음).

### Final chorus 1-variant lift
chorus 라인 5행 verbatim, 마지막 6행만 "오늘은 한 번 부르고 싶었어" → "오늘은 작게라도 부르고 싶었어" 로 변형. "한 번" → "작게라도" 는 숫자 대 부사 변환으로 결심의 톤이 1단계 낮아지면서 (동시에) 더 정직해짐. 변형 1회 룰 안에서 사용. 음악적 lift 가 약하면 verbatim 3회로 회귀 가능.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스 그룹테이블/종이컵/사람들 웃음 → 제거 (v6 의 질투 thesis 신 흔적 모두 제거)
- 공감각/타인 감정 결 청취 → 제거
- 도현/타인 등장 → 제거
- 클리셰 (비/새벽/우산/너의 향기/빈자리/별/달) → 0건
- 보편 시어로 채택: "한 글자 / 호칭 / 마침표 / 물음표 / 이모지 / 첫 줄 / 손가락" — 메시지 작성이라는 동시대 한국인 공통 경험.
- 영어/숫자/외래어: "이모지" 1회만 사용 (대체 시 "웃는 그림" 가능하나 직관성 우선). 그 외 0건.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "지웠어 / 비웠어 / 멈췄어" 디귿·미음계열 의도적 반복 — 청취 시 motif 형성, 발음 명확.
- "한 번도 / 한 번 / 한 줄 / 한 시간 / 한 마디" — Native counting 일관 유지. Sino-Korean 숫자 0건. 시간 표현 (밤이 깊어지면) 추상화로 처리 — Sino vs Native 충돌 없음.
- "보내지 않은 말이 가장 오래 남는 걸" — 한 호흡 안에 들어감. "않은" 부분 약간 길지만 BPM 66에서 안정.
- 화법 일관: 반말 자기독백 ("-어/-야"). E001 v8 과 톤 매칭.
- 영어 0건 (이모지는 외래어이지만 일반화된 한국어 — Suno 발음 안정 확인됨).

### Section length
- 50 라인 안 압축 (현재 47라인): verse 8 / pre-chorus 4 / chorus 6 / verse 6 / bridge 6 / final chorus 6 + 섹션 구분. v6 의 outro 별도 섹션 제거, final chorus 안에 정서 lift 흡수.
- bridge 6행: 8-12행 권고 하한이지만 의도적 짧음 — "들키고 있다" 자기 의문이 마무리되지 않은 채 final chorus 로 떨어지게 함.

## Removed from v6 (thesis pivot 포함)

- v6 thesis "작은 질투" 자체 제거 — JSON anchor 가 "한 번도 해본 적 없는 종류의 안부 인사" 로 명시. v6 는 이전 brief (jealousy/paper cup) 기반이었고, 현재 lyric image seeds (새로운 인사 / 처음 쓰는 호칭 / 조심스러운 첫 글자 / 삭제와 다시 쓰기) 와 mismatch. v8 는 JSON anchor 우선.
- v6 chorus lock "괜찮은 척한 자리에서 / 손 안의 컵만 조금 구겨졌어" → 폐기. anchor 변경에 따른 정당한 교체.
- v6 종이컵/그룹테이블/시선 모티프 → 전부 제거. v8 모티프 셋: 안부 작성, 호칭 망설임, 마침표/물음표, 보내지 않은 말.
- v6 standout "이름을 짓고 싶지 않은 마음이 하나 있어서 / 그 마음은 이름 없는 채로 더 무거워졌어" 의 정서적 무게는 v8 verse 1 의 "호칭 한 줄을 비워 둔 채로 한참 앉아 있었어" + chorus 의 "한 번도 불러본 적 없는 이름을" 으로 계승. "이름 없음" 의 정서가 "호칭 망설임" 으로 외화됨.

## Unanswered for Conductor

1. **Thesis pivot 정당성 확인** — v6 brief (작은 질투) vs JSON anchor (첫 안부). v8 는 JSON anchor 를 신뢰. brief 와 JSON 이 충돌하면 어느 쪽이 source of truth 인지 Conductor 결정 필요. (brief 가 우선이면 v6 유지, JSON 우선이면 v8 적용.)
2. **이모지 사용** — verse 2 에 "이모지 하나를 붙였다가 다시 떼어 냈어" 1회. 외래어 0건 룰 엄격 적용 시 "웃는 그림 하나를 붙였다가" 로 교체 가능. 동시대성 vs 시적 거리감 trade-off.
3. **변형 라인 1회 사용** — final chorus 마지막 "작게라도 부르고 싶었어" 가 음악적 lift 로 작동하는지는 Suno 생성 후 청취 판단. lift 약하면 verbatim 3회 ("한 번 부르고 싶었어") 회귀.
4. **v6 standout 라인 부활 옵션** — "이름을 짓고 싶지 않은 마음" 한 행이 lyrically 강력하므로 verse 2 끝에 1행 삽입 옵션 가능. 현재는 "호칭 한 줄을 비워 둔 채로" 로 환원해 두었음.
5. **여성 보컬 적합성** — "보내지 않은 말 / 호칭 망설임 / 손가락 멈춤" 류 메시지 작성 행위는 female intimate alto 톤에 안정 매칭. 인-월드 아린의 첫 안부 정서로 적합. male 회귀 시 "가까운 척도 먼 척도" 라인이 약간 페미닌하게 들릴 수 있음 — female lock 유지 추천.
