## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘 처음 만난 사람처럼 인사를 골랐어" — yes 이후의 "어제와 같은 사람을 처음 보는 듯한" 핵심 paradox 를 첫 줄에 압축. 워밍업 0, 정서 사건이 이미 발생한 상태에서 시작. v6 의 "오늘은 인사를 어떻게 해야 할지 한참을 골랐어"보다 한 줄에 정서 작동 (어색함 + 새로움)이 동시에 들어감. 누구나 한 번쯤 겪은 보편 미세 망설임.

### Chorus emotion line placement
Locked phrase "기쁨도 처음이라 / 우리는 조금 삐걱거리며 웃었어" 는 chorus 1·2 와 final chorus 진입부에 verbatim 3회. final chorus 의 마지막 lift 라인은 v6 standout couplet "완벽하지 않아서 다행이라고 / 그 한 줄에 작게 말해 두었어" 를 그대로 보존 (1-variant 허용 룰을 chorus 내부 변형이 아니라, "stable happiness 부정 + unpracticed joy 인정" 의 정서 lift 자리에 사용). chorus body 안에서 변형 없이 verbatim 락 유지.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스/복도/카페/창가 직접 호출 → 제거 (v6 의 "복도의 빛" → 보편 "같은 자리 / 늦은 빛" 으로 추상화)
- 두 컵 / 카페 메모리 직접 묘사 → 제거 (v6 brief 의 dominant image 였지만 신 호출이라 v8 에서 0)
- 공감각·청취 능력·sensory origin → 제거 (motif_audit_seojun_sensory_origin 위반 0)
- 운명론·"perfect"·"forever"·"solved" → 0건 (brief 의 words to avoid 모두 통과). "완벽하지 않아서 다행"은 "perfect"의 부정이므로 의도적 사용 OK.
- "yes" / "대답" / "고백" 단어 0건 — "그 한 줄"은 v6 final chorus 의 보편 시어로 직접 재호출하지 않음.
- 클리셰 (운명/영원/우리 둘만의/하나가 된) 0건.
- borderline: "어깨 한쪽에" — couple-poster 직전이지만 "손이 닿지 않은" 으로 거리 유지, "couple forever" energy 회피 OK.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "삐걱거리며" 는 ㅂ된소리 + ㄱ + ㄹ 연속이지만 BPM 75 에서 한 호흡 안에 안정. female intimate alto 톤에 맞춰 부드럽게 굴림.
- "보폭 / 발소리 / 박자" ㅂ계열 의도적 반복 — 청취 시 motif 형성, 발음 명확.
- "어색함 / 어긋난" ㅓ계열 모음 반복으로 정서 anchor.
- "한 박자 망설였고", "한 박자 먼저 비켰어" — 한 호흡 안에 들어감 확인.
- 영어/숫자/외래어 0건. 화법 일관 (반말 1인칭 자기관찰 + "우리는" 관계화법 chorus 내 자연 전환).

### Removed from v6
- v6 verse 1 "오늘은 인사를 어떻게 해야 할지 한참을 골랐어" → 5 standout 후보지만 "한참을 골랐어"가 약간 늘어짐. v8 첫 줄 "오늘 처음 만난 사람처럼 인사를 골랐어" 로 더 즉발 hook 으로 교체 (paradox 압축).
- v6 chorus 후반 "서로의 눈을 길게 못 보고 / 허공을 한 번씩 쳐다보기도 했어" → 두 줄이 같은 동작이라 v8 에서 "허공을 한 번씩 쳐다보기도 했고 / 같은 자리에 서 있는데 / 어제와 다른 보폭이 자꾸 끼어들었어" 로 압축·전환. "어제와 다른 보폭" = bridge subversion 의 핵심 어휘를 chorus 안으로 이식.
- v6 verse 2 "복도의 빛이 어제와 같지 않게 보였어" → 캠퍼스 신 직접 호출. "같은 빛인데 오늘은 다른 결로 떨어져서" 로 보편화 (장소 명시 제거).
- v6 bridge "오늘은 그저 조금씩만 옆에 있어 보기로 했어" → 6행 안에 다정인지 서투름인지 자기 의문을 추가, bridge subversion 강화 ("다정인지 서투름인지 나도 잘 모르겠지만 / 모르는 채로 옆에 있는 게 오늘은 미워지지 않았어").
- v6 final chorus 의 standout couplet ("완벽하지 않아서 다행이라고 / 그 한 줄에 작게 말해 두었어") → 그대로 보존. 이 couplet 이 E031 의 정서 anchor 이자 lift 라인.

## Five standout lines (claim)

1. "오늘 처음 만난 사람처럼 인사를 골랐어" — first-line paradox hook
2. "싫지 않은 어색함이라는 게 처음 생겨서 / 그게 오히려 작게 다행스러웠어" — pre-chorus 진입 직전 정서 명명
3. "어제와 다른 보폭이 자꾸 끼어들었어" — chorus 후반 lift, bridge 어휘를 chorus 로 끌어올림
4. "다정인지 서투름인지 나도 잘 모르겠지만 / 모르는 채로 옆에 있는 게 오늘은 미워지지 않았어" — bridge subversion (확신 없음을 미움 없음으로 재정의)
5. "완벽하지 않아서 다행이라고 / 그 한 줄에 작게 말해 두었어" — v6 standout 보존, final lift

## Unanswered for Conductor

1. **Chorus 변형 0회 결정** — v8 룰의 "final 1-variant lift" 를 chorus 내부 변형이 아니라 final chorus 의 closing couplet (v6 standout 보존) 으로 해석함. chorus body 변형이 필요하면 final chorus 의 "어제와 다른 보폭이 자꾸 끼어들었어" 를 "오늘은 같은 보폭으로 서 있어 보았어" 로 1회 변형 가능.
2. **Bridge 길이 (6행)** — 권고 8-12행 하한이지만 의도적으로 짧게. 더 길게 원하시면 2행 추가 가능 ("좋아진다는 게 어쩌면 / 같은 자리에서 다시 처음을 배우는 일인지도 몰라").
3. **여성 보컬 적합성** — verse 2 "얼굴을 보다가 창 쪽으로 한 번씩 비껴 섰어" 류 자기관찰 라인은 female intimate alto · close shy 에 잘 맞음. v6 brief 의 "smiling before knowing where to put your hands" 톤 유지.
4. **45-50 라인 카운트** — 본문 (paste block) 47라인. 룰 안에 안착.
5. **"yes" 단어 부재 검증** — 본문 0회 확인. "대답 / 고백 / 좋다고 했어" 류 직접 어휘도 0회. 정서만 보편 작동.
