## Rationale (post-script, not pasted to Suno)

### First-line hook design
"어깨가 거의 닿을 만큼 가까이 앉았는데" — 시작 한 줄에 거리의 역설(가까움이 곧 가시)이 이미 들어 있음. 청자는 즉시 "옆에 있어도 멀게 느꼈던 한 사람"의 기억으로 환원 가능. 도서관/책상 직접 호출 없이 "어깨 / 가까이 앉다" 의 보편 감각만 사용. 워밍업 0, 이미 정서 사건이 발생한 상태로 진입.

### Five standout lines
1. "한 뼘이 한 뼘이 아닌 것 같았어" (verse 1) — 같은 거리가 다르게 느껴지는 thesis 의 가장 짧은 시화.
2. "가까운 게 이렇게 시린 줄은 몰랐어" (verse 1) — phase 1 closed-spring 의 시각/체감 합치. "시리다"가 lofi-bedroom 톤에 안착.
3. "가까운 자리에 가시가 있었어 / 보이지 않는데 자꾸 찔렸어" (chorus 1·2행) — derived chorus anchor. "가시"는 thesis 의 키워드를 그대로 시화하고, "보이지 않는다"가 가시의 정체를 외부가 아니라 화자 안으로 옮겨 놓음.
4. "혼자만 다른 박자로 숨을 쉬었어" (chorus closer) — 같은 공간 안의 비동기. 도서관 묘사 없이 "공기·박자·숨"으로 universal.
5. "어쩌면 가시는 너에게 있는 게 아니라 / 내가 들고 온 건지도 몰라" (bridge 1·2행) — bridge subversion 의 시작. "당신 때문" 에서 "나의 울타리" 로 정서 책임 이동.

### Chorus emotion line placement & lift
Derived anchor "가까운 자리에 가시가 있었어" 는 chorus 1행에 verbatim 3회 (chorus 1·2·final) — anchor lock ≥2x 충족. 4행만 final chorus 에서 자연 lift 변형: "뻗을 수가 없어서 더 아팠어" → "뻗지 못한 게 오늘의 거리였어" (1-variant 룰). 같은 동사 어간(뻗다)을 유지하면서 정서 무게가 통증 → 인정으로 이동, restraint 안의 lift 확보. 6행도 final 에서 "혼자만 다른 박자로 숨을 쉬었어" → "오늘은 그게 작은 용기였어" 로 종결, hopeless 가 아닌 phase 1 closure (작은 용기 = 닫힘 안의 첫 균열) 안착.

### Universal-imagery audit
- 도서관 / 책 / 페이지 / 창가 / 책상 / 의자 → 0건 (v6 의 "한 페이지 한 페이지" 제거, "어깨 / 무릎 / 손 / 공기" 보편 신체 좌표로 치환)
- 작품 motif 직접 호출 (이어폰, 공감각, 청취 능력) → 0건. "박자 / 숨 / 공기" 는 일반 시어, motif_audit (heard vs said, sensory origin) 충돌 없음
- 클리셰 (비/새벽/우산/너의 향기/봄바람) → 0건. "시리다" 한 번만 사용, phase 1 회색 봄 톤과 합치
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백)
- 잠재 borderline: "한 뼘이 한 뼘이 아닌 것 같았어" — "한 뼘"은 일반 단위, 청자가 자기 거리 경험으로 즉시 환원 가능
- 캐논 충돌 점검: contact-equals-danger 모티프와 phase 1 단계 정합 (가까움 = 위험·시림, 닿음 보류). over-romantic 으로 흘러가지 않음

### Korean pronunciation pass
- BPM 70 lofi-bedroom 한 호흡 안 들어가는지 점검: 모든 라인 7-13 음절대 (verse 1 "이불을 한 번 더 끌어안고 누웠어" 류 12음절 한도 안에서 안전)
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음
- "가까이 / 가까운 / 가까운 게" — ㄱ계열 의도 반복으로 가시(ㄱ-ㅅ) motif 의 청각 anchor 형성. 발음 명확, 무리 없음
- "찔렸어 / 찌르다" 없음 — "찔렸어"는 chorus 의 핵심 동사, ㅈ된소리 1회만 (반복 6+ 회 위험 회피)
- "뻗으면 / 뻗을 수가 / 뻗지 못한" — ㅃ된소리 3회 변형. 빠른 hook 자리가 아닌 chorus 중반·후반 슬로우 자리에 배치, lofi 70 BPM 에서 명확 발음 가능
- 영어/Sino숫자 0, 외래어 0, jamo emoticon 0
- "괜찮은 / 괜찮지" 의도적 반복 (E001 chorus 와 album-level motif 호응, 직접 인용 아님)

### Removed from v6 / repositioned
- v6 thesis 가 "조용한 가까움 = 평온" 으로 흘러간 것 → v8 은 JSON anchor "가까이 있어도 닿지 않는 거리의 가시" 로 교정. brief의 quiet co-presence 는 살리되, phase 1 closed grayness (가시·시림·다른 박자) 가 dominant 정서가 되도록 재배치
- v6 "둘만 남은 자리가 처음이라서" → v8 "어깨가 거의 닿을 만큼 가까이 앉았는데" (정황 → 거리 역설 직접 진입)
- v6 "한 페이지 한 페이지 천천히 넘어갔어" / "공기에 한 사람의 결이 섞이고" → 도서관 직접 묘사. v8 "옆에 있는 사람의 결을 따라 / 호흡을 맞추려다 자꾸 어긋났어" 로 universal 화 (책 제거, 결·호흡만 유지)
- v6 chorus "낮은 목소리로도 충분한 날이 있었어" → 안락한 결말로 닫혀서 thesis 와 어긋남. v8 derived anchor "가까운 자리에 가시가 있었어" 로 교체
- v6 bridge "오늘은 가까운 게 무겁지 않았어 / 같이 앉을 수 있는 거리였어" → 정서 해소가 너무 빠름 (phase 1 에 어울리지 않음). v8 bridge subversion: 가시의 출처가 상대 → 화자 자신("내가 들고 온 건지도") 으로 이동, 해소 보류
- v6 final "오늘은 그게 가능한 날이었어" → 너무 안락. v8 final lift "오늘은 그게 작은 용기였어" — 닫힘 안의 미세 균열로 phase 1 closer 격에 맞춤. 50라인 안 압축 (현재 49라인)

## Unanswered for Conductor

1. **Brief vs JSON anchor 충돌 — v8은 JSON 우선** — `E008_song_brief.md` 의 emotional thesis ("둘만 남은 시간은 생각보다 조용하다") 는 평온 톤, `120_episode_genre_assignment.json` E008 anchor ("가까이 있어도 닿지 않는 거리의 가시") 는 thorned 톤. 사용자 지시문이 후자를 명시 인용했으므로 v8 은 JSON anchor 채택. brief 의 "quiet co-presence" 요소는 verse 2 "침묵이 편한 척" / final "같은 공기 안에 있어 본 것만으로" 에 잔향만 유지. brief 톤 회귀 원하시면 chorus 4행 "더 아팠어" 를 "더 조용했어" 로 완화 가능.

2. **Bridge 길이 (7행)** — 8-12행 권고 하한이지만, phase 1 closer 답게 답을 내지 않은 채 final chorus 로 떨어지게 함. 더 길게 원하시면 2행 추가 후보: "그래서 멀게 앉을 핑계도 / 가깝게 앉을 핑계도 둘 다 없는 자리에서".

3. **Final lift 변형 1회 사용 — 4행 "뻗지 못한 게 오늘의 거리였어"** 가 음악적 lift 로 작동하는지 Suno 생성 후 청취 판단. 약하면 verbatim "뻗을 수가 없어서 더 아팠어" 회귀.

4. **여성 alto intimate 보컬 적합성** — "어깨 / 무릎 / 손 / 작게 줄여 두었어" 는 female intimate withdrawn 톤에 잘 맞춤. "가시"의 경자음(ㄱ-ㅅ)이 alto 호흡 자리에 명확 안착. 톤 회귀 시 male 적용 어색해짐 — female lock 유지 권고.
