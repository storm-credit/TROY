2. **2부 Suno paste pack** → Suno UI Custom Mode → Title / Lyrics / Style / Negative / Settings 각 필드에 paste → 곡 1차 생성 (3-4회 재생성 권장)
3. **3부 Storyboard sheet prompt** → GPT Image 2 또는 Imagen 4 에 paste (1부 시트 image 첨부) → 콘티 시트 1장 산출
4. **4부 Video prompts (8개)** → 각 prompt 를 GPT 에 paste 해서 톤 다듬기 → Seedance 2.0 UI 에 paste (1부 캐릭터 시트 image 첨부) → 15초 영상 8개 산출
5. **5부 Editing guide** 따라 CapCut 또는 외부 NLE 에서 조립 → 음악/영상 sync → 자막 → 컬러 → 렌더링

각 단계 완료 후 산출물 위치는 사용자 자유 선택 — 권장 위치:
- 캐릭터 시트 image: `export/ost/E001/characters/{name}_sheet_v1.png`
- Suno 결과: `export/ost/E001/audio/E001_v{N}.mp3`
- 콘티 시트: `export/ost/E001/storyboard/E001_storyboard_v1.png`
- 비디오: `export/ost/E001/videos/CUT{NN}_v{N}.mp4`
- 마스터 MV: `export/ost/E001/E001_master_v{N}.mp4`
