## Rationale (post-script, not pasted to Suno)

### First-line hook design
"같은 시간을 같이 보냈는데 이름이 달라졌어" — Phase 6 늦가을 청자가 즉시 자기 경험으로 환원할 수 있는 보편 정서. "관계의 호칭은 바뀌었는데 같이 보낸 시간 자체는 사실로 남는다" 는 thesis 의 첫 음절이자 정서 균열의 즉발 hook (워밍업 0). v6 의 "손을 잡지 않은 채로 같은 박자로 걸었어" 는 walking 모티프였는데, harness thesis 가 "시간의 무게 변화 / 다른 의미의 함께"로 이동했으므로 시간-호칭 축으로 hook 재설계. "끝까지 같이 가지 않아도 되는 자리에서" (v6 standout) 는 verse 1 마지막 행에 보존.

### Chorus emotion line placement
Locked phrase "같은 시간이 다른 이름이 되는 자리에서" 는 chorus 1행 verbatim 2회 (chorus 1·2), 5행에서도 동일 verbatim 1회 더. final chorus 마지막 lift 라인은 자연 변형 "같은 시간이 다른 이름으로 남는 자리에서" 로 종결 — "되는" → "남는" 한 동사만 미세 변형해 시제를 완료형으로 닫는 음악적 안착 (restraint 안에서의 lift). v8 final 1-variant rule 안에서 사용. chorus lock ≥ 2x 충족 (실제 3회 + 변형 1회).

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 긴 복도 / 외부 보행로 → 제거 (보편 "자리", "오후"로 치환)
- 손 잡지 않는 손 (motif) → 추상화 ("더 가깝지도 더 멀지도 않게 앉아 있었어")
- 서준/아린 고유명 → 0건
- forever / sealed reunion / complete peace / happy ending → 0건 (forbidden cliches 클리어)
- "오래 알았다는 말이 이제 가장 정확한 말이 됐어" → 보편 늦가을 정서, 작품 신 호출 없음
- "지난여름" → 일반 시제 표현, 작품의 특정 phase 호출 아님
- 클리셰 (낙엽/단풍/마른 가지/이별/안녕) → 0건
- borderline: "늦가을" 직접 명시 — harness thesis 가 "늦가을"을 직접 키워드로 쓰므로 1회만 verse 1, bridge 마지막, verse 2 마지막에 분산 사용 (3회). 제목으로 빼지는 않음.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음. acoustic-ballad BPM 67 fit.
- "같은 / 같이" / "다른 / 달라" 디귿계열 의도적 반복 — 청취 시 motif 형성, 발음 명확.
- "또렷해지는" — 한 호흡 안에 들어감 확인. ㄸ-ㄹ-ㅎ 구조지만 BPM 67 에서 안정.
- "이름이 달라졌어 / 이름이 바뀌었는데 / 다른 이름이 되는 / 다른 이름으로 남는" — "이름" 어휘 의도적 반복으로 chorus motif 형성. 발음 위험 없음.
- "오래 알았다" — "알았다" ㄹ-ㅇ-ㅆ 명료. female intimate alto 호흡 fit.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백). 자모 emoticon 0건. 스테이지 디렉션 0건.

### Removed from v6 (v8 deltas)
- v6 first hook "손을 잡지 않은 채로 같은 박자로 걸었어" → walking 모티프가 harness thesis "시간의 무게 변화" 와 정확히 일치하지 않아 시간-호칭 축으로 교체. v8 hook "같은 시간을 같이 보냈는데 이름이 달라졌어" 가 thesis 직격.
- v6 chorus lock "같은 박자로 걷는 밤이 우리를 덜 두렵게 했어" → "같은 시간이 다른 이름이 되는 자리에서" 로 교체. "걷는 밤 / 두렵게" 는 phase 5-6 transition 톤이지만, E110 (Phase 6 mid) 에선 "두려움" 정서가 이미 자리 잡지 않음 — "더 가깝지도 더 멀지도 않게" 의 안정된 거리감으로 이동.
- v6 standout "끝까지 같이 가지 않아도 되는 자리에서" → verse 1 마지막 행에 verbatim 보존 (사용자 지시 표창 라인 안전).
- v6 verse 2 "긴 그림자가 두 사람을 같은 길이로 늘어뜨렸어" → 작품 신 (긴 복도 그림자 motif) 직접 호출 위험. v8 에선 "오래 알았다는 말이 이제 가장 정확한 말이 됐어" 보편 라인으로 교체.
- v6 bridge "이건 다시 시작이라는 말이 아니야 / 오늘 밤만이라는 말이야" → bridge subversion 약함 (단순 부정). v8 bridge 는 "끝 vs 챕터 vs 그냥 한 오후" 3중 가설 자기의문으로 재작성, "이름을 정하지 않은 채로 옆에 있는 게" 라는 v8 다운 정서 우회.
- v6 final chorus 의 "큰 약속 없이도 받쳐지는 자리가 있어서" → final lift 가 주장적 (positive declaration). v8 에선 final chorus 의 마지막 라인을 "같은 시간이 다른 이름으로 남는 자리에서" 1-variant lift 로 닫아, 종결을 완료형 "남는" 으로만 미세 이동 — Phase 6 livable uncertainty 톤 보존.
- 라인 수: v6 39행 → v8 47행 (verse/pre/chorus/verse/chorus/bridge/final = 7 sections, 45-50 권고 안). 5 standout 라인 분포: v1 첫 줄 hook + v1 last (v6 standout 보존) + chorus 1행 lock + bridge "이름을 정하지 않은 채로" + final 마지막 1-variant lift.

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 마지막 "같은 시간이 다른 이름으로 남는 자리에서"** 가 음악적 lift 로 작동하는지 Suno 생성 후 청취 판단 필요. lift 가 약하면 verbatim 4회 ("같은 시간이 다른 이름이 되는 자리에서") 로 회귀 가능. 또는 verbatim + 1행 잔향 ("이름이 달라져도 시간은 사실로 남았어") 추가 옵션.
2. **여성 보컬 적합성** — "더 가깝지도 더 멀지도 않게 앉아 있었어" / "오래 알았다는 말이 이제 가장 정확한 말이 됐어" 류 자기관찰 라인은 female intimate alto Phase 6 톤에 적합. 무리한 lift 없이 medium-close, steady 유지 가능.
3. **Bridge 길이 (5행)** — 8-12행 권고 하한 미달이지만, Phase 6 의 "확신은 줄어들었지만 태도는 더 분명해졌다" 정서를 짧게 닫는 게 톤 적합. 더 길게 원하시면 2행 추가 옵션 ("끝이라는 말도 챕터라는 말도 / 오늘은 너무 큰 말이어서 입에 올리지 않았어").
4. **늦가을 직접 명시 3회 (verse 1, verse 2, bridge)** — harness thesis 키워드라 보존했으나, 시각적 계절 묘사 (낙엽/단풍/찬바람) 0건으로 추상 보편화. 시각 묘사 1행 추가 원하면 verse 1 에 "창에 닿는 햇살이 더 낮은 각도로 들어오는" 류 1행 추가 가능.
5. **v6 walking-rhythm 모티프 제거 결정** — harness thesis 가 "시간의 무게 변화 / 다른 의미의 함께" 로 이동해 walking 축을 시간-호칭 축으로 전환. v6 walking 모티프 회귀 원하면 verse 2 에 "같이 걸었던 자리도 이제 같이 앉아 있는 자리가 됐어" 1행 흡수 옵션.
