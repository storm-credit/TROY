## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오래 마주 앉아 있다는 걸 오늘에야 알았어" — 한국 청자가 즉시 환원할 수 있는 '관계의 시간 인식' hook. 시작 직후 이미 두 사람이 같은 자리에 있음을 보여 주고, '오늘에야' 라는 부사 하나로 v8 thesis (처음으로 둘 다 이해) 의 시간성을 첫 줄에 박는다. 워밍업 0행. v6 의 "오늘 아침은 흐렸고" (혼자 자세) 대비, "둘이 같이 인지" 로 출발선을 옮김.

### Five standout lines
1. "오래 마주 앉아 있다는 걸 오늘에야 알았어" — 첫 줄, 인지 늦음의 정직.
2. "이해받았다는 게 아니라 이해되었다는 자리" — 능동/수동의 방향 전환 (능동성 강요 없는 상호 인지).
3. "나는 너의 모양을 처음으로 줄이지 않았어 / 너도 나의 모양을 처음으로 늘리지 않았어" — 미러 페어, '진짜 모양' thesis 의 동사화.
4. "오늘 한 걸음은 회복이 아니라 자세였고" — v6 canon anchor 보존, Phase 5 closer 라인.
5. "사랑이 아니어도 미움이 아니어도 / 그 사이에 있는 진짜 모양이 있었어" — bridge 의 정의 거부 후 정의 (subversion).

### Bridge subversion
v6 bridge ("운명이 다시 내 어깨를 두드릴 거라 믿지 않았어") 는 자기 단언이었음. v8 bridge 는 자기 단언을 넘어, 관계를 정의하던 두 큰 단어 (사랑/미움) 를 동시에 부정한 뒤 '사이' 라는 세 번째 모양을 제시 — "진짜 모양" 의 의미를 thesis 적으로 풀어 둠. final chorus 직전 의미가 한 번 비틀림.

### Final chorus 1-variant lift
Final chorus 마지막 라인 "그 자세를 둘 다 같이 골라 들었어" 는 chorus 1·2 의 "골라 입었어" 에서 '입다 → 들다' 로 한 차례만 변형. '입다' 가 자세를 몸에 붙이는 정착이라면, '들다' 는 그 자세를 같이 받쳐 든 공동의 동작 — 둘 다-이해의 끝지점을 한 동사로 살짝 들어올림. lift 약하면 verbatim "골라 입었어" 회귀 가능.

### Chorus emotion line / lock placement
Locked phrase "우리는 처음으로 같은 모양을 보고 있었어" 는 chorus·final chorus 의 1행에 verbatim 3회 배치 (chorus lock ≥2x 충족). 보조 라인 "오늘 한 걸음은 회복이 아니라 자세였고" (v6 canon anchor) 는 4행에 verbatim 3회 보존 — 두 anchor 의 이중 lock 으로 thesis (둘 다 이해) 와 phase (회복 아님, 자세) 를 동시에 닫는다.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스/카페/특정 공간/벤치 → 제거 (보편 "마주 앉음", "컵", "옆자리" 만 사용).
- 측면 출입구/계단/횡단보도 (v6 motif set) → 제거. v8 thesis 가 '함께 자리' 로 이동했기에 이전 솔로 motif 도 의도적으로 차단.
- 청취 회복/이름 회복/공감각 → 0건. "크게 부르지 않아도 들리는 자리" 는 음향 비유가 아닌 정서 비유 (해석 잠금).
- 운명 / 재회 / 다시 시작 → 0건. bridge 가 "다시 처음이 될 거라고 약속하지 않았어" 로 명시 차단.
- 클리셰 (비/새벽/우산/너의 향기/빈자리) → 0건.
- 계절 직접 선언: "이전과 다른 가을 빛" 한 줄만, Phase 5 색감 신호로 사용.
- borderline: "컵 안의 식어 가는 온도" — 카페 신을 환기할 수 있으나, '컵' 은 어디든 있는 보편 사물. 신 묘사 아님.

### Korean pronunciation pass
- BPM 76 chamber-pop fit 확인. 모든 라인 한 호흡 안에 들어감.
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "이해받았다는 / 이해되었다는" — 자음 군집 보이지만 BPM 76 에서 모음 길이 충분, 명확.
- "줄이지 / 늘리지" 미러 페어, 'ㅈ/ㄴㄹ' 대비로 청취 motif 형성.
- "골라 입었어 / 골라 들었어" — final chorus lift 변형, 'ㄹㄹ-ㅂ/ㄹㄹ-ㄷ' 의 종성 차이로 음악적 닫힘.
- "둘 다" 반복은 의도적 (thesis anchor).
- 영어/숫자/외래어/jamo emoticon 0건. 화법 일관 (반말 자기독백).

### Removed / shifted from v6
- 솔로 motif 군 (측면 출입구/계단/횡단보도/흐린 아침) → 전부 제거. v6 가 '혼자 다음 문턱' 이라면 v8 은 '둘이 진짜 모양' — episode 내부 thesis 가 "관계의 진짜 모양을 처음으로 둘 다 이해하는 자리" 로 이동했기에 비주얼 축도 솔로 보행 → 마주 앉음 으로 교체.
- chorus emotion line "돌아온 소리는 없었고 / 그래도 다음 질문 앞에 설 수 있었어" → 솔로 정조. v8 의 둘 다-이해 thesis 와 충돌하여 폐기. "우리는 처음으로 같은 모양을 보고 있었어" 로 교체.
- v6 standout "오늘 한 걸음은 회복이 아니라 자세였고" → 보존, chorus 4행으로 이동 (지지 anchor 역할).
- "묻기로 했어" 의문 동사 axis → "보기로 했어 / 인정하기로 했어" 시각·인정 axis 로 교체 (둘 다 같이 보는 행위가 thesis).
- v6 outro 2행 → 제거 (45-50 라인 안에 final chorus 로 흡수).

### Line count
verse 1 (6) + verse 2 (6) + pre-chorus (4) + chorus (5) + verse 3 (6) + chorus (5) + bridge (6) + final chorus (5) = 43 라인. 45-50 라인 권고 대비 43 라인 — 의도적 압축 (chamber-pop 76 BPM 의 호흡 보호). 더 길게 원하시면 verse 3 끝에 2행 추가 가능 ("오래 닫혀 있었던 한 칸을 같이 열어 두었고 / 그 한 칸의 공기가 둘 사이에 머물렀어").

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 마지막 "골라 들었어"** 가 음악적 lift 로 작동하는지는 Suno 생성 후 청취 판단. lift 약하면 verbatim "골라 입었어" x3 회귀.
2. **이중 anchor 락 (둘 다 이해 + 자세) 동시 운용** — chorus 안에 anchor 두 개를 동시에 lock 하는 구조라 chorus 가 약간 무거워질 수 있음. 음악적 가벼움이 부족하면 4행 ("오늘 한 걸음은 회복이 아니라 자세였고") 을 final chorus 에서만 1회 잔향 형태로 잡고 chorus 1·2 에서는 빼는 옵션 가능.
3. **Bridge 의 사랑/미움 단어 직접 등장** — v8 standout #5 가 강하지만, 두 단어가 가사 안에 직접 등장하는 첫 자리. 작중 thesis 와 충돌 없음 (정의 거부 → 사이 정의) 이지만 phase 5 의 절제 톤에 비해 단어 무게가 클 수 있음. 약하게 가려면 "큰 말이 아니어도 작은 말이 아니어도" 로 추상화 가능.
4. **여성 보컬 적합성** — "이해되었다는 자리", "둘 다 같이 골라 입었어" 류 수동/공동 동사는 female intimate alto 톤에 잘 맞음. male vocal 회귀 시 일부 라인 ("작게 끄덕였어") 의 시각이 약간 페미닌하게 들릴 수 있어 female lock 유지 추천.
