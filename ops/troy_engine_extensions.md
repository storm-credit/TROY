# TROY Engine Extensions

## 1. Why Expand

현재 `TROY 엔진`의 플롯 뼈대는 충분하다.

이미 잠긴 것:

- ending-first lock
- reverse design
- seed -> echo -> payoff
- character arc
- scene rhythm
- revision loop

추가가 필요한 이유:

- 120화 장편에서는 플롯보다 감정 피로와 반복이 먼저 무너진다
- 이 작품은 `누가 무엇을 들었고, 무엇을 말하지 않았는가`가 핵심이다
- 아름다운 이별은 사건보다 여운과 이미지 관리가 더 중요하다

## 2. Orchestra Consensus

구조, 감정, 문체 파트 전문가 의견을 합치면, 추가 핵심 모듈은 아래 다섯 개로 압축된다.

1. relationship thermometer
2. reveal matrix
3. motif ledger
4. silence transition engine
5. emotion budget

## 3. Relationship Thermometer

역할:

- 각 화마다 관계가 어떻게 이동했는지 기록한다
- 사건이 아니라 `선택`이 관계를 바꾸게 만든다

핵심 상태:

- closer
- farther
- stalled
- distorted closeness
- redefined

## 4. Reveal Matrix

역할:

- 정보가 어떻게 흐르는지 관리한다
- 감정 개연성 붕괴를 막는다

핵심 축:

- what Seojun hears
- what Seojun misreads
- what Arin withholds
- what reader knows
- when truth surfaces

## 5. Motif Ledger

역할:

- 반복 이미지와 장소, 사물의 의미 변화를 관리한다
- 복선이 아닌 정서적 회수감을 만든다

핵심 축:

- object
- first meaning
- transformed meaning
- reprise episode
- ending residue

## 6. Silence Transition Engine

역할:

- `청취 소실`에서 `침묵의 완성`으로 가는 후반부 전용 엔진

핵심 축:

- hearing loss stage
- question frequency
- spoken truth frequency
- silence type
- trust without proof progression

## 7. Emotion Budget

역할:

- 화수와 악장 단위로 감정 피로를 관리한다
- 독자가 무뎌지지 않도록 고저를 배치한다

핵심 축:

- intensity
- recovery
- residue
- escalation ceiling
- phase pressure

## 8. Rule

앞으로 `TROY 엔진`에서 핵심으로 취급하는 것은 아래 열한 가지다.

- ending-first lock
- reverse design
- seed -> echo -> payoff
- character arc
- scene rhythm
- relationship thermometer
- reveal matrix
- motif ledger
- silence transition engine
- emotion budget
- revision loop
