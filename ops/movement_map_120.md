# Movement Map: 120 Episodes

## 1. Decision

내부 실무 단위는 `Phase`를 유지하고, 독자-facing 감정 구조는 `악장`으로 병기한다.

- internal: Phase 1-6
- reader-facing: 제1악장-제6악장

## 2. Selected Movement Set

선정안:

### 제1악장: 얇은 봄빛

- episodes: 1-20
- phase alias: Phase 1
- emotional line: 아직 차가운 세상에, 처음으로 얇은 빛이 스민다

### 제2악장: 가까워지는 숨

- episodes: 21-35
- phase alias: Phase 2
- emotional line: 말보다 먼저, 호흡과 거리의 변화가 사랑을 앞지른다

### 제3악장: 여름의 과열

- episodes: 36-60
- phase alias: Phase 3
- emotional line: 가장 크게 웃고, 가장 쉽게 오해하는 계절

### 제4악장: 젖은 균열

- episodes: 61-80
- phase alias: Phase 4
- emotional line: 장마 끝 공기처럼, 관계의 균열이 늦게 젖어든다

### 제5악장: 가을의 잔향

- episodes: 81-105
- phase alias: Phase 5
- emotional line: 지나간 사랑은 끝나지 않고, 다른 결의 잔향으로 남는다

### 제6악장: 늦가을의 종지

- episodes: 106-120
- phase alias: Phase 6
- emotional line: 함께하지 않아도 사랑이 진짜였음을 이해하는 맑고 단단한 끝

## 3. Naming Rule

- 악장명은 계절과 감각을 우선한다
- 과도한 음악 이론 용어는 피한다
- 제목만 읽어도 감정 곡선이 보이게 한다
- 마지막 악장은 해피엔딩의 환희보다 `맑은 종지`의 정서를 택한다

## 4. Use Rule

- planning docs may use `Phase`
- presentation docs may use `제N악장`
- chapter and episode docs should allow both labels when useful
