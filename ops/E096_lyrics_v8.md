## Rationale (post-script, not pasted to Suno)

### First-line hook design
"같이 가지 않기로 한 길이 너에게도 길이기를 바랐어" — 시작부터 결정이 이미 끝난 상태("같이 가지 않기로 한")이고, 그 결정을 자기 손해로 두지 않고 상대의 길로도 인정하는 한 줄. Phase 5 의 핵심 감정(분리 후의 응원)이 첫 줄에 압축됨. 워밍업 0, 첫 글자부터 보편 청자 환원 가능.

### 5 standout lines
1. "같이 가지 않기로 한 길이 너에게도 길이기를 바랐어" (verse 1·1행 hook)
2. "멀리 있는 응원은 소리가 없어서 더 오래 가는 것 같아" (pre-chorus 핵심 정의)
3. "오늘은 너를 한 페이지가 아닌 책으로 두었어" (v6 canon 라인 보존, chorus 5행)
4. "같이 있어도 따로인 사람이 있고 / 따로 있어도 같이인 사람이 있더라" (bridge 인식 전환)
5. "오늘은 그 책을 덮어 두고 너에게 길을 비켰어" (final chorus 변형, lift 종결)

### Bridge subversion
v6 의 bridge 는 "나의 결말을 위해 너를 다시 부르지 않기로 했어" — 자기 통제 선언. v8 의 bridge 는 인식 전환이다: "처음엔 따로 사는 게 헤어짐인 줄 알았어 → 따로 있어도 같이인 사람이 있더라 → 보지 않아도 좋다는 말이 무심한 줄 알았는데 / 그게 가장 깊은 응원이라는 걸 늦게 알았어." 같음/따로의 정의 자체를 뒤집는 자리. Phase 5 가 Phase 6 로 넘어가기 전 마지막 인식 갱신.

### Final chorus 1-variant lift
- chorus 1·2 lock: "오늘은 그 책을 펴지 않고 응원만 보냈어" (verbatim 2회)
- final chorus 변형 1행 only: "오늘은 그 책을 덮어 두고 너에게 길을 비켰어"
- 같은 책 메타포 안에서 "펴지 않고 → 덮어 두고", "응원만 보냈어 → 너에게 길을 비켰어" — restraint 안에서의 lift. 변형은 1회 룰 안. 펴지/덮어 두기 의 동사 어간 음운적 연결 유지.

### Korean pronunciation pass (BPM 72 fit)
- 자음 군집 위험 라인 0건. 된소리 hook 없음.
- "잘 지내" 의도적 반복 — 표제 hook (chorus 1·2행). female intimate alto 와 호흡 일치.
- "바랐어 / 알았어 / 두었어 / 보냈어 / 비켰어" 과거형 어미 의도적 반복 — Phase 5 의 "이미 일어난 일을 천천히 인정하는" 시제 일치.
- "닿지 않아도 / 보내고 있는 / 들리지 않아도" — 부정·진행형의 부드러운 ㄴㄷㅇ 흐름, BPM 72 walking pace 와 맞음.
- "한 사람을 곁에 두지 않고도 잘 두는 법" — '두' 반복이 살짝 무거우나 한 호흡 안에 들어감.
- 라인 카운트: 정확 47행 (45-50 라인 룰 안).

## Unanswered for Conductor

1. **v6 → v8 thesis 이동** — v6 는 "다시 읽는 아린(독립 인물 회수)" frame, v8 은 user 지시대로 "함께가 아닌 따로 살아도 서로를 응원" frame. 둘 다 Phase 5 자장 안이지만 정서 톤이 다름 (v6=재독해 / v8=원격 응원). v8 thesis 가 episode_harness 의 "Arin 실제성 회수" 와는 약간 다른 각도 — user override 로 진행.
2. **"같은 하늘 아래" 보편성 점검** — 클리셰 경계 시어. female intimate alto 가 부르면 진부하지 않게 떨어질 수 있으나, Suno 청취 후 다른 보편 시어 ("같은 계절 안에서" / "같은 시간을 살면서") 로 교체 가능.
3. **chorus 표제 hook "잘 지내"** — 한국 청자에게 가장 보편적이지만 동시에 가장 닳은 표현. v8 craft 에서 hook 의 안전성 vs 신선도 trade-off 를 의도적으로 안전 쪽에 둠 (Phase 5 register 의 unsentimental 톤이 hook 의 보편성과 잘 맞음). 더 신선한 hook 원하시면 "너는 그쪽에서 잘 살아" 등으로 교체 검토 가능.
4. **bridge 7행 길이** — 8-12 권고 하한 미만이나 인식 전환 (헤어짐 → 응원) 이 7행 안에 닫힘. 추가 2행 필요시 "너를 위해 내가 비운 자리가 / 너에게 좁지 않기를 바라는 일" 류 삽입 가능.
5. **final chorus outro 흡수** — 별도 outro 라인 없음. final chorus 마지막 변형 라인이 outro 역할 겸함 (v8 craft 의 "final 1-variant lift" 룰).
