# Episode Song Prompt Sheet: 001-010

## use rule

- each episode gets one theme song
- each prompt should focus on one emotional function only
- no direct references to existing copyrighted songs or artists
- lyrics should prioritize image, season, air, light, distance, and after-feeling

---

## E001

- title anchor: 무료한 일상
- function: Seojun inner monologue
- emotional thesis: 세상은 시끄럽고, 나는 닫혀 있다
- chorus emotion line: 너무 많은 마음들 사이에서, 나는 내 귀부터 닫고 살았다
- lyrical taboo: 운명, 사랑 고백, 희망 과잉

### Suno prompt
```text
Korean modern emotional ballad, intimate male vocal, muted piano, soft noise texture, very restrained arrangement, late-spring campus air, gray-blue palette, emotionally overloaded but quiet, theme of shutting out the world with earphones, no explosive chorus, cinematic and subtle
```
