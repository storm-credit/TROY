## Calibration 가이드 (시트 받은 후 본 섹션 사용)

시트 이미지 받으면 다음 항목 체크:

- [ ] **얼굴**: 22세 한국 남대생 인상, 무뚝뚝/withdrawn baseline
- [ ] **머리**: short messy 검은 머리 + 약간 overgrown 앞머리 (NOT 길거나 styled)
- [ ] **이어폰**: 모든 view 에 wired white earphones 보임 (signature lock)
- [ ] **후드**: charcoal-gray oversized 일관 (모든 view)
- [ ] **표정 6개**: Neutral / Shut-out / Fatigued / Observant / Recoil / Residue 각각 식별 가능
- [ ] **의상 4개**: 메인 / 강의실 / 자취방 / 디테일 분리 표시
- [ ] **전신 3개**: 정면 / 측면 / 후면
- [ ] **포즈 5개**: 걷기 / 강의실 / 회피 / 천장 응시 / 식당
- [ ] **컬러**: Phase 1 muted 팔레트 (#E8F0F5 / #F5E1D0 / #7A9CB0 / #2C2E33)
- [ ] **톤**: 일러스트 (NOT photo, NOT 3D, NOT moe)
- [ ] **텍스트**: 한국어 primary + 영어 parentheses (NO Japanese)

문제 발견 시:
- 머리 길이 다름 → "더 짧게 / 더 헝클어지게"
- 이어폰 누락 → "every view 에 wired white earphone 강제"
- 표정 너무 밝음 → "smile baseline 제거, withdrawn 강조"
- 톤 너무 밝음 → "Phase 1 muted gray-blue 더 강하게"
- 일본 글자 등장 → "NO Japanese characters" 강조
- 일러스트 톤 안 맞음 (너무 anime / 너무 photo) → ART STYLE 섹션 톤 단어 조정

→ 이 calibration 결과를 v2 prompt 에 반영 후 재생성. v2 결과가 OK 면 sheet locked.
