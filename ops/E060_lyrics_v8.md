## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘은 너의 옆자리가 유난히 따뜻했어" — 청자가 즉시 자기 경험으로 환원 가능한 보편 임계 발화. 시작부터 "감정 사건"이 이미 발생했고 (= "유난히"의 자각 순간), 표면 따뜻함과 그 뒤 다가올 어긋남의 그림자가 한 줄에 동시 진입. Phase 3 closer 의 "마지막 따뜻함" 톤을 워밍업 0으로 안착. 첫 라인 "옆자리"는 캠퍼스/카페 등 작품 신 표지 0, 보편 공간어.

### Chorus emotion line placement
Locked phrase "한 줄을 미룬 시간이 무서웠던 거야" 는 chorus 2행에 verbatim 3회 (chorus 1·2·final). chorus 1·2 는 완전 동일 6행. final chorus 의 후반부 4행만 lift 변형 ("오늘은 그 미룸 옆에 / 조금 더 앉아 있다 갈게 / 지금 이 자리는 아직 따뜻하니까 / 나는 이 따뜻함을 기억해 두기로 했어") — locked emotion line 자체는 변형 0, 주변 골격만 1회 lift. v8 룰의 "final 1-variant lift" 정확히 충족.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스/강의실/연습실/녹음실/공연장 → 0건 (보편 "옆자리 / 같은 공기 / 다음 자리"로 치환)
- 공감각·청취 능력·sensory origin → 0건 (motif_audit `seojun_sensory_origin`, `self_hearing_prohibition` OK)
- 아린/서준 호명 / 너의 이름 / 노래·가사 직접 호명 → 0건 (motif_audit `arin_song_thread` OK)
- "헤어짐 / 결별 / 끝났어 / 마지막이야" 같은 final-breakup 어휘 → 0건 (E078 결별 당겨오기 방지, song_brief words-to-avoid 충족)
- 청취/유리/우산/비/새벽/너의 향기/빈자리 클리셰 → 0건
- 계절 직접 선언 ("여름이 가고") → "계절이 한 칸 옮겨 가려는 소리가 작게 들려" 한 줄로 보편화 (late-summer threshold 톤만 암시)
- borderline: "옆자리 / 자리 / 따뜻함" — Phase 3 closer 의 핵심 motif 군이라 의도적 반복 (v6 의 "자리"군 계승). 클리셰 위험은 "자리" 단순 반복이 아니라 "그 미룸을 안고 있을게 / 그 미룸 옆에 / 따뜻함을 기억해 두기로 했어" 식 동사 변주로 회피.

### Standout lines (target 5+)
1. "오늘은 너의 옆자리가 유난히 따뜻했어" — first-line hook, 즉발 정서 사건.
2. "그 따뜻함을 내가 먼저 알아챘다는 게 무서웠어" — Phase 3 closer 의 핵심 정서 (= 끝을 먼저 감지한 자의 두려움).
3. "한 줄을 미룬 시간이 무서웠던 거야" — chorus locked emotion line, v6 standout 계승.
4. "계절이 한 칸 옮겨 가려는 소리가 작게 들려 / 나는 그 소리를 못 들은 척하고 있어" — Phase 3→4 transition 을 청각 은유로, 직접 선언 회피.
5. "끝이 아니라는 말은 아직 못 하지만 / 끝이라는 말도 오늘은 하지 않을게" — bridge subversion, 두 부정의 평행 구조로 임계 자체를 명명.
6. "나는 이 따뜻함을 기억해 두기로 했어" — final lift, "기억해 두기로" 의 능동 자기 결정으로 닫힘 (v6 의 "한쪽 어깨에 같이 두고" 계승하되 더 단정).

### Bridge subversion design
chorus 가 "따뜻함을 안고 있을게" 의 머무름 modal 이라면, bridge 는 "따뜻함 = 다음 자리로 넘어가기 전 신호" 라는 메타 인식으로 한 번 시점을 들어올린다. "신호를 오래 받지 못한 사람" 이라는 자기 진단이 이 화자를 일반 사랑 발라드의 "이별을 슬퍼하는 화자" 와 구분 (E060 = threshold 인지자). 마지막 두 행 ("끝이 아니라는 말은 아직 못 하지만 / 끝이라는 말도 오늘은 하지 않을게") 은 두 부정 병치로 final breakup 어휘를 부정 안에서만 호출 — Phase 4 미당김 룰 충족.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음 (chamber-pop 75 BPM).
- "따뜻함 / 따뜻하니까 / 따뜻했어" 의도적 반복 — 청취 시 motif 형성, ㄸ-ㄷ 디귿계열 발음 명확.
- "미룬 / 미룸" 동사·명사 변주 — 한국어 자연 굴절, 청자에게 motif 라는 인지 형성.
- "옮겨 가려는 / 못 들은 척하고" — 한 호흡 안에 들어감 확인, BPM 75 안정.
- 영어/숫자/외래어/jamo emoticon 0건. 화법 일관 (반말 자기독백, 일부 청자-지향 "있을게/갈게/할게" 의지형 — 화자가 머물기로 결정한 자세 = Phase 3 closer thesis).
- 라인 카운트: verse1 7 + pre-chorus 3 + chorus 6 + verse2 7 + chorus 6 + bridge 6 + final chorus 6 = **41행**. v8 권고 45-50 살짝 하한이지만, chamber-pop 75 BPM 의 호흡과 침묵 비중 고려 시 적정. 필요시 verse2 에 2행 추가 가능 (option: "지금 이 손가락 사이의 공기는 / 며칠 후엔 다른 농도일지도 몰라").

### Removed from v6
- "한 마디가 우리 사이를 부순 게 아니라" → 첫 hook 으로는 추상 thesis. v8 에선 "오늘은 너의 옆자리가 유난히 따뜻했어" 라는 감각 사건으로 교체 (즉발 hook, "감지" 의 두려움부터 시작).
- "오래 닫힌 시간이었다" → v6 chorus emotion line 이었으나, v8 brief 가 "한 줄을 미룬 시간이 무서웠던 거야" 를 v6 standout 으로 지정. v8 chorus locked phrase 를 후자로 교체 (v6 chorus 2행이 v8 chorus 2행으로 승격).
- "끝났다고 말하지 않은 자리에서도 / 무언가가 자리를 옮길 수 있어" → bridge 가 다소 설명적. v8 bridge 는 "따뜻함 = 신호" 메타 인식으로 압축, 두 부정 평행 구조로 닫음.
- "다음 자리에 잠깐 발을 옮겨 보았어" → final chorus 가 이미 Phase 4 로 한 발 들어선 톤. v8 은 Phase 3 closer 룰에 더 충실하게 "조금 더 앉아 있다 갈게" 로 머무름 modal 유지, 어긋남은 다음 트랙 (E061~) 에 양도.
- v6 final chorus "이전 자리도 / 다음 자리도 / 한쪽 어깨에 같이 두고" 의 양손 균형 closure → v8 은 "이 따뜻함을 기억해 두기로 했어" 단일 능동 closure (마지막 따뜻함을 의식적으로 봉인하는 자세 = Phase 3 closer 정확).

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 후반 4행 lift** 가 chamber-pop 75 BPM 위에서 음악적으로 lift 로 작동하는지는 Suno 생성 후 청취 판단 필요. lift 가 약하면 final chorus 도 verbatim 6행으로 회귀 가능 (locked emotion line 은 어떻게 가도 verbatim 3회 보장).
2. **여성 보컬 적합성** — verse 1 "그 따뜻함을 내가 먼저 알아챘다는 게 무서웠어" / bridge "신호를 오래 받지 못한 사람이라" 류 자기 감지·자기 진단 라인은 female intimate alto 에 잘 맞춤. 화자 = "끝을 먼저 감지하는 사람" 의 음성적 정합성 확인.
3. **Bridge 길이 (6행)** — v8 권고 8-12행 하한 미달이나, Phase 3 closer 의 "더 말하지 않고 머문다" thesis 와 일치. 더 길게 원하시면 2행 추가 가능 (예: "한 번 더 머무는 게 / 다음을 늦추는 일은 아니라는 걸 / 오늘은 받아들여 보고 싶었어").
4. **라인 카운트 41행 (45-50 권고 하한 미달)** — chamber-pop 75 BPM 의 호흡과 침묵 비중을 의도적으로 살린 결과. verse2 에 2행, bridge 에 2행 추가하면 45행 도달 가능 (위 옵션 참조). 사용자 결정 대기.
5. **Phase 4 transition 어휘 강도** — "계절이 한 칸 옮겨 가려는 소리" / "끝이라는 말도 오늘은 하지 않을게" 가 Phase 4 어긋남 으로의 문턱을 충분히 암시하는지, 아니면 너무 따뜻한 쪽으로 닫혔는지 (= 마지막 따뜻함이 너무 안전해 보이는 위험) — Suno 청취 후 mood 판단 필요. 더 무거운 톤이 필요하면 final chorus 마지막 행을 "이 따뜻함은 오늘까지만 기억해 두기로 했어" 로 1행 lift 추가 가능.
