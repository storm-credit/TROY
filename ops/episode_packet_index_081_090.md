## 4. Function Balance

Song functions:

- Seojun inner monologue: E081, E083, E084, E085, E086
- relationship meaning reflection: E082, E089
- place / memory reflection: E087
- absence-world reflection: E088
- threshold reflection: E090

판단:

- E081-E085는 `Stage 4: bare silence`다. 아직 `침묵의 완성`이 아니다.
- E083-E085는 능력 없는 소통의 첫 훈련이지만, 성숙을 과장하지 않는다.
- E086-E087은 과거와 장소를 다시 듣는 것이 아니라 다시 해석하는 구간이다.
- E088은 아린을 유령화하지 않고, 부재를 안고 계속되는 생활권으로 다룬다.
- E089-E090은 재회 기대가 아니라 의미 정리와 다음 문턱이다.
