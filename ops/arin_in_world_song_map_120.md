# Arin In-World Song Map: 120 Episodes

## 1. Purpose

이 문서는 `아린의 실제 창작곡 / 실제로 부르는 곡`을 언제 사용할지 제한하는 배치표다.

이유:

- 아린 in-world song은 희소해야 힘이 생긴다.
- 너무 자주 쓰면 모든 화가 OST형 설명으로 흐른다.
- 아린의 예술성과 후반 이별의 잔향은 몇 개의 강한 지점에서만 터뜨린다.

## 2. Rule

아린 in-world song은 아래 조건 중 하나를 만족할 때만 사용한다.

- 아린의 내면이 서준의 청취로는 충분히 닿지 않는 순간
- 공연/녹음/창작 장면처럼 실제 diegetic music이 필요한 순간
- 후반부에서 사랑을 소유가 아니라 승인으로 바꾸는 순간
- 최종곡과 직접 연결되는 장기 회수 지점

## 3. Deployment Policy

| Range | Policy | Reason |
|---|---|---|
| E001-E020 | 사용하지 않음 | 아린을 아직 신비화하지 않고, 서준의 감지/관찰을 먼저 누적 |
| E021-E040 | 원칙적으로 사용하지 않음 | 고백과 연애 시작은 서준의 말하기 윤리 중심 |
| E041-E060 | 1회 후보 | 공개 연애/행복 정점 구간에서 아린의 음악 세계를 한 번만 열 수 있음 |
| E061-E080 | 1회 후보 | 균열/소실 구간에서 아린의 닫힘을 말 대신 음악으로 보여줄 수 있음 |
| E081-E105 | 1-2회 후보 | 이별 이후 재해석과 잔향 구간 |
| E106-E120 | 1회 강력 후보 | 최종곡/마지막 승인과 연결 |

## 4. Candidate Episodes

| Candidate | Use Type | Status | Note |
|---|---|---|---|
| E011 | 공연 장면 | rejected for now | 첫 공연이지만 아직 아린 창작곡으로 쓰기엔 이르다. 관계 테마로 처리 |
| E054 | 행복의 정점 | candidate | 황금기 정점에서 아린의 실제 곡이 처음 전면화될 수 있음 |
| E074 | 소실 심화 | candidate | 닫힘과 소실을 말보다 음악으로 보여줄 수 있음 |
| E091 | 우연한 재회 | possible echo | 재회가 아니라 잔향의 확인으로만 사용 가능 |
| E118-E120 | final connection | locked candidate | 최종곡 `너라는 계절`과 연결되는 마지막 승인 지점 |

## 5. Guardrails

- 아린 창작곡을 서준의 감정 설명 도구로만 쓰지 않는다.
- 아린의 곡은 아린의 주체성과 내면을 보호해야 한다.
- 실제 유명곡, 가수, 번안, 커버 프롬프트는 금지한다.
- diegetic song과 episode theme song이 같은 회차에 있을 경우, 문서에서 둘의 기능을 명확히 분리한다.
