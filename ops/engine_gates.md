# Engine Gates

## Gate 1. Canon Gate

질문:
- 이 회차가 잠긴 결말과 충돌하는가
- 인물의 성격과 엔드포인트를 흔드는가
- 세계 규칙을 새로 만들거나 무너뜨리는가

하나라도 yes면 진행 중지 후 총괄 판단.

## Gate 2. Structure Gate

질문:
- 이 회차는 120화 구조 안에서 기능이 분명한가
- 이전 화와 다음 화를 연결하는가
- 복선이 새지 않고 남는가

## Gate 3. Emotional Gate

질문:
- 이 화의 감정은 한 줄로 말할 수 있는가
- 감정이 설명이 아니라 장면으로 보이는가
- 여름-가을 계절 문법 안에서 어긋나지 않는가

## Gate 4. Song Gate

질문:
- 이 화의 곡은 한 감정만 맡는가
- 곡이 본문을 요약하는 대신 잔향을 만드는가
- 유명곡 복제 위험 없이 분위기만 번역했는가

## Gate 5. Visual Gate

질문:
- 컷 리스트가 감정 동선을 따라가는가
- 장소와 인물의 연속성이 유지되는가
- 이미지가 본문보다 과하게 설명적이지 않은가

## Gate 6. Prose Gate

질문:
- 음악 용어 남용 없이 감각 비유로 번역되었는가
- 설정어가 본문으로 새지 않는가
- 마지막 문장이 다음 화를 당기거나 잔향을 남기는가

## Final Gate. Director Sign-Off

총괄은 아래 셋을 확인한다.

1. 이 화는 엔진에 맞게 생산되었는가
2. 이 화는 전체 구조에 기여하는가
3. 이 화는 작품의 문학성을 깎지 않는가
