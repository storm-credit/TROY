## Rationale (post-script, not pasted to Suno)

### First-line hook design
"잘 안다고 생각했던 목소리에서 / 모르는 한 마디가 떨어졌어" — 안다고 믿은 대상에서 모름이 떨어지는 구조를 두 줄에 즉발. 한국 청자가 가족·연인·오랜 친구 누구로도 환원 가능한 보편 hook. 워밍업 0, 첫 줄에 이미 "감정 사건"이 일어난 상태. v6 의 "같은 노래를 듣고 있다고 믿었어" 패턴 (믿음 → 균열) 을 더 좁고 즉각적인 시청각 한 마디로 압축.

### Five standout lines (의도된 작품 안 펀치)
1. "처음 듣는 어조였어" — chorus emotion line 의 prefigure, verse 1 anchor.
2. "비슷함을 같음이라고 믿고 살았어" — v6 standout 보존 (verse 2 closer, 캐논 emotional grammar).
3. "이게 너인 줄 몰랐다는 게 / 가장 무거운 한 줄이었어" — chorus 의 self-indictment, 이별이 아닌 인식의 무게.
4. "내가 못 들은 너는 너가 아니었던 거야" — bridge 의 인지 전환.
5. "이게 너인 줄 이제야 안다는 게 / 가장 정직한 한 줄이었어" — final lift, "무거운"→"정직한" 한 단어 변형으로 final 1-variant lift 룰 충족.

### Bridge subversion
일반적 bridge 가 "그래도 너를 사랑해 / 우리 다시" 류로 회귀하는 것을 막고, **공포의 방향을 뒤집음**: "모르는 어조가 무서운 게 아니라 / 모르는 어조 안에 네가 있었다는 게 무서워". 이별 가속이 아니라 **자기 듣기의 폭** 으로 책임을 수렴 — phase 4 rupture 안에서도 "너의 잘못이 아니라" 라인으로 비난-회피, deferred ending 유지. 캐논 motif "Heard vs Said" / "Self-Hearing Prohibition" 결을 외부 가사로 자연 표출.

### Final 1-variant lift
chorus 마지막 두 행만 변형: "가장 무거운 한 줄이었어" → "가장 정직한 한 줄이었어". 같은 어간 (한 줄이었어) 유지, 형용사 한 단어만 교체 — "무거운(짐)"에서 "정직한(받아들임)"으로 phase 4 rupture 의 미세한 lift 작동. 또한 "오래 서 있던 그 한 줄에서 / 오늘은 천천히 한 발 내려와 봤어" 행이 첨가되어 정지에서 미세 동작으로 이행. lift 가 약하면 verbatim chorus 로 회귀 가능.

### Universal-imagery audit
- 작품 신 직접 묘사 0건: 카페/거리/이어폰/네온/유리 → 모두 제거. v6 의 "이어폰 한쪽 헐거워진" / "번진 네온이 빗물 위에서" 류 보편 시어조차 제거하고 **청각 추상 (어조·톤·음·한 마디)** 만으로 운용.
- 능력/공감각/sensory origin 직접 묘사 0건 (motif_audit_self_hearing_prohibition / seojun_sensory_origin OK).
- 결별 확정 어휘 0건: "끝", "이별", "마지막", "안녕" 미등장. "넘어가지도 못하고" / "한 발 내려와 봤어" 로 deferred ending 유지 (E078 결별 가속 방지).
- 클리셰 (비/새벽/우산/너의 향기/빈자리) 0건. "빈자리"는 1회 (verse 2) 사용되지만 **청각 빈자리** (한 번도 안 쓰던 톤의 빈자리) 로 재맥락화 — 클리셰 회피.
- "공기를 굳혀 놓았어" — 일반 시어, motif 와 무관 보편 정서.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음 (folk-electronic BPM 82 에 안전).
- "어조 / 어긋난 / 어조였어" 의 ㅓ계열 의도적 반복 — 청취 motif 형성, 발음 명확.
- "같음 / 같은 / 같다" 디귿계열 반복 (verse 1·2, chorus) — v6 의 "같은 노래" 결을 한 음절 단위로 응축.
- "다른 무게로 떨어졌어" — 한 호흡 안에 들어감 확인. 폭/넓음/좁음 류 단음 단어 균형 OK.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백). 수치·날짜·시각 0건.
- 자모 emoticon (ㅎㅎ/ㅋㅋ/ㅠㅠ) 0건. 스테이지 디렉션 () 0건.

- "이어폰 헐거워짐" / "네온 두 번 흔들림" / "빗물 위" → 청각 추상으로 전환 (보편화 강화).
- v6 standout "비슷함을 같음이라고 믿고 살았어" 는 verse 2 closer 로 verbatim 보존 — v6 의 정수 한 줄을 v8 에 그대로 이식.
- v6 bridge "끝이라고 말하지 마" → "모르는 어조 안에 네가 있었다는 게 무서워" 로 교체 (subversion 강화, 결별 어휘 회피).
- v6 final chorus "이건 다른 종류의 슬픔이야" → "가장 정직한 한 줄이었어" 로 교체 (슬픔 명명 → 인식 명명, 보편성 ↑).
