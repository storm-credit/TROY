## Rationale (post-script, not pasted to Suno)

### First-line hook design
"겨울이 오기 전 가장 조용한 빛이 있어" — Phase 6 opener thesis ("겨울 초입의 빛이 가장 조용한 자리에서 마음을 비춘다") 가 한 줄에 압축. "있어" 평서형 발화로 워밍업 0, 청자가 즉시 '그 빛'을 자기 기억에서 꺼내게 됨. 보편 시어 "조용한 빛 / 창틀" 만 사용, 캠퍼스/문학관/계단 모티프 0건.

### 5 standout lines
1. "겨울이 오기 전 가장 조용한 빛이 있어" — first-line hook, Phase 6 thesis 압축
2. "따뜻한 것도 아니고 차가운 것도 아닌 / 조금 더 정직해진 온도 같았어" — 회복 narrative 거부, livable register
3. "한 발은 멈춘 채로 한 호흡은 길게" (chorus 1행) — v6 standout 잔류, chorus_emotion_line 으로 승격
4. "괜찮아진 게 아니라 / 괜찮지 않아도 살 수 있게 된 거였어" — bridge subversion 핵, Phase 6 livable uncertainty 정의
5. "오늘은 비춰지는 쪽으로 한 뼘만 기울어 보기로 했어" — final-chorus 1-variant lift, "되어 보기로 했어" → "기울어 보기로 했어" 미세 변형

### Bridge subversion
"괜찮아진 게 아니라 / 괜찮지 않아도 살 수 있게 된 거였어" — 회복·치유·재시작의 일반적 closure 어휘를 거부. Phase 6 의 "침묵의 완성 / livable uncertainty" 를 직접 정의. 청자가 chorus 의 "다 끝났다고 / 다시 시작한다고" 양극을 모두 buffer 받았다가 bridge 에서 제3의 자리("닫지 않은 채로 / 조용히 겨울 쪽으로 걸어가")로 떨어지게 함. closure-by-reunion 회피 lock.

### Final chorus 1-variant lift
chorus 1, 2 의 마지막 행 "오늘은 비춰지는 쪽이 되어 보기로 했어" → final chorus 의 동일 위치 "오늘은 비춰지는 쪽으로 한 뼘만 기울어 보기로 했어" 로 단 1회 변형. "되어" (state) → "기울어" (motion·micro) 로 작은 운동을 부여하되 "한 뼘만" 으로 진폭을 잠금 — triumph swell 회피, Phase 6 livable closure 안에서의 lift. 변형 라인 1회 룰 준수.

- 작품 신 직접 묘사 0건: 캠퍼스 / 문학관 / 계단 / 낙엽 / 실루엣 / 두 사람 마주침 → 전부 제거. 보편 시어 "창틀 / 손등 / 손바닥 / 오후" 로 치환.
- forbidden cliché (destiny / finally / hearing-again / running-back / 운명 / 마침내) 0건.
- 공감각·청취능력·sensory-origin motif 0건 (motif_audit_seojun_sensory_origin / heard_vs_said 두 건 모두 OK).
- 재결합·해피엔드 예고·능력 회복 단어 0건 (closure-by-reunion lock 통과).
