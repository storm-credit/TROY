# Manuscript Revision Pass 01

## 1. Goal

초고 완료 이후 `완성본 1차 정리`를 수행한다.

우선순위:
- 파일명과 원고 단계 표기를 `정본초고` 체계로 통일
- 1악장부터 문체 밀도와 연결부를 안정화
- 후반부 엔딩 여운은 마지막에 한 번 더 통합 보정

## 2. Pass Order

### Pass A. Chapter 1 normalization

- E001-E020 파일명/문체/호흡 통일
- opening sentence mode와 ending residue 정렬
- 서준 시점의 거리감, 이어폰, 캠퍼스 공기 표현 재확인

### Pass B. Chapter 2-4 bridge cleanup

- E021-E080 장면 사이의 감정 급변 완화
- 도현, 유빈, 소나 개입 구간의 비중 재균형
- 여름에서 장마, 장마에서 가을로의 계절 증거 보강

### Pass C. Chapter 5-6 ending polish

- E081-E120 해석/성장/승인 축을 더 선명하게 정리
- E118-E120 장면 간 호흡을 한 톤으로 묶기
- `침묵의 완성`이 관념이 아니라 감각으로 남도록 보정

## 3. Started Items

- E001 파일명 `리라이트 -> 정본초고` 정규화 완료
- E002 파일명 `리라이트 -> 정본초고` 정규화 완료
- TROY 진행 로그와 외부 재작성 상태 문서 동기화 완료

## 4. Next Target

- E011-E020 첫 공연/체육대회 구간 리듬 보정
