## Brief rationale (post-script)

- First-line hook: "오늘은 인사를 어떻게 해야 할지 한참을 골랐어" — 사후-confession 의 어색함을 즉시 보편화. 누구나 겪어본 작은 망설임.
- Chorus lock: "기쁨도 처음이라 / 우리는 조금 삐걱거리며 웃었어" verbatim 3회. 브리프 "-었다" → "-었어" 통일.
- Universal-imagery audit: "perfect / destiny / couple forever / solved" 0건. "복도 / 어깨 / 발소리" 보편. "two cups remembered" 같은 직접 신 호출 회피 — 추상 ("어제와 다른 자리").
- Phase 2 register: shy and warm, joy learning to stand. "완벽하지 않아서 다행이라고" 가 stable happiness 가 아닌 unpracticed joy 표명.
