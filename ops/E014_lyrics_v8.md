## Rationale (post-script, not pasted to Suno)

### First-line hook design
"좋아하는 노래가 뭐냐고 먼저 물어 놓고 / 정작 대답이 올 때 시선을 떨궜어" — 누구나 한 번은 해본 어색한 질문 + 시선 회피의 신체 디테일. 시작부터 "감정 사건"이 이미 발생한 상태이고, 호기심과 부끄러움이 한 행에 동시에 들어감 (검정치마 「강아지」 / 신해경 「우리는 작은 별」 류 부드러운 자기풍자 hook). 워밍업 0. v6 의 hook 보다 두 행으로 분할하여 album-track 호흡(BPM 72)에 더 안정적으로 안착.

### Chorus emotion line placement
Locked phrase "그 노래가 너의 자리였구나" — chorus 1 첫 행, chorus 2 첫 행, final chorus 첫 행 + 마지막 행 lift 로 총 4회 verbatim. 1-variant 룰은 사용 안 함 (album-track 폴리시: 변형 없이 같은 구절을 lift 위치에 한 번 더 떨어뜨려 종결의 무게를 만든다). final chorus 마지막 행 반복은 Phase 1 의 조심스러움이 마침내 inner truth 로 번지는 작은 깨달음 — 음악적으로는 outro 없이 chorus 안에서 자연 종결.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 카페/강의실/캠퍼스/햇빛/창가 → 0건. 보편 "방", "문 앞", "손잡이", "공책 모서리", "혀끝", "집"으로만 구성.
- 가수명/곡명/장르/음악이론 단어 → 0건 ("멜로디", "한 소절", "가사", "박자" 만 일반 시어로 사용 — song brief 의 forbidden 리스트 통과).
- 아린/서준 직접 호명 → 0건 ("나/너" 인칭만 사용, Phase 1 자기독백 register).
- 공감각·청취 능력·sensory origin → 0건 (motif_audit_arin_song_thread.md 의 "future edits should protect that scale" 권고 준수 — 노래를 destiny proof / reunion banner 로 부풀리지 않음).
- 봄/계절 직접 선언 → 0건 (E014 phase preset 의 "봄/햇살" 시즈 의도적 배제 — shy conversation 의 internal time 만 유지).
- 클리셰 (비/새벽/우산/너의 향기/빈자리/운명) → 0건.
- borderline: "얇은 다리" — pre-chorus 의 metaphor. 클리셰 아님, song brief 의 "two people talking about music while actually talking about loneliness" 정서를 한 행에 압축.
- 도구 직접 호출 (이어폰/스피커/플레이리스트 앱) → 0건. "곡 목록을 만들어 본 적이 있었어" 는 추상 행위로만 처리.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음 (album-track 의 차분한 호흡과 BPM 72 에 부합).
- "흥얼거렸을", "만지작거렸어", "굴려 봤어", "흩어지는" 류 부드러운 ㅎ/ㅁ/ㄱ 흐름 — female intimate alto 발성에 안정.
- "모서리 / 자리 / 소절 / 결" 의 'ㅓ리/절/결' 라임이 의도적으로 차분한 호흡 motif 형성.
- "닮아 / 닿지 / 들어두고" 디귿계열 부드러운 반복 — 청취 시 motif 형성, 발음은 명확.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백, Phase 1 register).
- "좋아하는 노래가 뭐냐고 먼저 물어 놓고" — 한 호흡 안에 들어감 확인 (BPM 72 에서 11자, 안정).
- "이름이 없을 때만 들리는 결이 있어서" — 미묘한 라인. "결이 있어서" 발음이 "겨리써" 로 미끄러질 수 있으나 female alto 의 부드러운 종결 음역에서 안정적.
- 자모 emoticon (ㅎㅎ/ㅋㅋ/ㅠㅠ) 0건. stage direction 0건.
- 화자 일관: 1인칭 자기독백, 시제는 과거 + 잠정 현재 혼용 (Phase 1 회상 register).

### Standout-line lift (album-track 5선)
v8 album-track polish 에 요구되는 "5 standout" 라인:
1. "정작 대답이 올 때 시선을 떨궜어" — verse 1, hook 보강. 호기심과 회피가 동시에 작동.
2. "음악 이야기라는 얇은 다리 위에서 / 서로의 모서리를 한 칸씩 보여주는 자리" — pre-chorus, song brief 의 thesis 를 두 행으로 압축한 album-track signature.
3. "이름이 없는 채로 들어두고 싶은 곡이 있어서" — verse 1 종결, chorus 의 "이름" 모티프를 미리 심는 anchor.
4. "혀끝에서 사라지는 게 어울리는 노래도 있더라" — verse 2, "오래 남기지 않음의 미덕" 이라는 album-track 만의 성숙한 여백.
5. "대신 너의 한 소절을 데리고 집에 왔어" — bridge, "음악을 받아 가는" 이라는 album-track 의 closing image. v6 에 있던 라인을 그대로 살리되 다음 행 ("그게 답장이 아닌 줄 알면서도 / 오래 그 소절 옆에 앉아 있었어") 로 두 행 더 깊이 들어감.

### Bridge subversion design
v6 bridge 는 "보내지 못한 한 곡" 으로 끝났음 (행위의 부재). v8 bridge 는 "보내지 못함" 을 그대로 두되 마지막 두 행 ("그게 답장이 아닌 줄 알면서도 / 오래 그 소절 옆에 앉아 있었어") 으로 album-track 다운 inner act 를 추가 — 침묵이 표현이라는 사실을 자기 자신에게 처음 인정. final chorus 진입 직전, 이 한 줄이 chorus 의 "그 노래가 너의 자리였구나" 를 새로운 무게로 받게 한다 (단순 호감 → 자리잡힘의 발견).

### Removed / changed from v6
- v6 verse 1 첫 행 단일 → v8 두 행 분할 ("먼저 물어 놓고" + "정작 대답이 올 때 시선을 떨궜어"). album-track 호흡과 hook 폭발력 동시 보강.
- v6 verse 1 4행 "나는 늘 문 앞에서만 서성였어" → v8 "나는 늘 문 앞에서 손잡이만 만지작거렸어". "서성였어" 보다 손의 미세한 행동으로 더 구체화 (회피의 신체화).
- v6 pre-chorus 3행 "음악 이야기를 빌려서" → v8 "음악 이야기라는 얇은 다리 위에서". "빌려서" 의 평이함을 "얇은 다리" metaphor 로 album-track polish 강화 (song brief 의 "two people talking about music while actually talking about loneliness" 정서 압축).
- v6 bridge 6행 → v8 8행. "그게 답장이 아닌 줄 알면서도 / 오래 그 소절 옆에 앉아 있었어" 두 행 추가 — bridge subversion 의 inner-act lift.
- v6 final chorus 5행 → v8 6행. 마지막 행 "그 노래가 너의 자리였구나" 반복 추가 — chorus emotion line 의 자연 종결 (변형 없이 verbatim 4회로 종결).
- v6 chorus "한 소절이 너를 닮아 있어서" 그대로 유지 — v8 standout 으로도 작동.
- v6 verse 2 "혀끝에서 사라지는 게 어울리는 노래도 있었어" → v8 "있더라" (반말 자기인정 톤으로 미세 조정 — Phase 1 자기독백의 부드러움 강화).
- v6 verse 2 "그날 너의 말 끝에 따라온 작은 웃음이었어" → v8 "그날 너의 말끝에 따라온 작은 웃음이었어" (띄어쓰기 단일화로 발음 흐름 안정).

## Unanswered for Conductor

1. **chorus emotion line 4회 verbatim 종결 결정** — album-track 폴리시로 1-variant lift 사용 안 함. final chorus 의 마지막 lift 행을 변형 ("그 노래가 너의 자리가 됐어" 류) 으로 가고 싶다면 Suno 생성 후 청취 비교 필요. 현재 디자인은 verbatim 의 무게가 album-track 의 종결감에 더 적합하다고 판단.
2. **female intimate alto vs duet-like intimacy 결정** — song brief 는 "duet-like intimacy, but one vocal is enough" 이라 명시. v8 은 single female alto 기준으로 작성. duet 으로 전환 시 chorus 1·2 의 "그 노래가 너의 자리였구나" 한 행을 male soft harmony 로 겹치는 것이 자연스러움 — 가사 변경 없이 mix 단계에서 처리 가능.
3. **bridge 길이 (8행)** — v6 6행 → v8 8행. 권고 하한 (8-12) 안착. 더 길게 가져가면 "왜 한 곡도 못 보냈는지" 이유 한 행 추가 가능 ("나의 약함을 너에게 보내는 게 아직은 어색해서") 하지만 album-track 의 절제된 inner-act 미덕에서 멀어질 수 있음 — 현재 8행 유지 추천.
4. **outro 없음 결정** — final chorus 마지막 행 "그 노래가 너의 자리였구나" 반복으로 자연 fade. 별도 outro 추가 시 ("이름이 없는 채로 / 오래 들었어" 류 2행) 가능하나 album-track 의 "조용한 마침" 미덕에서 약간 늘어진다고 판단 — outro 별도 행 없이 chorus 종결 추천.
