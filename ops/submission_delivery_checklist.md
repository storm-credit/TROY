# submission delivery checklist

## 1. purpose

- 이 문서는 실제 외부 전달 직전에 빠르게 확인할 체크리스트를 잠근다
- 목표:
  - 잘못된 파일을 보내거나, showcase 없이 full manuscript만 보내거나, 반대로 전체 원고 없이 샘플만 보내는 실수를 막는다

## 3. checklist

- 보낼 목적이 무엇인지 확인
  - sample showcase only
  - full manuscript review
  - alpha feedback request
- 파일 묶음이 목적과 맞는지 확인
  - showcase만 보낼지
  - showcase + full manuscript를 같이 보낼지
- 시작 문서가 포함됐는지 확인
  - `00_START_HERE.md`
- 평가 축 안내가 필요한지 확인
  - `01_EVALUATOR_NOTE.md`
- 피드백을 받고 싶다면 아래 포함 여부 확인
  - `02_ALPHA_READER_FEEDBACK_GUIDE.md`
  - `03_FEEDBACK_TEMPLATE.md`
  - `04_FEEDBACK_TRIAGE_NOTE.md`
- 원고 기준 파일이 최신인지 확인
  - `너라는운율_submission_draft_v1.md`

## 4. director note

- 전달 직전에는 새 수정 금지
- 보내는 목적과 파일 묶음의 일치만 확인하면 된다
