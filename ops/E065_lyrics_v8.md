## Rationale (post-script, not pasted to Suno)

### First-line hook design
"같은 말인데 어딘가 한 칸 낮게 들렸어" — 한국 청자 누구나 즉발 인식 가능한 청취 경험 (안부 한 마디의 음높이가 미세하게 달라진 자리). genre JSON 의 "감정 톤이 minor key 로 가라앉는 첫 본격 어긋남" 을 첫 줄에 음악 어휘 ("한 칸 낮게") 로 박아 둠 — 90s 발라드의 self-aware 음악 메타포 전통 (조성모 · 신승훈 류) 안에서 자연스러움. 워밍업 0, 정서 사건이 이미 발생한 채로 시작.

### Chorus emotion line placement
Locked phrase "같은 말인데 톤이 달라졌어" — chorus 1·2 verbatim 2회, final chorus 도 동일 라인으로 시작 (총 3회 verbatim 충족). 변형 1회 허용은 final chorus 의 마지막 lift "나는 한 칸 낮은 자리에 가만히 앉아 있었어" 로 사용 — chorus 의 "한 칸 낮음" 메타포가 verse 1 첫 줄을 회수하면서 화자가 그 낮음을 거부하지 않고 받아들이는 자리 (Phase 4 의 livable melancholy). lift 가 약하면 verbatim "음 하나가 반음 내려간 것 같아" 로 회귀 가능.

### 5 standout lines
1. "같은 말인데 어딘가 한 칸 낮게 들렸어" (verse 1 · 첫 hook · 음악 메타포)
2. "음 하나가 반음 내려간 것 같아" (chorus · 보편 + 음악적 정확)
3. "헤어진 적은 없는데 / 헤어지는 법을 미리 배우고 있는 것 같았어" (verse 2 · Phase 4 thesis 정확 압축)
4. "큰일이 없어서 더 무서웠어 / 큰일이라면 붙잡을 수라도 있었을 텐데" (bridge subversion · "어긋남 = 작은 일" 의 진실)
5. "나는 한 칸 낮은 자리에 가만히 앉아 있었어" (final lift · 화자가 minor 를 받아들이는 순간)

### Bridge subversion
v6 bridge 는 "같은 시계를 본 적이 없는지도 몰라" — 시간차 thesis 의 메타 인식. v8 은 "사랑은 끝나는 게 아니라 / 조금씩 음을 잃어 가는 일" 로 재작성 — 끝/안 끝 이분법을 무너뜨리는 정의 변경 (subversion). 거기에 "큰일이 없어서 더 무서웠어" 라는 reverse-logic 한 줄을 박아 청자가 이 곡의 thesis 를 음악 어휘 안에서 다시 듣게 함 — Phase 4 의 핵심 (재난이 아닌 색의 침잠) 압축.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 엘리베이터/횡단보도/카페 문 (v6 의 도시 모티프) → 제거. 보편 "안부", "사진", "노래", "마디" 로 치환.
- 캠퍼스/이어폰/공감각·청취 능력 → 0건.
- 클리셰 (비/우산/새벽/너의 향기) → 0건.
- 결별 어휘 ("헤어졌어", "끝났어", "이별") → 회피. "헤어지는 법을 미리 배우고 있는 것 같았어" 한 줄로 Phase 4 의 "여전히 함께 / 이미 멀어지고 있음" 텐션 충족.
- 음악 메타포 ("한 칸 낮게", "반음 내려간", "마디", "곡", "음") → 5건 — 90s vintage ballad 의 self-aware 전통 안에서 의도된 motif. 곡 자체가 곡에 대해 말하는 구조 = lane 적합도 최상.
- 색 메타포 ("색이 달라졌어", "색이 빠진 사진", "회색") → genre JSON 의 lyricImageSeed "잃어가는 색" 직접 회수.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "달라졌어 / 달라졌어 / 달라" 디귿계열 의도 반복 — 청취 motif 형성, 발음 명확.
- "한 칸 낮게 / 한 박자 / 한 칸 낮은" 수량 어휘 반복 — 음악 메타포 anchoring, 발음 안정.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백, female intimate alto 적합).
- "음 하나가 반음 내려간 것 같아" — 한 호흡 안 안정, BPM 72에서 여유 있음. "반음" 은 음악 어휘지만 보편 한국어 (학교 음악 시간 어휘) — 클리셰 아님, 정확함.
- "끝의 마침표가 무거웠어" — "끝의 마" 자음 클러스터 약간 있으나 한 박자 안 처리 가능.

## Removed from v6

- v6 first hook "한 박자 늦게 도착한 답장 위로 하루가 지나갔어" → timing-mismatch 외피였음. v8 은 thesis 를 "톤 가라앉음" 으로 재정렬 → "같은 말인데 어딘가 한 칸 낮게 들렸어" 로 교체.
- "엘리베이터의 문이 닫히는 동안 / 너의 윤곽만 잠깐 보였어" → 도시 모티프 직접. "음 하나가 반음 내려간 것 같아" 로 교체 (음악 메타포 anchor).
- "횡단보도 반대편에서 너를 본 것 같았어" → chase-borderline. 제거 후 "괜찮다고 답한 끝이 평소보다 낮았어 / 나는 그 낮음을 못 들은 척 넘겼어" 로 교체 — 화자 자기 행위 안에서 정서 작동.
- v6 bridge "같은 시계를 본 적이 없는지도 몰라" → 시간차 thesis 잔존. v8 bridge 는 "사랑은 끝나는 게 아니라 / 조금씩 음을 잃어 가는 일" 정의 재작성.
- v6 final chorus "헤어진 건 아니었어 / 다만 만나는 법을 잠시 잃어가고 있었어" → "만나는 법" 은 timing 어휘. v8 은 "한 칸 낮은 자리에 가만히 앉아 있었어" 로 교체 — minor 수용의 자리.
- 50 라인 안에 outro 별도 행 압축 (final chorus lift 흡수).

2. **음악 메타포 밀도** — 5건 ("한 칸 낮게", "반음", "마디", "곡", "음을 잃어 가는") — 90s vintage ballad 의 self-aware 전통에 적합하지만, 너무 메타적이라고 느껴질 경우 verse 2 의 "색이 빠진 사진" 라인을 강화하고 음악 어휘 1건 (verse 2 의 "낮은 자리") 을 색/공간 어휘로 치환 가능.
3. **Bridge 길이 (6행)** — 의도적 짧음. "큰일이 없어서 더 무서웠어" 의 reverse-logic 이 self-contained 라 더 길게 풀면 정서 농도 떨어질 위험.
4. **chorus 종행 "같은 마디를 쥐고 있지 않은 것 같아" vs final chorus 의 "쥐고 있지 않아서"** — 자연스러운 종결 변형, verbatim lock 위반 아님 (조사 변경 only). Suno 가 동일 멜로디로 처리할지 청취 확인 필요.
5. **timing-mismatch thesis 흡수 정도** — v6 의 timing 모티프는 v8 의 "한 박자 짧아졌고 / 한 박자 길어졌어" pre-chorus 한 쌍에만 흡수. 더 명시적으로 timing 을 살리고 싶다면 verse 2 의 후반 2행을 timing 라인으로 교체 가능 (단, thesis 는 "톤 가라앉음" 으로 유지).
