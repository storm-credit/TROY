## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘은 같은 길을 조금 더 천천히 걸었어" — 한국 청자가 즉시 자기 경험으로 환원할 수 있는 보편 행위. "기다리는 골목"이라는 harness 시드를 직설적으로 부르지 않고 행동으로 번역 (천천히 걸음 = 우연을 만들고 싶은 무의식). 시작부터 정서 사건이 이미 작동 중. 워밍업 0행. 검정치마/이상은 류의 평범한 발화 안에 균열 패턴.

### 5 standout lines (의도적 배치)
1. "약속이 없는데 시간을 비워 두는 마음" — verse 1 mid. 우연 대기의 정의.
2. "오늘이 채워질까 봐 무서웠어" — pre-chorus 종결. 두려움의 방향 반전 (비어서가 아니라 채워질까 봐).
3. "한 번 더가 무서운 거야 / 한 번 더가 기다려지는 거야" — v6 user-locked couplet, chorus 핵.
4. "이름을 부르지 않은 사람을 떠올릴 때 / 그 사람의 얼굴이 더 또렷해지는 게 이상해" — verse 2 hook. 미명명의 또렷함.
5. "기다린다고 말하면 너무 큰 말이 되고 / 안 기다린다고 말하면 거짓말이 되고" — bridge subversion 진입선.

### Chorus emotion line placement
v6 user-protected couplet "한 번 더가 무서운 거야 / 한 번 더가 기다려지는 거야"가 chorus 3·4행에 verbatim 3회 (chorus 1·2·final). final chorus 마지막 lift 라인 "한 번 더를 오늘은 그냥 두기로 했어"는 1-variant 허용 사용 — 같은 "한 번 더" 어구를 결정 동사로 닫음 (음악적 안착). chorus lock ≥2회 룰 충족 (verbatim 3회).

### Bridge subversion
이전 chorus는 "두 마음이 같은 자리"의 진동을 노래했지만, bridge는 진동의 언어 자체를 거부 ("기다린다 / 안 기다린다" 둘 다 거짓). 정서를 더 끌어올리는 대신, 작명을 보류하기로 결정 — "이게 시작인지 아닌지조차 / 나는 아직 정하지 않기로 했어". v6 bridge ("가까운 게 위험해진다는 걸")의 깨달음 톤을 의도적으로 떼어내고, 미정의 자기 결정으로 대체. final chorus가 lift 가능해지는 토대.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 운동장/공/체육대회/물병/소매 → 제거 (v6 "손끝이 스친" 사건 자체를 배경화)
- 캠퍼스/강의실 직접 호출 → 0건 ("같은 길", "골목", "신호등"으로 보편화)
- 공감각·청취 능력·sensory origin → 제거 ("바깥 소리가 멀어지는" 류 sensory 라인 0건)
- 클리셰 (비/새벽/우산/너의 향기/계절 직선언) → 0건
- magic/destiny/electricity/soulmate → 0건 (song_brief 금지어 lock)
- borderline: "잠이 한 번 깨" — v6 verbatim 라인. 보편 시어로 안전, motif 지속.
- harness lyricImageSeeds 4개 활용: "기다리는 골목" → "같은 골목을 한 번 더 지나갔어"; "지나가는 사람들" → "누가 지나갈 때마다 고개가 먼저 돌아갔어"; "한 번 더의 우연" → chorus 핵; "잠 깬 새벽" → "오늘 밤은 잠이 한 번 깨".

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "한 번 더" 어절 반복은 의도적 motif (chorus 3회 verbatim + final lift 1회 = 총 4회 등장). 발음 명료, 78 BPM 한 호흡 안.
- "기다린다 / 안 기다린다 / 기다려지는" 디귿계열 motif — 청취 시 의미 진동 형성, 발음 안전.
- "신호등 앞에서 한 박자 더 머물렀어" — "ㅎ박자ㄷ머" 자음 약간 밀집하나 78 BPM에서 안정. 한 호흡 통과 확인.
- 영어/숫자/외래어/jamo emoticon 0건. 화법 일관 (반말 자기독백, 여성 intimate alto).
- 라인 길이 평균 14-18 음절, 78 BPM 한 호흡 안 안정.

### Removed from v6
- "스쳤다고 부르기엔 너무 또렷한 한 점이 있어" → 사건(접촉) 직접 호출. v8은 사건 이후의 대기 register로 전환.
- "손끝이 스친 뒤로 소란이 잠깐 멀어졌어" (chorus lock) → 사건 + sensory ("소란이 멀어졌어" = 청취 metaphor) 직접. v8 chorus는 user-protected couplet으로 재축조.
- "그 자리만 따로 그어진 것처럼 / 다른 소리가 들리지 않았어" → sensory + magic-adjacent. 제거.
- "가까운 게 위험해진다는 걸 / 오늘 처음 알게 된 것 같아" → bridge가 깨달음으로 닫힘. v8은 미결정 상태로 lift 진입을 위해 재작성.
- final chorus "다시 시끄러워진 자리에서도 / 그 한 점만큼은 조용한 채로" → sensory metaphor. v8 final은 결정 보류 라인으로 대체.
- v6 verse 2 "오늘 밤은 잠이 한 번 깨" 1행만 chorus 마지막에 흡수 (motif 보존).
- 라인 수 v6 = 약 35행 → v8 = 47행 (45-50 lock 충족).

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 마지막 "한 번 더를 오늘은 그냥 두기로 했어"** 가 음악적 lift 로 작동하는지는 Suno 생성 후 청취 판단. lift 약하면 v6 verbatim 4회 ("한 번 더가 기다려지는 거야"로 종결)로 회귀 가능.
2. **여성 intimate alto 적합성** — verse 1 "가방 안의 손은 자꾸만 움켜쥐었다 폈고" 류 자기관찰 라인은 female alto withdrawn 톤에 맞음. 78 BPM korean-indie-folk lane에서 한 호흡 안 안정 확인 필요.
3. **Chorus 6행 vs 5행** — v6은 chorus 5행, v8은 6행 ("두 마음이 같은 자리에 서 있어서 / 오늘 밤은 잠이 한 번 깨"). 호흡 부담 시 마지막 1행 제거 가능.
4. **Phase 1 closure** — v8은 "한 번 더를 오늘은 그냥 두기로 했어"로 마침. Phase 1 의 "닫힌 듯 열린" 톤 유지 — 결정 미보류 = 다음 화로 자연 전이. E019 "닫힌 듯 열린 마음의 첫 균열"과 음악적 인접.
5. **사용자 이전 v6 chorus lock 의 보존 여부** — v6 chorus는 "손끝이 스친 뒤로 소란이 잠깐 멀어졌어"였으나 user 지시는 "한 번 더가 무서운 거야 / 한 번 더가 기다려지는 거야" couplet 보존. v8은 후자를 chorus 핵으로 승격 — 만약 전자도 어딘가에 잔존시켜야 한다면 verse 2 의 "스친 적 있는 시간은 무게가 다른 것 같아" 라인을 "손끝이 스친 시간은 무게가 다른 것 같아" 로 대체 가능 (하지만 작품 신 호출 위험 재발).
