## Rationale (post-script, not pasted to Suno)

### First-line hook design
"사람이 너무 많은 자리에 처음으로 한 사람이 보였어" — 군중 속 외로움이라는 보편 정서를 깔자마자 "한 사람이 보였다"는 반전이 같은 줄 안에 충돌한다. 워밍업 0행, 첫 줄이 곧 emotional event 의 발생지. v6 은 "아무도 안 보이는 줄 알았어" 까지 한 박자 끌었지만, v8 은 그 부정 단계를 압축해서 정서 즉발. 검정치마 「강아지」 류의 평발화 안에 균열을 심는 패턴.

### Chorus + lift design
Locked phrase "수많은 소리 사이에서 너 하나만 또렷이" 는 chorus 1·2·final 모두 1행 verbatim 으로 3회 (이중 lock 안전 보장). final chorus 의 마지막 라인만 "한 호흡만 길게 남았어" → "한 사람의 윤곽만 남아 있었어" 로 자연 변형 — 동일한 "남다" 어간을 잔향에서 윤곽으로 옮기며 phase 1 종결 register 안에서의 미세한 lift (huper-uplift 아님). 변형 1회만 허용된 룰 안에서 사용.

### Universal-imagery audit
- 캠퍼스 / 공연장 / 무대 / 마이크 / 조명 / 동아리 → **0건** (v6 의 "무대 끝의 한 줄 빛"은 "누군가의 어깨에 잠깐 머문 한 줄의 빛"으로 보편화 — 무대 직접 호명 제거)
- 이어폰 / 음향 차단 도구 → 0건
- sensory origin / 공감각 / 청취 능력 직접 진술 → 0건 (motif_audit 두 건 모두 OK)
- 봄 / 계절 직접 선언 → 0건 (Phase 1 closure register 자체로 처리)
- idol fantasy / destiny / miracle / forever / superstar → 0건
- 클리셰 (비/새벽/우산/너의 향기/빈자리) → 0건
- 사용자 보호 명령 ("공연장/이어폰/공연 직접 호명 등 보편화") 충실 — "박수", "옆얼굴", "한 줄의 빛", "한 호흡" 만으로 군중-속-한 사람 register 구축

### Korean QA
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "또렷이 / 또렷해지는 / 또렷했던" 의도적 반복 — chorus motif 형성, alto 음역에서 발음 명확.
- "남았어 / 남아 있었어 / 머문 / 데리고 돌아왔어" 잔향 동사군 — phase 1 closure register 의 음향적 일관.
- "기울었어" — 한 호흡 안 안정. "잦아들고" 도 BPM 75에서 fit.
- 영어 / 숫자 / 외래어 / Sino-Native 충돌 0건. 화법 일관 (반말 자기독백, 여성 intimate withdrawn alto 톤 적합).
- 자모 emoticon 0건. stage direction 0건. 50라인 안 (47 라인 paste block).

### Standout 5 hand-picked lines
1. **"사람이 너무 많은 자리에 처음으로 한 사람이 보였어"** — first-line hook. 군중 속 부재가 단 한 사람의 출현으로 즉시 균열되는 패러독스 1행 압축.
2. **"한 음만 내 쪽으로 천천히 걸어왔어"** — 공연장 직접 호명 없이 "발신성 → 수신자" 구도를 한 음 의인화로만 그림. v6 에 없던 v8 신규.
3. **"멀어진 게 아니라 내가 골라 듣고 있었어"** — pre-chorus 의 인식 전환. v6 "내가 듣지 않기로 한 것 같아" 보다 능동/책임 톤이 더 정직 (수동 → 능동의 미세 이동, 그러나 confession 까지는 가지 않음).
4. **"누군가의 어깨에 잠깐 머문 한 줄의 빛"** — v6 "무대 끝의 한 줄 빛"에서 무대 어휘를 제거하고도 동일한 영상미. 보편화 명령 충실 + 시적 밀도 유지.
5. **"그 한 점을 주머니에 넣고 잠들었어"** — bridge 의 비-명백한 subversion image. v6 "그 한 점을 데리고 잠들었어" 의 추상 → "주머니에 넣고" 라는 일상 구체로 떨어뜨려 phase 1 의 withdrawn intimacy 를 신체화. 한 사람을 비밀로 가져가는 행위가 거창하지 않게.

### Removed from v6
- "사람이 너무 많아서 아무도 안 보이는 줄 알았어" → 부정 → 반전 2 박자 구조. v8 은 한 줄 안에 충돌 압축 ("처음으로 한 사람이 보였어").
- "누가 누구의 박자를 보내는지 몰랐어" → 음악 어휘 직접. 제거하고 "한 음만 내 쪽으로 천천히 걸어왔어" 로 발신성 보편화.
- "무대 끝의 한 줄 빛이 떨어지는 자리" → 무대 직접 호명. "누군가의 어깨에 잠깐 머문 한 줄의 빛" 으로 무대 어휘 제거.
- "내가 듣지 않기로 한 것 같아" → 자기 청취 능력 결정자 진술 (motif_audit Self-Hearing Prohibition 위험). "내가 골라 듣고 있었어" 로 일반 선택 행위로 완화 — 청취 능력 자체가 아닌 주의의 방향.
- v6 verse 2 "빛이 한 군데로 모이는 걸 처음 봤어" → 시각만 강조. v8 은 "공기 중의 먼지가 한 군데로 천천히 모였어 / 빛이 어디로 떨어지는지 처음으로 알아챘어" 2행으로 분리해 시간감 추가.
- v6 final chorus "박수가 다 사라지고도 / 한 사람의 윤곽만 남았어" → v8 은 "박수가 다 사라진 뒤에도 / 한 사람의 윤곽만 남아 있었어" 로 시제 살짝 늦춰 phase 1 closure 잔향감 강화.
- bridge 2행 추가 ("이런 시선을 가져 본 적이 없어서 / 가져도 되는 건지 한참을 몰랐어" 는 verse 2 끝으로 이동) — 자기 의문이 곡 후반에 쌓이도록 재배치.

## Unanswered for Conductor

1. **Final chorus 변형 ("한 사람의 윤곽만 남아 있었어")** 가 음악적 lift 로 작동하는지 Suno 생성 후 청취 판단 필요. lift 약하면 v6 의 "한 사람의 윤곽만 남았어" 로 회귀 가능 (verbatim 보존도 OK).
2. **Bridge 6행** — 8-12 권고 하한이지만, 의도적 압축으로 final chorus 진입을 빠르게. 더 길게 원하시면 "오늘 본 게 사람이었는지 / 한 줄의 공기였는지 모르겠어" 2행 추가 후보.
3. **Verse 2 마지막 2행 ("이런 시선을 가져 본 적이 없어서 / 가져도 되는 건지 한참을 몰랐어")** — 자기 의문의 정서가 phase 1 withdrawn 톤에 적합한지. Seojun POV 직접 노출이 아니라 화자 보편 톤 안에서 처리됨. female intimate withdrawn alto 로 부를 때 confession 직전의 망설임 register 로 작동 예상.
4. **48 라인** — 권장 45-50 안. 더 압축 원하시면 verse 1 의 "숨이 빠르게 오가는 자리에서 / 나는 늘 작아지는 쪽이었어" 2행 → 1행 통합 가능.
