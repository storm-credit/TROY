## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘은 이상하게 말이 빨리 줄었어" — 평범한 자기관찰 발화 안에 이미 정서 사건(가까움이 발화를 줄였다는 역설)이 일어난 상태. 워밍업 0. 한국 청자가 즉시 "같이 있는데 말이 줄어드는 그 저녁"으로 환원 가능. 검정치마/이바다 류 인디 chamber-pop 의 첫 줄 패턴 — 작은 발화로 큰 정서 안착.

### Chorus emotion line placement & 5 standout lines
Locked chorus 두 행 ("오래 알았던 사람처럼 / 우리는 가까워진 저녁이 되었어") = chorus 1·2 verbatim 2회. Final chorus 의 마지막 lift 라인 "오늘은 그 저녁을 / 조금 더 오래 알고 싶었어" 가 변형 1회 (시제 전환 — 머물고 싶음 → 알고 싶음, 인지 동사로 닫음).

5 standout lines:
1. "오늘은 이상하게 말이 빨리 줄었어" — first-line hook, 역설적 친밀.
2. "모퉁이에서 같은 쪽으로 같이 돌았어" — 보편 이미지로 동기화 표현 (춤도 가사도 아닌 동작).
3. "어디서 본 것 같은 게 아니라 / 오래 봐 온 것 같다는 게 더 가까운 말이야" — 데자뷔 클리셰를 한 칸 비틀어 정의함.
4. "꺼내면 작아질 것 같은 말이라 / 손바닥 안에 잠깐 접어 두었어" — 발화 보류의 촉각화.
5. "의심보다 먼저 도착한 가까움이 있어서" — bridge subversion 의 anchor.

### Bridge subversion
Bridge 는 chorus 의 "가까움 = 좋음" 톤을 의도적으로 흔든다 — "빨리 가까워진 자리는 빨리 멀어질 수도 있다" 의 외부 경고를 호명. 그러나 부정으로 닫지 않고 "오늘 저녁만큼은 가까운 채로 두기로 했어 / 의심보다 먼저 도착한 가까움이 있어서" 로 의식적 보류 선택. 행복의 미래-위험 톤(과신의 작은 씨앗)을 outright 부정 없이 끌어안음. Phase 3 chamber-pop 의 sweet-with-shadow 룰 충족.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 카페/캠퍼스/데이트 루트 직접 호명 → 제거 (보편 "모퉁이/길/저녁"으로 치환)
- 여름컵/브런치 등 specific prop → 0건
- 공감각·청취 능력·sensory origin → 0건 (motif_audit 통과)
- 클리셰 (영원/완벽/운명/never wrong/destiny) → 0건 (brief words-to-avoid 준수)
- borderline: "같은 박자" — 박자 모티프는 v6 에서 이어진 보편 시어, 음악-루프 직설 아니라 동기화 추상 — OK.
- "저녁" — 계절보다 시간대 anchor. JSON image seed "정지한 저녁" 직접 반영, 클리셰 아님.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "오래 알았던 / 오래 봐 온 / 오래 머물러 / 오래 알고" — 디귿·리을계열 의도적 motif 형성. 발음 명확.
- "모퉁이에서 같은 쪽으로 같이 돌았어" — 한 호흡 안 안정. BPM 78 chamber-pop 에서 여유.
- "꺼내면 작아질 것 같은 말이라" — 모음 흐름 부드럽고 호흡 1회.
- 영어/숫자/외래어/jamo emoticon 0건. 화법 일관 (반말 자기독백, 여성 intimate alto).
- "박자/저녁/가까움" 명사 반복은 음악적 anchor — 우연 모노톤 아님.

### Removed / Replaced from v6
- v6 chorus "다시 웃기 시작하자, 불안도 잠깐 박자를 놓쳤다" → JSON thesis 가 "행복의 리듬"이 아닌 "오래 알았던 사람처럼 가까워진 어느 저녁" 으로 정의됨. v8 은 새 thesis 안착이 우선 — chorus 를 새 anchor 로 전면 교체.
- v6 "한 줄이 도착했어 / 손바닥에 한참 두었어" → 화해-가벼움 신 직설. v8 은 그 손바닥 motif 만 한 줄로 흡수 ("손바닥 안에 잠깐 접어 두었어" — 다른 정서로 재배치).
- v6 "가벼움은 무거움을 지우지 않고 / 무거움 옆에 한 칸을 만들어 두는 일이야" → 메타 진술 톤이 강함. v8 bridge 의 "옆으로 둔다고 사라지는 건 아니지만" 한 줄로 압축.
- v6 final chorus 의 "다시 좋아지려는 자리에 둘이 같이 서 있다는 걸" → 화해 후 회복 톤. v8 은 thesis 변경으로 "오래 알았던 저녁을 더 오래 알고 싶다" 로 종결 (인지적 lift).
- v6 의 50라인 직전 → v8 47라인 (45-50 권고 안).
