## Rationale (post-script, not pasted to Suno)

### First-line hook design
"하고 싶은 말이 입 끝에서 자꾸 멈췄어" — 한국 청자가 즉시 자기 경험으로 환원할 수 있는 보편 망설임. 첫 줄에서 곡의 emotional event ("멈춤") 이 이미 발생한 상태이고, 다음 줄 "물음표로 둘지 마침표로 둘지 몰라서"가 즉시 thesis 본체로 진입 — 워밍업 0, 설명 0. v6 의 "오랫동안 굴려 둔 한 마디" hook이 confession 후행적이었다면, v8 은 confession 이전의 punctuation 자리에서 hook을 잡음 (브리프 thesis 일치).

### Chorus emotion line placement
Locked thesis "물음표와 마침표 사이" 가 chorus 1행에 verbatim 2회 (chorus 1·2). final chorus 의 첫 행 "물음표와 마침표 사이에" 는 동일 verbatim이고, 2행에서만 "조금 더 오래 앉아 있을게"로 lift — 변형 1회 룰 안에서 사용 (verbatim 3 lock + tail-only lift). 하드락 ≥2회 충족.

### Universal-imagery audit (작품 신 풀이 0건 확인)
- "카페 창가" 직접 진입 → "창문 너머의 빛이 한 칸씩 옮겨가는 동안" 한 줄만 보편 진술
- "접힌 종이 / 두 컵 사이의 정적" 작품 motif 직접 호출 → "컵을 두 번 다시 잡았고" 한 줄로 일반 동작화 (특정 prop 호출 회피)
- forever / destiny / soulmate / grand promise / confession anthem swell → 0건
- "고백" 어휘 자체 → 0건 (브리프의 must-not-include "grand confession anthem" 준수, 곡이 confession 행위가 아니라 punctuation 자리의 윤리로 작동)
- 클리셰 (비/새벽/우산/너의 향기/빈자리) → 0건
- 능력 공개 / 청취 능력 / sensory origin → 0건 (motif_audit_heard_vs_said: 발화 이전의 멈춤만 다루고 청취 결과는 다루지 않음)

### v6 → v8 standout 보존 전략
v6 의 가장 강한 라인은 bridge 의 "한 마디가 한 사람을 옮길 수 있다는 걸 / 오늘 처음 알았어" (사용자 지정 standout). v8 bridge 에 verbatim 으로 보존하되, 이어지는 3행 ("그래서 더 천천히 골랐어 / 누구를 옮기려는 게 아니라 / 나를 먼저 옮겨 두고 싶어서") 으로 의미를 subvert: v6 에서는 "옮길 수 있다" 가 confession 의 사후 깨달음이었다면, v8 에서는 "그래서 더 천천히" 로 옮겨 — bridge subversion (confession→punctuation 으로 framing 전환).

### Removed / replaced from v6
- v6 첫 줄 "오랫동안 굴려 둔 한 마디를 오늘 처음 입에 올렸어" → 이미 발화 후 시점. v8 은 thesis 가 발화 전 punctuation 이므로 "하고 싶은 말이 입 끝에서 자꾸 멈췄어"로 교체 (동일 톤, 다른 시점)
- v6 chorus "말하고 나서야 / 나는 처음으로 기다리는 사람이 되었어" → confession 후행. v8 thesis 와 충돌하여 전체 chorus 재작성 ("물음표와 마침표 사이에 / 나는 잠깐 머물러 있어")
- v6 verse 2 "컵 두 개가 식어 가는 동안" → 카페 신 직접. "컵을 두 번 다시 잡았고"로 보편화 (단수 "컵", 일반 동작)
- v6 final chorus "오늘은 도망치지 않을게 / 한 마디를 입에 올린 자리에서 / 나는 끝까지 앉아 있을게" → confession 사후 다짐. v8 은 punctuation 자리에 머무는 윤리 ("끝맺지 않은 한 줄을 부끄러워하지 않을게")로 교체
- v6 bridge "어떤 말은 입 밖에 나오는 순간 / 하는 사람을 다른 사람으로 만들어" → bridge 의 다른 행은 제거하고, 사용자 지정 standout 두 줄만 보존 + 3행 subversion 추가

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음
- "물음표 / 마침표" 디귿·미음·표 계열 의도적 반복 — chorus motif 형성, 발음 명확 (BPM 76 안에서 한 호흡 안 들어감 확인)
- "끝맺지 / 끝나지도 / 끝맺기 / 끝을 정하지 / 끝맺지 않은" 끝-계열 의도적 반복 — 청취 시 motif 형성. "ㄱㅓㅌ" 받침 명료
- "다음 글자가 무엇이든" — chorus 5행 두 번째 단음절 강세 자리, female intimate alto 에 적합
- "한 박자 / 한 줄 / 한 글자 / 한 마디" 한-계열 반복 — Phase 2 careful counting 톤 형성
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백)
- 첫 줄 "하고 싶은 말이 입 끝에서 자꾸 멈췄어" — 한 호흡 안에 들어감 확인. "입 끝" 부분 짧은 자모 군집이지만 BPM 76 의 첫 박자 진입 자연

## Unanswered for Conductor

1. **standout 위치 변경 검토** — v6 standout 두 줄을 bridge 머리에 verbatim 보존했지만, "오늘 처음 알았어" 는 confession-후행 톤. v8 thesis (punctuation 자리) 와 미세하게 어긋날 수 있음. 더 강한 정합을 원하면 bridge 첫 두 줄을 "한 글자가 한 사람을 옮길 수 있다는 걸 / 오늘 처음 알았어" 로 변형 가능 (마디 → 글자 = punctuation framing 일치). 변형 1회 quota 사용 시 final chorus lift 에서 회수 필요.
2. **final chorus lift 강도** — "조금 더 오래 앉아 있을게" 가 v6 의 "끝까지 앉아 있을게" 보다 의도적으로 약함 (Phase 2 = breathing closer, 결단 X). Suno 생성 후 lift 가 너무 약하게 들리면 final chorus 마지막 두 줄을 "다음 글자가 늦게 와도 / 나는 같은 자리에서 한 번 더 기다릴게" 로 강화 가능.
3. **45-50 line target** — 현재 47 라인 (섹션 태그 제외 본문 기준), v8 craft band 안. pre-chorus 가 3행으로 짧음 — 4행으로 늘리려면 "지금은 아직 답이 아니라 자리가 필요한 시간" 다음에 "그 자리에 한 번 더 앉아 보기로 했어" 추가 가능.
4. **bridge 길이 (5행)** — 8-12행 권고 하한 미달. 의도적으로 짧게 가져가서 standout 두 줄이 subversion 3행에 의해 즉시 reframe 되도록 함. 더 길게 원하시면 2행 추가 가능 ("옮긴다는 말 자체가 무거워서 / 나는 그 무게부터 받아 들고 싶었어").
5. **여성 보컬 적합성** — verse 1 "한 글자를 오래 손에 쥐고만 있었어" 류 자기관찰 라인은 female intimate withdrawn alto 에 적합. Phase 2 register lock 유지.
