## Rationale (post-script, not pasted to Suno)

### First-line hook design
"한 번 잘못 들은 말이 두 번째엔 사실이 됐어" — thesis 즉발. "misunderstanding-solidifying-into-fact" 의 핵심 메커니즘 (반복이 사실을 만든다)이 첫 줄에 그대로 들어감. 폭발 없이 정보적 진술처럼 시작해 청자가 한 번에 그림을 잡게 함. 워밍업 0, E001 v8 의 "잘 지내냐는 말에 잘 지낸다고 답했어" 와 동일한 평이한 어조 안의 균열 패턴.

### Chorus emotion line placement
Locked phrase "아닌 게 사실이 되어 가고 있었어" 는 chorus 1·2 verbatim 2회, final chorus 도 verbatim 첫 행으로 유지 (3회 lock). Final chorus 의 lift 는 마지막 1행 "나만 그게 아니라고 작게 적어 두었어" — chorus 의 "혼자서 작게 말하고 있었어" 를 "작게 적어 두었어" 로 동사만 미세 변형 (말 → 글, 휘발 → 잔존). 변형 1회 룰 안에서 바로잡지 못한 흔적이 글로 남는 lift 만들어줌.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 음악실 / 복도 / 캠퍼스 / 우산 / 컵 → 제거 (v6 의 "복도 끝", "젖은 우산", "밀린 컵" 모두 작품 motif 직접 호출이라 삭제)
- 공감각 / 들은 감정 근거 단정 → 제거 (서준 sensory origin motif 회피)
- 둘이 마주 선 신 / 관계 직접 명시 → 제거 (보편 "내 이야기", "내 얼굴", "한 줄"로 치환)
- 클리셰 (눈물/이별/마지막/안녕) → 0건
- 폭력·EDM·trailer 어휘 → 0건 확인
- borderline: "단단한 모양", "굳어 있었어" — 일반 시어로 작동, motif 직접 호출 아님

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 된소리 hook 없음.
- "아닌 게 사실이 되어 가고 있었어" — 한 호흡 안. "사실/사이/사람" ㅅ 계열 자연 반복으로 motif 형성.
- "굳어진 / 굳어 / 단단한" 의도적 반복 (정서 anchor — 고체화 모티프).
- "한 번 잘못 들은 말이 두 번째엔 사실이 됐어" 첫 줄 — 음절 약간 많지만 BPM 74 에서 안정. 한 호흡 OK.
- 영어/숫자/외래어/jamo emoticon 0건. 화법 일관 (반말 자기독백, 여성 intimate alto).

### Removed from v6
- "컵을 살짝 밀었을 뿐인데 / 젖은 우산이 손에서 미끄러지듯이" → 작품 신 직접 묘사. thesis ("misunderstanding-solidifying-into-fact") 와도 어긋남 — v6 는 "충돌 시작" thesis 였으나 genre_assignment.json 의 E066 anchor 는 "오해가 쌓여 사실이 되어 가는 무서움" 으로 다름. v8 에서 thesis 자체를 corrected.
- "복도 끝에 너와 내가 서 있었고" → 둘 마주선 신 직접. "내가 한 적 없는 말이 내 입에서 나왔다고 적혀 있고" 보편 자기소외로 교체.
- "아무도 소리치지 않았는데 공기는 이미 부딪히고 있었어" (v6 chorus lock) → thesis 변경에 따라 "아닌 게 사실이 되어 가고 있었어" 로 교체. 충돌의 외부성(공기) 에서 misunderstanding 의 내부 고체화로 anchor 이동.
- "끝을 내려는 게 아니야 / 끝이 다가오는 걸 보고 있는 거야" → bridge 가 종결-impulse 쪽으로 기울어 thesis 와 어긋남. "어쩌면 나도 어딘가에선 누군가의 가짜였을지도" 자기 반사로 재작성 — bridge subversion (피해자 위치에서 가해 가능성으로 시점 전환).

## Unanswered for Conductor

1. **Thesis correction 확인** — v6 (충돌 시작) 와 v8 (오해 고체화) 가 다른 anchor 를 갖는다. genre_assignment.json 의 emotionalAnchor 를 v8 이 따랐고, v6 song_brief 의 chorus emotion line 은 폐기됨. song_brief 자체를 v8 에 맞게 갱신할지 Conductor 판단 필요.
2. **Bridge subversion 강도** — "어쩌면 나도 누군가의 가짜였을지도" 시점 전환이 phase 4 register 안에서 너무 멀리 가는지 검토. 더 작게 가져가려면 "그래서 더 무서웠어 / 내가 누구의 사실인지 모르겠어서" 정도로 수축 가능.
3. **Final chorus lift "작게 적어 두었어"** — "말 → 글" 변형이 음악적으로 약하면 verbatim "혼자서 작게 말하고 있었어" 로 회귀 가능. 또는 "작게 적어 두었어" 를 "오래 적어 두었어" 로 시간축 변형 옵션.
4. **여성 보컬 적합성** — verse 2 의 자기 가해 자각 ("나도 한때 누군가의 이야기를 그렇게 옮긴 적이 있어서") 는 female intimate alto 에 잘 맞춤. male 회귀 시 자기-비판 톤이 더 무거워짐 — female lock 유지 추천.
5. **45-50 라인 준수** — 본 v8 은 47 라인 (헤더/태그 제외 본문 카운트) 으로 범위 안. 짧게 가려면 verse 2 마지막 2행 압축 가능.
