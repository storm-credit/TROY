# ten point upgrade plan - 2026-04-11

## 1. purpose

이 문서는 현재 `TROY` 원고를 `완결 원고`에서 `10점권 원고`로 올리기 위한 마지막 업그레이드 조건을 잠근다.

전제:

- 원고는 이미 완결 상태다
- 구조 재설계나 대수정은 하지 않는다
- 길이 잠금은 유지한다
- 작품의 장르 정체성은 `문학형 감성 로맨스`로 고정한다

## 2. current score position

오케스트라 종합 추정:

- 현재 위치:
  - `8.8 ~ 9.1 / 10`
- 현재 상태:
  - complete manuscript
  - no major rewrite needed
  - targeted precision pass remaining

## 3. what 10 means for this project

이 작품에서 `10점`은 아래를 뜻한다.

- 더 큰 사건이 생기는 것
- 더 뜨거운 재결합이 붙는 것
- 더 자극적인 플랫폼 후킹이 들어가는 것

이 아니라,

- 이미 있는 고유성을 끝까지 배신하지 않는 것
- 좋은 결론을 한 번만 말하고 장면이 대신 수행하게 하는 것
- 문장 호흡이 전권에서 더 투명하게 읽히는 것
- 외부 독자 검증에서 `이 작품답게 정확하다`는 반응이 공통으로 잡히는 것

## 4. locked positioning

반드시 유지할 포지셔닝:

- `감각과 윤리의 문학형 청춘 로맨스`
- `듣는 것으로는 사랑할 수 없고, 끝내 말로 물어야 하는 두 사람의 이야기`
- `아린`은 치유 오브젝트가 아니라 창작자 / 선택자 / 경계의 주체
- 결말은 `재결합 보상`보다 `낮은 종지의 성숙`

## 5. top upgrade axes

### A. line precision

가장 큰 체감 개선축:

- 좋은 결론을 반복 설명하는 문장을 줄인다
- `추상 진단 3연타`를 `핵심 문장 1개 + 구체 동작 1개`로 정리한다
- 교훈화 직전의 종결 문장을 생활 장면이 대신 말하게 만든다

우선 회차:

1. `E058`
2. `E045`
3. `E120`
4. `E119`
5. `E094`
6. `E118`
7. `E113`

### B. read-aloud cadence

필요한 것:

- full-book read-aloud cadence pass 확대
- 대사 착지
- 문단 호흡
- 한 호흡 과밀 문장
- chapter5-6 후반부의 리듬 대비 조정

### C. serialized feel reduction

필요한 것:

- 제출본 기준으로 일부 회차 말미의 `정리 문장`, `회차 끝 강조`를 조금 더 자연스럽게 녹인다
- 회차형 쪼개짐보다 장편 흐름이 먼저 느껴지게 만든다

### D. external validation

필요한 것:

- 맞는 독자에게 맞는 질문으로 검증
- 초반 1-3화 진입감
- 중반 반복 체감
- 후반 재회와 엔딩 온도
- 아린의 주체성 유지 여부

### E. submission precision

필요한 것:

- 작품 소개 문구 통일
- 외부 전달 문구 정밀화
- purpose-specific packet 유지

## 6. do not touch

절대 건드리면 안 되는 것:

- `재결합 아님 / 능력 회복 아님 / 운명 보상 아님`의 결말 문법
- `아린`의 selfhood와 선택 논리
- `서준`의 변화 방향:
  - 더 잘 듣는 능력이 아니라 더 늦게 판단하고 묻는 사람
- `heard vs said`, `Arin exception`, `self-hearing prohibition`, `silence`의 payoff 구조
- 후반부를 더 극적으로, 더 낭만적으로, 더 해피엔드처럼 만드는 보강

## 7. execution order

1. 알파리더 검증 매트릭스 고정
2. `package A`로 1차 외부 반응 수집
3. 공통 병목이 반복 확인될 때만 line polish 패스 개시
4. line polish는 `E058 -> E045 -> E120 -> E119 -> E094` 순서
5. full-book read-aloud cadence pass는 원고 수정이 실제 열릴 때 병행
6. 수정 후 재평가

## 8. ten point conditions

10점 판정 조건:

- 초반이 느려도 `계속 읽어야 하는 이유`가 남는다
- 중반이 반복이 아니라 `누적 긴장`으로 읽힌다
- 결별과 재회가 모두 과열 없이도 충분히 아프고 설득력 있다
- 아린이 끝까지 독립적 인물로 남는다
- 엔딩이 `심심하다`보다 `이 작품답게 정확하다`로 더 많이 읽힌다
- 서로 다른 타입의 외부 독자 3종 이상이 공통으로
  - 후반 보상감
  - 관계 윤리의 설득력
  - 문장/분위기 몰입감
  을 인정한다

## 9. director verdict

이 작품은 이미 완결이다.

10점으로 가기 위해 필요한 건 `더 큰 것`이 아니라 `더 정확한 것`이다.

즉, 지금의 오케스트라 임무는:

- 원고를 보호하고
- 반복 설명을 줄이고
- 읽는 호흡을 더 투명하게 만들고
- 맞는 독자에게 맞는 질문으로 검증하는 것

이다.
