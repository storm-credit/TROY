## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘은 조금 일찍 불을 껐어" — pre-sleep register 즉시 진입. 행동 한 줄로 시간대(밤)·심리(피로보다 자처한 단축)·1인칭 자기독백을 동시에 연다. 워밍업 0. v6의 "오늘은 창가에 앉아 한참을 그대로 두었어" (낮·창가) 대비 v8은 "방·어둠·베개"로 thesis (잠 들기 직전 떠오르는 얼굴) 에 직격. 평범한 발화 안에 정서 균열이 이미 발생함 (E001 v8 패턴 답습).

### Five standout lines
1. **"이름은 그대로인데 이름의 자리만 비어 있었어"** (v1 last) — 이름 vs 자리 분리, 보편 망각의 정확한 메커니즘 묘사.
2. **"잊는 게 아니라 닳고 있다는 걸 알았어"** (v2) — 망각 ≠ 마모 인식의 thesis line. 적극 거부의 미화 없는 정직.
3. **"미안하다는 말이 먼저 떠오르다가 / 아무한테도 닿지 않을 말인 걸 알았어"** (pre-chorus) — 사과 대상의 부재. 보편 잔존 죄책감을 cure 없이 인식.
4. **"선명한 사랑보다 흐릿한 안부가 / 오늘의 나를 더 오래 재워 줬어"** (v3) — 흐림이 잠을 부르는 역설. Phase 5의 anti-cure-event 핵심.
5. **"사라짐은 한 번이고 옅어짐은 매일이라서 / 매일의 작은 이별을 매일 모르고 지나갔어"** (bridge) — 일회성 ≠ 일상성. v6 standout 계승하되 시간 구조로 확장.

### Chorus emotion line placement
Locked phrase "이름 없는 자리도 한동안은 살 만해서" 는 chorus 4행에 verbatim 3회 (chorus 1·2·final), final chorus 의 마지막 lift 라인 "이름 없는 자리에 나는 먼저 잠들었어"는 1-variant lift — "살 만해서"의 미래형 가능성을 잠 행위로 안착(restraint 안에서의 lift). v6의 "이름 없는 자리도 한동안은 살 만하다는 걸" 의 종결어미만 음악적으로 다듬어 (~하다는 걸 → ~해서) chorus 호흡에 맞춤. canon anchor 의미 100% 보존. 변형 1회만 허용된 룰 준수.

### Bridge subversion
v6 bridge ("이름을 한 글자씩 풀었어 / 새 이름을 붙이지 않기로 했어") = 이름 풀기. v8 bridge = "사라짐 vs 옅어짐" 구조 발견. chorus 의 정서적 반복을 끊고 인지적 reframe ("매일의 작은 이별") 로 이동, 다시 chorus 로 떨어뜨릴 때 "빈 자리"의 의미가 한 겹 두꺼워짐. 6행으로 짧게 유지해 final chorus lift 를 받침.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 카페 달 / 창가 / 컵 → 제거 (보편 "방·베개·천장"으로 치환)
- 16세 상처 / 유빈 대면 / 첫 상처 형태 → 제거
- 카페 신 직접 호명 0건. healed/closure/first-love returned/cure-through-conversation 0건.
- 클리셰 (비/새벽/우산/너의 향기/빈자리·계절직접선언) → 0건. "빈 자리"는 thesis 의 자기 결정 행위로 재정의됨 (클리셰 정서 ≠).
- borderline: "오래 만진 사진의 가장자리처럼" — 사진은 일반 시어, 작품 motif 직접 호출 아님. OK.
- 공감각·청취·sensory origin 0건 확인. seojun_sensory_origin / arin_song_thread / heard_vs_said 모티프 침범 0건.

### Korean pronunciation pass
- BPM 66 acoustic-ballad fit. 각 라인 한 호흡 안 들어감 확인.
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "옅어 / 옅음 / 옅은" 의도적 반복 — 청취 시 motif 형성, "옅" 발음 명확.
- "이름 / 이름의 자리 / 이름 없는" 디귿계열 ㅇ-ㄴ 반복 — chorus anchor.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백). 잠 직전 register 유지 — 빠른 클러스터 회피 ("한 톤 / 한 칸 / 한 번" 단음절 단위로 호흡 끊음).

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 마지막 "이름 없는 자리에 나는 먼저 잠들었어"** lift 작동 여부는 Suno 생성 후 청취 판단. lift 약하면 verbatim 4회 ("이름 없는 자리도 한동안은 살 만해서") 회귀 가능.
2. **여성 보컬 적합성** — verse 2 "오래 만진 사진의 가장자리처럼", verse 3 "흐릿한 안부가 더 오래 재워 줬어" 류 자기관찰 라인은 female intimate alto withdrawn 톤에 정확히 매칭. close unsmoothed vocal directive 와 "어두운 천장 / 끄덕임" 사적 행위 일치.
3. **chorus 행수 6행** — 표준 4-6행 권고 상한. anchor 라인 + lift 변형 위해 6행 유지. 5행으로 줄이면 anchor 의 자리 내림 효과 약해짐.
4. **canon anchor 종결어미 변경** ("살 만하다는 걸" → "살 만해서") — v6 의 chorusEmotionLine null 상태에서 v6 standout 을 v8 chorus anchor 로 승격하면서 chorus 호흡에 맞춘 음운 조정. 의미 완전 보존. v6 와의 연속성 인지 후 승인 요청.
5. **Outro 제거 결정** — final chorus 마지막 lift 라인이 outro 역할 흡수. 별도 outro 행 필요하면 "그 얼굴은 그래도 한 번 더 옅어졌어" fade 추가 가능.
