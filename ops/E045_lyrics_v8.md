## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘 네 말을 한 번 잘못 들었어" — 한국 청자가 즉시 자기 경험으로 환원할 수 있는 보편 고백 (오해의 첫 자리). 시작부터 "감정 사건"이 이미 발생한 상태이고, 평범한 발화 안에 정서 균열이 동시에 들어감 (E001 v8 hook 패턴 = 일상 발화 + 즉발 균열). 워밍업 0. "네 말을 잘못 들었어"는 본편 캐논 motif (heard vs said) 의 보편판이지 작품 신 직접 묘사 아님 — 누구나 한 적 있는 경험.

### Standout line audit (5 line target)
1. "내가 들은 말과 네가 한 말이 / 같은 자리에 닿지 않는 저녁이었어" (verse 1) — over-reading thesis 의 보편판, 시적이면서 즉발 공감.
2. "흔들렸어, 그래도 봤어" (chorus 1행) — 짧은 hook, 두 동사 사이의 쉼표가 의미 락. v6 의 길었던 "붙잡은 건 소리였고…" 를 대체해 첫 hook 폭발력 강화.
3. "기다리는 동안 내가 만든 해석들이 / 조용히 자기 자리를 바꿔 앉았어" (verse 2) — 해석이 살아 움직이는 의인화, sensory origin 호출 0.
4. "다 알고 사랑하는 사람은 없다고" (bridge 1행) — 격언적 lift, bridge subversion 의 entry. 자기 단정에서 보편 진리로 카메라 zoom-out.
5. "틀린 자리에서도 다시 묻는 마음이 / 가까운 거였어" (bridge 후반) — thesis "흔들리는 해석 속에서도 서로를 더 알게 되는 자리" 의 정직한 번역.

### Chorus emotion line placement (v8 신규 락)
v8 신규 chorus emotion line: **"틀려도 너를 더 알게 됐어"** — chorus 2행에 verbatim 2회 (chorus 1·2). final chorus 의 마지막 lift 라인은 자연 변형 "틀려도 너를 더 알아 가는 자리" 로 1회 (현재완료 → 진행형 + 명사화) — 같은 동사어간 "알다" 를 미세하게 열어 두어 음악적 안착 (restraint 안에서의 lift). 변형 1회만 허용된 룰 안에서 사용. chorus lock ≥2회 충족.

### Bridge subversion design
v6 bridge 는 자기 반성 ("나는 그 기다림을 견디기 어려웠어") 으로 닫혔음. v8 bridge 는 자기 반성에서 → 보편 진술 ("다 알고 사랑하는 사람은 없다고") 로 카메라가 zoom-out 하며 화자의 "오답" 이 사실은 사랑의 형식이었다는 재해석으로 뒤집힘. "정확하게 알아서 가까운 게 아니라 / 틀린 자리에서도 다시 묻는 마음이 / 가까운 거였어" — thesis 의 가장 압축된 라인. 이 자리는 final chorus 직전이라 lift 효과 극대화.

### Final-chorus 1-variant lift
Final chorus 마지막에 1행 추가 ("틀려도 너를 더 알아 가는 자리"). 이 라인 한 줄만 변형 — 동사 어간 "알다" 를 "알아 가다" 진행형으로, 그리고 마지막 명사 "자리" 로 닫음. JSON thesis "서로를 더 알게 되는 자리" 의 verbatim 회수 + 진행형 확장. 1-variant rule 정확 준수.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스/강의실/식당/창가 → 0건
- 흐린 유리 / 닫힌 노트 / 이어폰 / 음향 도구 → 제거 (v6 의 brief 이미지 패밀리 직접 호출 안 함)
- 공감각·청취 능력·sensory origin → 0건 ("잘못 들었어" 는 일반 경험, 능력 호출 아님)
- mind-reading / power-up / truth-signal → 0건 (brief 의 prohibited words 모두 회피)
- 봄/여름/계절 직접 선언 → 0건 ("저녁" 한 번만, 시간대 설정)
- 클리셰 (비/새벽/우산/너의 향기/빈자리) → 0건
- borderline: "같은 한 줄을 두 번 듣고도" — v6 에서도 사용했지만 작품 직접 묘사 아니라 보편 청취 경험. 유지.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "잘못 / 잘 / 정확" 등 의도적 ㅈ-반복 — 청취 시 motif 형성, 발음 명확.
- "틀려도 / 틀린 / 틀려도" 디귿계열 의도적 반복 — chorus motif. 발음은 명확.
- "기다리는 동안 내가 만든 해석들이" — 한 호흡 안 안정 확인 (BPM 65 acoustic-ballad).
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백, 가끔 너에게 향함).
- 자모 emoticon 0건. stage direction 0건.
- "조용히 자기 자리를 바꿔 앉았어" — 자음군 확인 ("바꿔 앉" 은 약간 길지만 BPM 65 에서 안정).

### Female intimate alto adequacy
- 자기관찰 + 너에게 도달하지 못한 말의 회상 — female intimate withdrawn 톤에 정확 매칭.
- "흔들렸어, 그래도 봤어" 의 쉼표 호흡 — alto 가 가장 잘 살리는 자리 (남성 보컬은 단정적으로 들림).
- bridge 의 격언 "다 알고 사랑하는 사람은 없다고" — female alto 의 회고적 톤이 가장 자연스러움.

### Removed from v6 (or repurposed)
- "더 잘 들으려고 자꾸 가까이 다가갔어" → 첫 hook 으로는 "기능 자랑" 톤이 묻어남. "오늘 네 말을 한 번 잘못 들었어"로 교체 (즉발 보편 hook).
- "붙잡은 건 소리였고, 놓친 건 네 말이었다" → v6 chorus lock 이지만 부정 결산 톤. v8 thesis 가 "여전히 더 알게 됨" 이므로 "흔들렸어, 그래도 봤어 / 틀려도 너를 더 알게 됐어" 로 교체 (긍정 결산).
- "잘 듣는 자세가 가끔 잘못 듣는 자세가 되고" → over-explanatory. verse 1 에 흡수해 더 짧게 ("내가 들은 말과 네가 한 말이 / 같은 자리에 닿지 않는 저녁").
- "한 박자 먼저 답을 했고 / 한 박자 먼저 한 답이 너의 다음 말을 닫았어" → verse 2 에 보존하되 톤 반전 ("빠르게 답하던 버릇을 한 번 내려놓고 / 한 박자 늦게 너의 다음 말을 기다렸어") — 같은 motif 가 회복으로 작동.
- v6 outro/final chorus "그냥 옆에 앉아 있기로 했어" → v8 chorus 안에 "모르는 채로 옆에 앉아도" 로 흡수, 의도적으로 같은 행위 (옆에 앉기) 가 thesis 의 핵심으로 격상.
