## Rationale (post-script, not pasted to Suno)

### First-line hook design
"처음 보는 풍경 앞에 둘이 멈춰 서 있었어" — 작품 신 (캠퍼스/특정 장소) 직접 호출 없이 보편 "처음 보는 풍경 + 둘 + 멈춤"으로 즉시 정서 안착. v6 의 "얼굴이 한 뼘 가까워진 자리" 키스-인접 첫 줄을 폐기하고, E040 새 thesis ("손을 잡는 무게") 와 정렬. 워밍업 0 — 시작부터 "이미 멈춰 서 있는" 상태로 정서 사건이 발생 완료.

### Chorus emotion line placement
v6 standout couplet "가까워진 거리가 답이 되는 게 아니라는 걸 / 오늘은 처음 알았어" 를 chorus 1·2행에 verbatim 2회 (chorus 1·2), final chorus 도 verbatim 1·2행 유지 — 룰상 chorus lock ≥2회 충족 (실 3회). final chorus 의 마지막 lift 라인은 자연 변형 "나는 그 무게를 오래 들고 서 있었어" (verse "가만히 들고 있었어" → final "오래 들고 서 있었어") 로 시간 확장 lift, 변형 1회 룰 안에서 사용.

### 5 standout 라인
1. "가까워진 거리가 답이 되는 게 아니라는 걸 / 오늘은 처음 알았어" (canon anchor, chorus)
2. "손에 쥔 건 약속이 아니라 / 잠깐 같이 멈춰 선 무게라는 걸" (thesis 직접 번역, chorus)
3. "손끝이 손끝에 먼저 닿았어 / 무겁지도 가볍지도 않은 자리에서" (pre-chorus 무게 미세 측정)
4. "손가락 사이에 작은 공간이 남아 있었어 / 다 채우지 않은 채로 두는 게 좋아서" (verse 2, after-silence 시각화)
5. "나는 이 손을 끝까지 가져갈 자신이 없었어 / 그렇다고 놓을 자신도 없었어" (bridge subversion)

### Bridge subversion
첫 손잡기를 보통 노래는 "확신" 또는 "약속"으로 처리 — v8은 정반대로 "끝까지 갈 자신도, 놓을 자신도 없는" 양가 상태를 정직하게 noting. "사랑이라고 부르지 않기로 했어" 로 명명 거부 → final chorus 의 "약속이 아니라 / 무게" 재진입에 정합. 해피엔딩 증거 회피, Phase 3 "가까움의 정점이 곧 이해는 아님" 락 유지.

- 키스/입술/얼굴 한 뼘 → 제거 (v6 첫 줄 폐기, 손 motif 로 완전 이전)
- 공감각·청취 능력·sensory origin → 0건 (motif_audit_seojun_sensory_origin OK)
- 봄/여름 직접 선언 → 0건 (계절 중립)
- 클리셰 (운명/영원/소유/완전한 이해/destiny/mine/forever/complete) → 0건 (forbidden-language-guard pass)

### Korean pronunciation pass (chamber-pop 76 BPM)
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "손끝이 손끝에" — 디귿/시옷 반복 motif 형성, 한 호흡 안 안정.
- "손에 쥔 건" — 받침 ㄴ→ㄴ 자연 연음, 76 BPM 에 안착.
- "가만히 / 가까워진" 가 계열 의도적 반복 — 정서 anchor.
- "잠깐 같이 멈춰 선 무게" — 한 호흡 안에 들어감 확인. "멈춰 선" 부분이 약간 길지만 chamber-pop 76 에서 안정.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백, 1인칭 화자).
- 자모 emoticon 0건. 스테이지 디렉션 0건.

### Removed from v6
- "얼굴이 한 뼘 가까워진 자리에서 / 나는 숨을 한 번 더 골라 두었어" → 키스 신 인접 묘사. E040 새 thesis ("손의 무게") 미스매치. "처음 보는 풍경 앞에 둘이 멈춰 서 있었어"로 교체.
- "한 사람의 옆얼굴이 이렇게 길게 보이는 줄 몰랐어" → 옆얼굴 응시 = 키스 직전 시선. 손-중심으로 폐기, "손가락 사이에 작은 공간이 남아 있었어"로 교체.
- v6 chorus "숨이 가까워져도, 마음을 다 안다는 뜻은 아니었다" → 숨/키스 anchor 였음. v8 은 "가까워진 거리가 답이 되는 게 아니라는 걸" (v6 의 두 번째 standout 이지만 실제로는 더 thesis-정합) 로 anchor 교체.
- v6 bridge "가까운 자리는 답을 주는 자리가 아니라 / 질문이 자꾸 새로 도착하는 자리야" → 추상 과잉. v8 bridge 는 "끝까지 갈 자신도, 놓을 자신도 없는" 구체적 양가 상태로 재작성.
- v6 final chorus "한 칸을 비워 두었어" → 손-motif 와 결합 약함. "나는 그 무게를 오래 들고 서 있었어" lift 로 교체.

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 마지막 "나는 그 무게를 오래 들고 서 있었어"** 가 음악적 lift로 작동하는지는 Suno 생성 후 청취 판단 필요. lift가 약하면 verbatim 회귀 ("나는 그 무게를 가만히 들고 있었어") 가능.
2. **여성 보컬 적합성** — verse2 "손가락 사이에 작은 공간이 남아 있었어" 류 미세 관찰 라인은 female intimate alto 톤에 정합. chamber-pop 76 BPM 에서 close breath-aware vocal 자연스러움 확인됨.
3. **Bridge 길이 (7행)** — 8-12행 권고 하한 직전. 의도적으로 짧게 가져가서 "사랑이라고 부르지 않기로 했어" 의 단호함이 final chorus 로 떨어지게 함. 더 길게 원하시면 2행 추가 가능 ("그게 미루는 일인지 아끼는 일인지 / 오늘은 굳이 가르지 않기로 했어").
4. **canon anchor 교체 결정** — v6 의 "숨이 가까워져도..." anchor 를 폐기하고 "가까워진 거리가 답이 되는 게 아니라는 걸 / 오늘은 처음 알았어" 로 승격. 사용자 지시 ("v6 standout was [이 couplet]") 에 정합. 만약 두 anchor 를 동시 유지해야 한다면 chorus 를 8행으로 확장 가능.
5. **"풍경" 5회 반복** — 의도적 motif. 청취 시 단조로움이 우려되면 1-2회를 "거리/자리"로 분산 가능하나, thesis 단어 직결이라 현 상태 권장.
