## Rationale (post-script, not pasted to Suno)

### First-line hook design
"한 장을 오래 들여다봤어 거기 있던 색이 어딘가로 빠져나갔어" — episode thesis (기억의 색이 바래기 시작하는 첫 자리)를 첫 줄에 즉발 설치. "사진"이라는 보편 사물을 매개로 (1) 시간 경과, (2) 본인이 인지한 첫 순간, (3) 무엇이 사라졌는가를 한 호흡에 압축. 워밍업 0행. E001 v8 의 "잘 지내냐는 말에 잘 지낸다고 답했어" 와 동일한 패턴 — 평범한 행위 안에 정서 사건이 이미 진행 중.

### Chorus emotion line placement
Locked phrase "색이 바래는 자리에 내가 먼저 서 있었어" 는 chorus 1-2행에 verbatim 2회 (chorus 1·2). Final chorus 의 1-2행은 동일 verbatim 유지하되 3행부터 자연 변형 lift ("한 톤씩 옅어진다는 걸 / 오늘은 천천히 인정해 보려고 해 / 잊는 게 아니라 / 조금씩 다른 색으로 두는 일이라고") — chorus emotion line 자체는 3회 verbatim, 부속 라인만 1회 변형으로 음악적 안착. v8 룰의 "final 1-variant lift" 안전 사용.

### 5 standout lines
1. "한 장을 오래 들여다봤어 거기 있던 색이 어딘가로 빠져나갔어" (verse 1, hook)
2. "잊는다는 건 한 번에 오는 일이 아니라 / 디테일 하나씩 옅어지는 일이라고" (pre-chorus, 정의 라인)
3. "잊는 게 아니라고 말하면서 / 잊어 가고 있었어" (chorus 종결, self-recognition)
4. "이름은 또렷한데 그날의 온도가 흐릿해" (verse 2, 사소함의 무게)
5. "조금씩 다른 색으로 두는 일이라고" (final chorus lift, reframing)

### Bridge subversion
v6 패턴 (거리두기 / 결별 그림자) 는 thesis 변경에 따라 폐기. Bridge 는 "잊는다는 죄책감"의 일반적 진행을 뒤집음 — "바래는 게 잘못된 일은 아니라고 / 누가 말해 줬으면 했어" 로 시작해 "바래는 것도 결국은 한때 선명했다는 뜻" 이라는 reframe 으로 이동. 자기 책망 → 받아들임의 미세한 한 칸 이동, Phase 4 의 "damp fracture" 안에서 닫히지 않은 채 final chorus 로 떨어짐.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스 / 강의실 / 식당 / 유리문 / 복도 → 제거 (보편 "사진 한 장", "한 곡", "한 마디"로 치환)
- 공감각 / 청취 능력 / sensory origin → 제거 (`motif_audit_self_hearing_prohibition` 준수, `motif_audit_seojun_sensory_origin` 무관)
- 캐릭터 호명 (서준/아린/도현) → 0건
- 구체 대화 인용 / 특정 장소 / 계절 직접 선언 → 0건
- 클리셰 (빈자리/우산/너의 향기/마지막 인사/안녕) → 0건
- borderline: "같이 들었던 한 곡" — 「노래」 시어지만 작품 음악 모티프 직접 호출 아님, 보편 회상 도구로 작동 OK
- 영어 / 숫자 / 외래어 / jamo emoticon (ㅎㅎ/ㅋㅋ/ㅠㅠ) 0건
- Korean tense-consonant 군집 위험 0건. 반말 자기독백 일관. "끌어안고" 류 긴 어절 없음 — BPM 69 dream-pop 호흡 안정.

### Removed from v6
- v6 chorus lock "끝내자는 말은 아닌데 / 우리는 이미 한 걸음씩 멀어져 있었어" → episode thesis 갱신 (거리두기 → 기억의 색이 바래기 시작하는 첫 자리). v6 는 "Ending reflection / pre-breakup pause" 프레임이었지만 `120_episode_genre_assignment.json` 의 E071 emotional anchor 와 정합되지 않아 v8 에서 thesis 일치형으로 전면 교체.
- "유리문 / 복도 불빛 / 책상 양 끝" → 제거 (작품 신 직접 묘사 위험, E001 v8 universal 룰 적용).
- "헤어지자 / 끝내자 / 한 걸음씩 멀어졌어" 류 결별 어휘 → 0건. Phase 4 의 "잊어감 / 옅어짐"으로 톤 이동. forbidden_cliches (final goodbye, dramatic ultimatum, healing turn) 0건.
- v6 bridge 의 "멀어진 것과 헤어진 것은 다른 말이야" 정의형 → "잊는 게 아니라고 말하면서 / 잊어 가고 있었어" 자기 인식형으로 교체. 정의 → 자각의 무게 이동.
- v6 outro 부재 → v8 도 별도 outro 없음. final chorus 의 "조금씩 다른 색으로 두는 일이라고" 에 흡수 종결 (E001 v8 패턴 동일).
