## Rationale (post-script, not pasted to Suno)

### First-line hook design
"끝까지 가지 않기로 한 마음에도 사랑이 있어" — 첫 줄에 곡 전체의 명제를 즉시 선언. "끝까지 = 사랑의 증거" 라는 사회적 디폴트를 한 줄로 뒤집고, 청자가 "이 노래는 그 디폴트를 따지지 않는 노래" 라는 걸 0초에 인지하게 함. 워밍업 없이 정서 사건이 이미 시작된 상태. 검정치마/이소라 류의 평탄한 발화 속 균열 패턴.

### Five standout lines
1. "끝까지 가지 않기로 한 마음에도 사랑이 있어" (verse 1, 즉발 명제 hook)
2. "사랑의 끝을 같이 만들지는 않겠다는 뜻이야" (chorus, 통념 분리: 사랑의 종료 ≠ 함께 만드는 종료)
3. "지운 줄이 적은 줄보다 더 정확할 수도 있다는 걸" (verse 2, v6에서 보존 — 자기 편집의 윤리)
4. "끝까지 라는 말을 작게 접어 두면 / 함께 라는 말이 더 오래 펴져 있을 수 있어" (bridge, 두 어휘의 부피 교환)
5. "이 평화는 같이 가지 않은 자리에서 자라났어" (final chorus 마지막 lift, 결정의 후일담)

### Bridge subversion
v6 bridge 는 "이건 결론이 아니야 / 선택의 시작일 뿐이야" 식 자기 격려형. v8 bridge 는 통념을 비틀음 — "멀어지는 일이 아니라 / 가까이 있기 위해 거리의 모양을 바꾸는 일." "끝까지" 와 "함께" 를 부피 가진 단어로 다루어 한쪽을 접으면 다른 쪽이 펴진다는 관계식으로 재구성. 이별-과-지속의 이항대립을 해체하는 단계.

### Final 1-variant lift
final chorus 의 다섯 행은 chorus 1·2와 verbatim 동일 (chorus lock ≥2회 충족: chorus 1, chorus 2, final chorus 의 1-4행까지 4회 lock). final chorus 마지막 6행째에만 1행 lift 추가 — "이 평화는 같이 가지 않은 자리에서 자라났어." 변형이 아니라 추가이므로 chorus lock 손상 0. 결정의 사후 풍경을 한 줄로 봉인하며 곡을 닫음.

### Universal-imagery audit
- 작품 신 직접 묘사 0건 (창가/짧아진 햇빛/이어폰/노트 모두 제거). v6 verse 2 의 캠퍼스 사물 라인을 모두 추상 동사형으로 치환 — "지운 줄이 적은 줄보다 더 정확할 수도 있다는 걸" 만 보존 (사물성 0, 행위 메타포만).
- destiny / certainty / reunion / triumph 어휘 0건 (brief words-to-avoid 준수).
- 클리셰 (비/새벽/우산/너의 향기/빈자리/꽃/별) 0건.
- 작품 motif (창가, 이어폰, 노트, 햇빛) 직접 호출 0건 — 보편 어휘 ("자리", "거리", "길", "문장") 로 치환.
- 청취 능력/공감각/sensory origin 0건 (motif_audit_seojun_sensory_origin / heard_vs_said 이중 OK).
- 능력 회복·운명·재회·확신 어휘 0건 (must-not-include 전부 통과).
- "사랑" 단어 사용: chorus 마다 2회씩 등장 — brief 의 "destiny/triumph 회피" 와 충돌 없음. "사랑" 자체는 보편 어휘이고, 곡의 명제가 사랑의 정의 재구성이므로 필수.

### Korean pronunciation pass (chamber-pop · BPM 73 · female intimate alto)
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "끝까지" 가 곡 전체에서 6회 반복 — chamber-pop BPM 73 에서 "끝-까-지" 3음절 안정. tense consonant ㄲ 청취 명료.
- "사랑의 끝을 같이 만들지는 않겠다는 뜻이야" — 14음절, 한 호흡 안에 들어감. "만들지는 / 않겠다는" 사이 미세 호흡 가능.
- "지운 줄이 적은 줄보다 더 정확할 수도 있다는 걸" — 19음절, 가장 긴 라인. BPM 73 chamber-pop 에서 약간 길지만 v6 에서 검증된 라인. 보존.
- "조금 슬프고 많이 평화로웠어" — chorus 종결 라인. ㅍ-ㅍ (슬프고/평화) 자음 반복이 음악적 motif 형성.
- 영어/숫자/외래어 0건. 화법: 반말 자기독백 일관 (E001 v8 와 호흡 동일).
- 잼오 emoticon (ㅎㅎ/ㅋㅋ/ㅠㅠ) 0건. 무대지시 괄호 0건.
- 라인 수: 45행 (45-50 권고 안). chorus lock 4회 (chorus 1·2 verbatim 5행, final chorus 1-4행 verbatim + 1행 lift).

### Removed from v6
- "오랫동안 증거를 찾는 사람으로 살았어" 류 "증거 찾기" 전체 모티프 → emotional thesis 가 v8 에선 "끝까지 함께 가지 않는 결정의 평화" 로 이동했으므로 chorus lock 자체를 교체. v6 의 "증거를 찾지 않는 쪽을 골랐어" → v8 의 "끝까지 함께 가지 않기로 했어." 동일한 "선택의 윤리" 톤은 유지하되, 명제만 thesis 정합으로 교체.
- "짧아진 햇빛이 책상 위에 길게 누웠어" / "이어폰을 한 번 폈다가 다시 접었어" → 작품 motif 직접 호출. 제거. "자리/거리/길" 추상화로 대체.
- "한 줄을 적었다가 한 줄을 지웠어" → "지운 줄이 적은 줄보다" 라인의 setup 이지만 v8 에선 setup 없이 standout 라인만 보존 (15음절 절약). 행위 자체는 라인 안에서 자기 완결.
- v6 bridge "이건 결론이 아니야 / 선택의 시작일 뿐이야" → 자기 격려형 → v8 의 "거리의 모양을 바꾸는 일" 통념 전복으로 교체 (bridge subversion 룰 충족).
- v6 final chorus "오늘부터 매일 한 번씩 다시 고르기로 했어" → 의지 선언 → v8 의 "이 평화는 같이 가지 않은 자리에서 자라났어" 로 교체 (의지가 아니라 결과의 풍경 — 평화의 사후 시점).

## Unanswered for Conductor

1. **"사랑" 어휘 사용 — chorus 마다 2회**. brief 의 forbidden cliches (destiny / certainty / triumph / comeback cue) 에는 포함되지 않으나, TROY 가 곡 안에서 "사랑" 단어 자체를 절제해 왔다면 1회로 줄일 수 있음 ("사랑이 끝났다는 뜻이 아니라 / 우리의 끝을 같이 만들지는 않겠다는 뜻이야" 식). 현 버전은 thesis 명료성 우선 선택.
2. **chorus lock 5행 길이** — E001 v8 의 chorus lock (6행) 보다 1행 짧음. chamber-pop BPM 73 의 정적 톤에 부합하지만, 마스터링 시 chorus 가 약하게 들리면 1행 추가 가능 ("끝나지 않은 채로 닫는 법을 배우는 중이야" 류 후보).
3. **standout 3번째 라인 (v6 보존)** — "지운 줄이 적은 줄보다" 가 chamber-pop BPM 73 에서 19음절 한 호흡으로 무리 없을지 청취 검증 필요. 무리면 "지운 문장이 적힌 문장보다 더 정확할 수도 있어" 로 17음절 압축 가능.
4. **bridge 길이 (6행)** — chamber-pop 권고 8-12행 하한 미달. 의도적으로 짧게 가져가 final chorus 로 떨어뜨림. 더 길게 원하시면 2행 추가 가능 ("같이 만든 끝이 가장 무거운 끝이라는 걸 / 늦게 알아도 늦은 게 아니야").
5. **여성 인티밋 알토 적합성** — verse 1 의 "함께라는 말이 항상 끝까지를 의미하지는 않아서" 류 정의-재구성 라인은 인티밋 알토의 절제된 호흡에 맞음. lift 폭주 위험 없음. female 락 유지 추천.
