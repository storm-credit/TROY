## Rationale (post-script, not pasted to Suno)

### First-line hook design
"사람들이 다 돌아간 자리가 더 환해" — Phase 1 closer 의 핵심 정서 (소란이 빠진 후의 오히려 또렷해지는 한 자리)를 첫 줄에 즉발 제시. v6 의 "혼자 남아 있어" 는 상태 설명 (setup) 이라 hook 으로 약했음. "더 환해" 는 역설 — 사람이 빠지면 어두워야 하는데 환해진다는 진술이 한 줄에 정서 사건을 일으킴 (E001 의 "잘 지내냐는 말에 잘 지낸다고" 의 한 줄 안 균열 패턴 동형). 워밍업 0행.

### 5 hand-picked standout lines
1. **"사람들이 다 돌아간 자리가 더 환해"** — first-line hook. 역설 1행 안 정서 사건.
2. **"끝났다고 말하기엔 너무 따뜻하고 / 시작이라 말하기엔 아직 이른데"** — Phase 1→2 transition 을 가장 정확히 한 쌍으로 잡는 라인. 경계의 자리 register 의 본체.
3. **"설명할 수 없는 게 처음은 아닌데 / 설명하지 않고도 괜찮은 건 처음이야"** — 서준 사연의 universal 번역 (예외적 현상이 기다리게 되는 사람으로 바뀐 결론). harness 의 closing turn 과 정확히 매칭하면서 보편으로 추상화됨.
4. **"누가 옆에 있는 게 아니라 / 누군가 있었다는 자리가 옆에 있어"** — 부재의 현존 (Phase 1 가 닫히는 방식). 빛/자리 모티프의 재구성, 클리셰 우회.
5. **"그 자리를 두고 가는 게 아니라 / 그 자리를 안에 두고 걷기로 했어"** — Bridge 의 작은 결심. closure 와 transition 을 한 동작으로 처리. final chorus 의 "안에 두고 갈게" 와 연결되는 결정 라인.

### Chorus emotion line placement
Locked phrase "소란이 걷힌 자리에 네가 앉았던 빛만 남았어" 는 chorus 1·2 verbatim 2회. final chorus 에서 1-variant lift 적용:
- 첫 줄: "...남았어" → "...남아서" (연결어미 변형, 다음 행으로 흐르게)
- 마지막 줄: "...남아 있어" (현재진행 종결, 머무는 시간성으로 lift)
Verbatim 핵심부 ("소란이 걷힌 자리에 네가 앉았던 빛") 는 final chorus 의 두 인용 모두에 보존. 변형 1회 룰 안에서 사용. 또한 chorus 마지막 행 "계속 거기 있을 것 같아" 를 의도적으로 2회 반복하여 emotion line 의 잔향을 chorus 내부에서 한 번 더 닫아 줌 (Phase 1 closer 의 음악적 마침표).

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 뒤풀이/카페달/캠퍼스 후문/긴 테이블/종이컵 → 제거 (보편 "의자, 자리, 한 칸" 으로 치환)
- 형광등 / 골목 가로등 / 단체 자리 → 제거
- 서준 청취 능력 / 공감각 / 미세한 결 → 제거 (motif_audit_self_hearing_prohibition: prohibition-as-habit 보존, 능력 노출 0)
- 서준 sensory origin (열여섯의 봄 / overload) → 0건 (motif_audit_seojun_sensory_origin: lore spectacle 거부 룰 준수)
- 직접 고백 / 약속 / 이름 / 호칭 → 0건 (brief words to avoid: confession, promise, forever, destiny, couple — 모두 0)
- 클리셰 (밤거리/우산/너의 향기/별빛/달빛/벚꽃) → 0건
- borderline: "다음 봄이 어떤 모양일지" — "봄" 은 Phase 1 의 시즌이므로 Phase transition 시그널로만 1회 사용. 직접 선언 아님.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 75 BPM 에서 모든 행 한 호흡 안에 들어감.
- "남았어 / 남아서 / 남아 있어" — 변형 lift 의 ㄴ-ㅁ 라인. 발음 명확, 음악적 안착 OK.
- "끝났다고 / 끝나도 / 끝나지 않은" 의 ㄴ-ㄴ-ㄹ 결 — 의도된 motif 형성. tense-consonant 클러스터 없음.
- "설명할 수 없는 / 설명하지 않고도" — 어두 ㅅ 마찰 반복으로 자기진술 톤 형성. 빠른 된소리 0.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백, female intimate withdrawn alto 적합).
- 한자어 의존도 낮음 ("정리, 계절, 시작" 정도). Phase 1 closer 의 reflective 톤에 맞음.

### Bridge with non-obvious subversion
v6 Bridge 는 "그 사이의 자리에 오늘이 와 있어" 로 상태 묘사로 닫혔음 (정적). v8 Bridge 는 마지막 두 행에서 미세한 의지/방향 전환을 넣음:

> "나는 그 자리를 두고 가는 게 아니라 / 그 자리를 안에 두고 걷기로 했어"

Phase 1 closer 의 일반적 grammar 는 "남겨 두고 떠난다" — 의도적으로 그 grammar 를 **뒤집어** "두고 가지 않고 안에 가지고 간다" 로 처리. 이게 brief 의 "open door" + harness 의 "기다리게 되는 사람" 결론을 동시에 충족. "걷기로 했어" 는 결심이지만 결심처럼 들리지 않는 평범한 동사 — 이게 non-obvious 한 부분.

### Final chorus 1-variant lift
final chorus 5행 구조:
1. **lift 변형 1**: "남았어" → "남아서" (연결, 다음으로 흐름)
2. "다음 봄이 어떤 모양일지 몰라도" — Phase 2 미지 시인
3. "오늘 이 빛만큼은" — 보존 의지 anchor
4. "조용한 채로 안에 두고 갈게" — Bridge 결심의 음성화 (반말의 부드러운 약속체 "갈게")
5. **lift 변형 2 (final landing)**: "소란이 걷힌 자리에 네가 앉았던 빛이 남아 있어" — 처음 쓴 "빛만 남았어" 가 "빛이 남아 있어" 로 lift. 한정조사 "만" 제거 + 현재진행 "있어" 로 시간을 늘려 머물게 함. 노래의 마지막 음이 "있어" 의 ㅣ 모음 로 길게 떨어지도록 설계 (alto 의 sustained note 에 적합).

### Removed from v6
- "사람들이 다 빠져나간 자리에 혼자 남아 있어" → 상태 setup. "사람들이 다 돌아간 자리가 더 환해" 로 교체 (역설 hook).
- "의자를 정리하는 소리 같은 작은 것들이 / 오늘은 평소보다 또렷하게 들려" → 청취 능력 시사 위험. 능력 호출 회피하여 "작은 것들이 보여" (시각으로 치환), "의자를 정리하다 한 칸을 비워 두었어" (행위로 치환) 로 분리.
- "한 사람의 윤곽이 거기 머물러 있어" → 직접적. "누군가 있었다는 자리가 옆에 있어" 로 추상도 상승.
- "이게 시작인지 끝인지 정하지 못했어 / 정해야 다음으로 갈 수 있을 텐데" → 메타 진술 약함. "끝났다고 말하기엔 너무 따뜻하고 / 시작이라 말하기엔 아직 이른데" 로 정서 진술로 교체.
- "오늘이 그 경계 같아" → 메타 호명. "그게 오늘의 모양이었어" 로 보편화.
- v6 Bridge 4행 (상태 묘사) → 6행으로 확장하면서 마지막 2행에 결심 동사 진입.
- v6 final chorus "가지고 가고 싶어" → "조용한 채로 안에 두고 갈게" + verbatim lift 1회 추가로 마무리 강도 보강.

## Unanswered for Conductor

1. **변형 라인 lift 작동성** — final chorus 의 "남아서 → 남아 있어" 두 단계 lift 가 한 곡 안 (verbatim 2회 + variant 2회) 에서 chorus lock 룰의 "verbatim ≥2회" 요건과 어떻게 카운트되는지 확인 필요. 현재 해석: chorus 1·2 의 "남았어" 를 verbatim 2회로 카운트, final chorus 는 변형 (1회 허용) 로 처리.
2. **Bridge 6행 길이** — 8-12행 권고 하한이지만 Phase 1 closer 의 정적 톤을 유지하기 위해 의도적 압축. 더 길게 원하시면 "처음은 아닌데 처음 같은 거리야 / 그게 오늘 내 안에 자리 잡았어" 2행 추가 가능.
3. **"다음 봄" 표현 보유 여부** — Phase 1→2 시그널로 1회 사용. Phase 2 의 시즌 grammar 와 충돌 없음 확인되면 유지. 충돌하면 "다음 계절" 로 중성화 가능.
4. **"걷기로 했어" 어조** — Phase 1 의 female intimate withdrawn alto 에서 결심 동사가 너무 능동적으로 들릴 위험. 청취 결과에 따라 "걷고 있어" (현재진행) 로 약화 가능.
5. **chorus 마지막 행 2회 반복 ("계속 거기 있을 것 같아")** — verbatim 룰 외 추가 안전 반복. Suno 가 두 행 같이 부르지 않고 약한 echo 로 처리하면 의도대로 작동. 강하게 반복 부르면 한 행으로 축약 가능.
