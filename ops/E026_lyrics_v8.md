## Rationale (post-script, not pasted to Suno)

### First-line hook design
"약속한 시간보다 오 분 일찍 도착해 버렸어" — '5분 일찍 도착' 은 한국 청자 거의 100% 가 한 번쯤 해본 신체 행동. v6 의 "한참 일찍" 보다 "오 분" 으로 구체화해 즉시 청자의 시계 감각을 깨움. "도착해 버렸어" 의 '버렸어' 가 의도하지 않은 마음의 누설을 한 글자로 처리 — 표면 사건 + 내면 균열이 첫 줄에 동시에 들어감 (E001 의 "잘 지내냐는 말에 잘 지낸다고 답했어" 와 동일 패턴: 평범한 발화 안에 정서 균열). 워밍업 0.

### 5 standout lines
1. "약속한 시간보다 오 분 일찍 도착해 버렸어" — 첫 줄 hook, 즉발 공감.
2. "앉을까 서 있을까 두 번이나 마음을 바꿨어" — 신체적 망설임의 전형, 누구나 자기 경험으로 환원.
3. "공기가 한 톤 다르게 흘렀어" — 청각·촉각 감각의 보편 추상화, sensory origin 우회.
4. "들키지 않으려고 더 깊이 손을 모았어" — bridge subversion 핵심 (들키고 싶음 + 들키지 않으려는 신체 모순).
5. "침묵이 먼저 나를 닮아 있었어" — final chorus 1-variant lift, chorus 의 "침묵이 먼저 달라졌어" 를 "나를 닮아 있었어" 로 변형 — 외부 사건이 자기 정체로 회귀하는 구조적 lift.

### Chorus emotion line placement
Locked phrase "아직 말하지 않았는데 / 침묵이 먼저 달라졌어" 는 chorus 1·2 verbatim 2회, final chorus 첫 두 줄에서 verbatim 3회째 등장. final chorus 의 마지막 lift 라인 "침묵이 먼저 나를 닮아 있었어" 는 변형 1회 룰 안에서 사용 — '달라졌어' (외부 변화) → '나를 닮아 있었어' (내부 정체) 로 정서 무게중심 이동. chorus lock ≥2x 룰 충족.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- '벤치 / 카페 / 약속 장소' 등 캐논 세트 → 제거. 보편 "자리 / 손목 / 손등 / 공기 / 시계" 만 사용.
- 공감각·청취 능력·sensory origin → 제거. "발소리가 한 번 들렸다가 멀어졌고" 는 일반인 모두 가능한 청각 경험.
- 'proposal / forever / destiny / final answer' → 0건 (브리프 금지어 통과).
- 운율을 고백의 근거로 삼기 → 0건 (must-not 통과).
- 고백 확정·미래 약속 → 0건. final chorus 까지 pre-confession 압력만 유지, 폭발 없음.
- 클리셰 (비/새벽/우산/너의 향기/심장박동) → 0건. heartbeat 단어 회피, "숨이 한 박자 빨라졌다" 로 우회.
- borderline: "오 분 일찍" 의 '오 분' 은 Native 숫자로 통일 (Sino "5분" 아님), 발음 안정.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음 (lofi-bedroom BPM 70 적합).
- "괜찮다 / 알아챘어 / 알아채" 디귿·치읓 계열 의도 반복 — motif 형성, 발음 명확.
- "두 번이나 / 한 번이나 / 한 박자" Native 숫자 통일. "오 분" 도 Native — 일관성 lock.
- "손목 / 손등 / 손을 모았어" '손' 계열 3회 의도 반복 — Phase 2 first-touch 신체 register.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백, "-았/었어" 통일).
- 각 라인 한 호흡 안 — "지나가는 사람들이 다 나를 보는 것 같고" 가 가장 길지만 BPM 70 에서 안정.
- 라인 수: 49행 (45-50 권고 충족). bridge 7행 (의도적 짧은 자기 의문).
- 여성 인티메이트 알토 적합성: "손등만 자꾸 만졌어 / 더 깊이 손을 모았어" 등 자기관찰·신체 미시 동작이 female intimate withdrawn 톤에 직접 정합. Phase 2 first-touch hesitancy register 충족.
