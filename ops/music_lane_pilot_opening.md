# Music Lane Pilot Opening

## 1. Purpose

이 문서는 media orchestra의 첫 lane인 `music`을 production-ready 방향으로 열기 위한 파일럿 세트를 잠근다.

목표:

- 120곡 전체를 한 번에 밀지 않고, 작품의 감정 척추를 대표하는 파일럿 곡부터 연다
- song brief가 이미 있는 회차 중 상업성과 작품성을 동시에 보여줄 수 있는 앵커를 우선순위화한다
- 이후 image / MV lane이 참조할 감정 축을 먼저 세운다

## 3. Pilot Set

파일럿 6곡:

1. `E001`
2. `E029`
3. `E040`
4. `E078`
5. `E110`
6. `E120`

선정 이유:

- `E001`: 남주 고립과 세계의 기본 음색
- `E029`: 고백과 감정 진입점
- `E040`: 친밀감의 상업적 접근성
- `E078`: 관계 파열과 진정한 상처
- `E110`: 재접속의 현재형 리듬
- `E120`: 작품 전체 윤리를 닫는 마지막 승인

## 4. Commercial Role Split

- `E001`: 캐릭터 아이덴티티 트랙
- `E029`: 감정 전환 대표 트랙
- `E040`: 가장 넓게 퍼질 수 있는 로맨스 트랙
- `E078`: 깊게 꽂히는 이별 트랙
- `E110`: 재접속/관계 재정의 트랙
- `E120`: 작품 시그니처 엔딩 트랙

## 5. Production Order

권장 제작 순서:

1. `E001`
2. `E040`
3. `E078`
4. `E120`
5. `E029`
6. `E110`

이유:

- 시작, 상업성, 파열, 엔딩을 먼저 잡으면 작품의 음악적 윤곽이 빠르게 선다
- 그 다음에 고백과 재접속을 세밀하게 조정하면 중간 감정선이 더 정확해진다

## 6. Guardrails

- `E040`를 과장된 해피 로맨스 트랙으로 밀지 않는다
- `E078`를 oversung 이별 발라드로 미끄러뜨리지 않는다
- `E110`를 재결합 선언곡처럼 만들지 않는다
- `E120`를 sequel hook이나 triumph ending처럼 만들지 않는다

## 7. Carry-Over

image lane에 넘길 것:

- `E001`: 회색 캠퍼스와 이어폰 거리감
- `E040`: 여름 밤의 가까운 숨
- `E078`: 컵 하나 남은 테이블과 opposite steps
- `E110`: 같은 방향으로 걷는 박자
- `E120`: 이어폰 없는 한강의 공기

MV lane에 넘길 것:

- touching보다 spacing
- confession보다 after-silence
- reunion보다 present-tense rhythm
