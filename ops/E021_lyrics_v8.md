## Rationale (post-script, not pasted to Suno)

### First-line hook design
"손끝이 먼저 멈칫하는 인사를 했어" — Phase 2 first-touch register를 한 줄에 압축. "손끝/멈칫/인사" 세 어휘가 모두 신체화된 망설임을 가리키지만 작품 신을 직접 풀지 않음. 한국 청자가 "어, 나도 저런 인사 해봤다"로 즉시 환원 가능. 워밍업 0, 첫 라인부터 정서 사건이 이미 발생한 상태. 시작 자음 "ㅅ"이 부드러워서 female intimate alto 입천장에 잘 붙음.

### 5 standout lines (v8 craft lock)
1. "손끝이 먼저 멈칫하는 인사를 했어" (verse1 L1) — 신체화된 hesitancy hook.
2. "이상하지 않은 게 이상한 하루였어" (chorus L4) — v6 standout 보존, chorus 정서 anchor.
3. "스치듯 닿은 손등을 못 본 척했어 / 못 본 척하는 게 더 티가 났을지도 몰라" (verse2 L1-2) — first-touch awkwardness 의 자기관찰 이중구조.
4. "이름 붙이지 않은 마음이 / 이름 붙은 마음보다 더 무거울 수도 있다는 걸" (bridge) — Phase 2 답지 않게 성숙한 자기인식; 관계 확정 회피와 정서 무게의 동시 처리.
5. "한 발 가까이 가도 괜찮을지 / 오늘은 묻지 않고 두고 왔어" (final chorus 변형) — Phase 1의 침묵-회피와 다르게, 묻지 않음이 거절이 아니라 "두고 옴"이라는 능동적 보류. final lift.

### Bridge subversion
일반적 indie-folk bridge는 감정 격화 또는 결심 확언으로 가지만, 본 bridge는 의도적으로 평탄화 — "그냥 가만히 있어 보기로 했어"가 결단이 아닌 결단의 보류. 마지막 3행 "이름 붙이지 않은 마음이 / 이름 붙은 마음보다 더 무거울 수도 있다는 걸 / 오늘 처음 알았어" 는 chorus 후 자연스러운 카타르시스를 거부하고 인지적 깨달음으로 대체 — Phase 2 first-touch의 정서 정직성과 일치. 이름 붙이지 않음 = 관계 확정 회피 록 동시 충족.

### Final 1-variant lift
Chorus 1·2 의 마지막 두 행 "한 발 가까이 가면 한 박자 늦게 / 숨이 따라오는 그런 자리" 가 final chorus 에서 "한 발 가까이 가도 괜찮을지 / 오늘은 묻지 않고 두고 왔어" 로 변형. 같은 "한 발 가까이" 문두 유지하되 종결을 묘사(숨이 따라오는)에서 행동(묻지 않고 두고 왔어)으로 이동시켜 음악적 lift 확보. 변형 1회 룰 안에서 사용. v6 standout "이상하지 않은 게 이상한 하루였어" 는 final chorus 에도 verbatim 유지 (총 3회).

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스/강의실/식당/창가 → 제거 (보편 "자리", "길", "오늘" 사용)
- 신호등/모퉁이/계단 → 제거 (v6에서 사용했던 도시 시어 회피)
- 메시지/답장/저장된 음성 → 제거 (브리프 lyricImageSeeds 의 "처음 보낸 메시지/기다리는 답"은 인접 E022·E023 의 voice-memory 록을 침범하지 않도록 의도적으로 회피, 신체적 first-touch 라인으로 치환)
- 공감각·청취 능력·sensory origin → 0건 (motif_audit_seojun_sensory_origin / self_hearing_prohibition 둘 다 OK — 본 가사는 인-월드 아린 화자이지 서준 청취 라인 아님; 그러나 안전을 위해 청취/들림 어휘 자체를 제거)
- 운명/완벽한 거리/forever/couple → 0건 (브리프 avoid 어휘)
- 클리셰 (비/새벽/우산/너의 향기/빈자리/봄바람) → 0건
- "한 박자 늦게" 는 v6 verse2 에서 차용된 일반 표현; 클리셰 아님, motif lock 아님 → OK
- borderline: "숨이 따라오는" — 신체적 표현이지만 청취 motif 아님, 호흡 = first-touch 긴장 표현으로 작동 OK

### Korean pronunciation pass (female intimate alto · BPM 74)
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "멈칫 / 멀어진 / 못 본 / 멈춰" 미음계열 의도적 반복 — 입술 닫힘이 hesitancy 음향을 만듦.
- "이상하지 않은 게 이상한" — 의도적 동어반복, 청취 명확. "이상" 의 ㅇ-ㅅ 연결이 alto 톤에 부드럽게 안착.
- "한 박자 늦게 / 숨이 따라오는" — BPM 74 에서 한 호흡 안에 안정적으로 들어감 확인.
- "이름 붙이지 않은 마음이 / 이름 붙은 마음보다" — bridge 핵심 라인. "붙이지/붙은" 디귿 받침 반복은 명확 발음 가능, 자음 스택 아님.
- 영어/숫자/외래어/jamo 이모티콘 0건. 화법 일관 (반말 자기독백, 해요체 혼입 0).
- 라인당 한 호흡 룰 준수. 가장 긴 라인 "이상한 일은 아무것도 없었는데" (15자) — alto 한 호흡에 OK.
- 클리셰 없는 진성 hesitancy register 유지.

## Unanswered for Conductor

1. **인접 트랙 분리** — E022 (voice-memory · "익숙해진 목소리가 더 무서워서") 와의 정서 구분을 위해 본 가사는 "메시지/음성" 어휘를 의도적으로 제거. 브리프의 lyricImageSeeds "처음 보낸 메시지" 를 사용하지 않은 결정에 대해 Conductor 판단 필요. 사용 권장 시 verse1 을 메시지 라인으로 재작성 가능하나, E022 와의 motif 충돌 우려.
2. **Final lift 강도** — "오늘은 묻지 않고 두고 왔어" 가 final lift 로 충분한지 Suno 생성 후 청취 판단 필요. 약하면 "한 발 가까이 갔던 자리 / 오늘은 그냥 두고 왔어" 로 더 회상조 회귀 가능.
3. **Bridge 길이 (7행)** — 8-12 권고 하한이지만 first-touch 정서가 길게 풀리면 과잉설명 위험. "오늘 처음 알았어" 마지막 1행이 인지적 착지로 작동 — 그대로 유지 추천. 더 길게 원하시면 "그래서 오늘은 / 아무 이름도 붙이지 않기로 했어" 2행 추가 가능.
4. **Phase 2 첫 화 톤 캘리브레이션** — E001 v8 ("내 귀부터 닫고 살았어" · 자기 폐쇄) 에서 E021 v8 (첫 닿음의 어색함) 로의 phase 점프가 가사 톤에서 충분히 들리는지 — chorus 의 "어색한 게 싫지가 않아서" 한 줄이 phase 전환 신호 (Phase 1: 닫음 / Phase 2: 어색함을 받아들임). 청취 시 이 라인이 묻히면 chorus 1행을 "어색한데 싫지가 않아서" 로 강화 가능.
