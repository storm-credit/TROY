# Episode Song Allocation: 001-010

## 1. Scope

이 문서는 1화부터 10화까지의 `1화 = 1곡` 운영 기준을 고정한다.

각 화마다 아래를 지정한다.
- song function
- emotional thesis
- key image
- sound direction
- MV still count

주의:
- 제목은 바뀔 수 있으므로 episode id를 우선한다
- 4화는 외부 문서 간 제목 차이가 있으므로 id 기준으로 관리한다

## 3. Function Balance Check

1-10화 권장 분포:

- Seojun inner monologue: 4곡
- Relationship theme: 4곡
- Ending reflection: 2곡
- Arin in-world song: 0곡

판단:
- 초반 10화는 아린의 창작곡을 아직 남발하지 않는 편이 좋다
- 대신 서준의 고립과 예외 감지를 음악적으로 먼저 누적한다

## 4. Vocal Direction

- E001-E005: male close vocal or gender-neutral restrained vocal
- E006-E010: mixed warmth allowed, but still mostly Seojun POV

주의:
- 초반부에 보컬이 너무 확신에 차면 안 된다
- `좋아해`, `운명`, `영원` 같은 직설은 금지한다
