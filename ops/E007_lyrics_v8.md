## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘 밤 나는 같은 노래를 또 틀었어" — 워밍업 0. 한국 청자가 즉시 자기 경험으로 환원할 수 있는 보편 행동 (밤에 같은 곡을 반복). thesis ("혼자 듣는 노래가 누구의 노래인지 알게 되는 밤") 의 행위가 첫 줄에 이미 시작되어 있고, "또"가 반복의 무게를 한 음절에 압축함. 검정치마 「강아지」류의 평범한 발화 안에 정서 균열을 심는 패턴.

### Chorus emotion line placement
Derived anchor "이 노래가 누구의 노래인지 알겠어" 를 chorus 1행에 verbatim 2회 배치 (chorus 1·2). Final chorus 의 마지막 라인만 변형 1회 허용 사용 — "이 노래는 내가 듣고 싶었던 노래였어" — 외부 지목("누구의")에서 자기 인정("내가")으로 한 발 떨어지면서 lift 작동 (restraint 안의 lift). chorus lock ≥2x 만족.

### Bridge subversion
"누구의 노래" 를 외부 인물로 향하던 첫 두 chorus 의 방향성을, bridge 에서 "사실은 나를 듣고 있었던 건지도" 로 방향 뒤집음. 한 사람을 떠올린다는 행위가 "내 안의 무언가가 켜진다" 라는 자기 점화로 재정의됨. bridge subversion 룰 충족 — 같은 표면(노래·한 사람)인데 의미축이 외부→내부로 옮겨짐.

### Five standout lines (audit)
1. "두 번째에는 가사가 들리기 시작했어" — 청취 점진성을 한 줄에 압축, 평이한 어휘.
2. "틀어 둔 한 곡이 자꾸 한 사람을 데려와" — pre-chorus motor line, 노래의 행위 주체를 곡으로 옮김.
3. "거기에 내 마음이 먼저 적혀 있었어" — chorus 종결, 가사가 자기보다 먼저 도착했다는 역설.
4. "누구한테 들킬까 봐 였는지 / 나한테 들킬까 봐 였는지" — verse 2 종결 2행 페어, 자기인지 회피의 핵심.
5. "불은 작은데 방을 다 비춰 버려서" — bridge 의 보편 비유, 사건 지목 0.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 캠퍼스/조별과제/같은 종이 → 제거 (v6 의 직접 motif 전면 폐기, thesis 가 "혼자 듣는 밤"이므로 v6 의 사회적 구성 motif 자체가 부적합)
- 이어폰 / 음향 차단 도구 → 제거 (lyricImageSeeds 의 "이어폰 너머" 는 abstract "노래" / "한 곡" 으로 환원)
- 공감각·청취 능력·sensory origin → 제거 (motif_audit_seojun_sensory_origin · self_hearing_prohibition 모두 OK — 청취 행위는 일반 청자 행위로만 표현, 공감각 0)
- 성별 표지 0건 (female intimate alto 락 유지, 페미닌 라인 자기관찰 톤)
- 클리셰 (비/새벽/우산/너의 향기/빈자리) → 0건
- 계절 직접 선언 → 0건 (보편 "밤" 한 단어만)
- borderline: "이불 위에 가만히 누워서" — 일반 시어, 클리셰 아님.

### Korean pronunciation pass (BPM 60 ambient-piano · sparse)
- BPM 60 의 늘어진 호흡에 맞춰 한 행 평균 음절수 12-16 로 길게 가져감, 밀도 분산.
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "또 틀었어 / 들렸어 / 들었어" 디귿/리을계열 의도적 motif, 발음 명확.
- "세 번째인지 네 번째인지" — Native 숫자 일관, Sino 혼용 0.
- "한 줄만 다시 돌려서 들었어" — 한 호흡 안 안정, ambient-piano 60 BPM 에서 여유.
- 영어/숫자(아라비아)/외래어 0건. 화법 일관 (반말 자기독백).
- jamo emoticon (ㅎㅎ/ㅋㅋ/ㅠㅠ) 0건. stage direction 0건.

### Phase 1 closure & form
- 총 47라인 (45-50 권고 안). 5 섹션 구조 (verse1 / pre-chorus / chorus / verse2 / chorus / bridge / final chorus) — Phase 1 의 자기독백 closure 형태 충족.
- Phase 1 락: 외부 인물 지목 없음, "한 사람"이라는 추상으로만 등장, 화자 자신의 인지 변화에 닫힘.

### Removed from v6 / why this lane shift
v6 은 song_brief 의 "같은 조 / 같은 종이" 캠퍼스 motif 를 따랐으나, `120_episode_genre_assignment.json` 의 E007 emotionalAnchor 는 "혼자 듣는 노래가 누구의 노래인지 알게 되는 밤" 이며 lyricImageSeeds 도 "이어폰 너머 / 혼자의 멜로디 / 알게 된 목소리 / 잠 못 드는 밤" — 사회적 신이 아닌 사적 청취 신. v8 은 후자(genre assignment thesis) 를 우선 채택 (사용자 지시 + universal register 호환성). 캠퍼스 motif 전면 제거.

## Unanswered for Conductor

1. **Brief vs Genre-assignment 충돌** — `E007_song_brief.md` 는 "같은 종이 / 조별과제 후 정서" 를, `120_episode_genre_assignment.json` 은 "혼자 듣는 밤" 을 지시함. 사용자 디렉터 지시("Thesis: 혼자 듣는 노래가 누구의 노래인지 알게 되는 밤")에 따라 후자 채택. brief 측 통합 원하시면 verse 2 를 "낮에 들은 한 줄이 밤에 다시 켜졌어" 류 라인으로 재작성 가능 (낮의 사건 → 밤의 청취 회수).
2. **변형 라인 1회 허용 사용 — final chorus 마지막 "이 노래는 내가 듣고 싶었던 노래였어"** 가 외부→내부 회수 lift 로 작동하는지는 Suno 생성 후 청취 판단. 약하면 verbatim 3회 ("이 노래가 누구의 노래인지 알겠어") 로 회귀 가능.
3. **여성 보컬 적합성** — verse 2 "이불 위에 가만히 누워서 천장을 봤어" / "입술만 움직였어" 류 자기관찰 라인은 female intimate alto 톤에 잘 맞춤. 가성 lift 없이 chest/mid 만으로 안착 가능한 음역 분포.
4. **Bridge 길이 (6행)** — E001 v8 동일 정책으로 의도적 짧은 bridge. 자기 점화의 발견을 닫지 않은 채 final chorus 로 떨어짐. 8-12행 권고대로 길게 원하시면 "이 작은 빛이 며칠을 갈지 / 며칠 가지 않을지 모르는 채로" 2행 추가 가능.
5. **Phase 1 락 검증** — 외부 인물 명시 0, 관계 사건 0, 화자 자신의 청취·인지 closure 만으로 곡이 닫힘. Phase 1 의 자기독백 register 충족.
