## Brief rationale (post-script)

- First-line hook: "오후의 빛이 책상 위로 한 칸씩 물러나고 있었어" — 시각적 정지 + 시간의 미세 이동을 한 줄에 담음. "한 칸씩 물러나고" 가 곡 전체의 호흡 메타포.
- Chorus lock: "빈자리가 먼저 와서 / 네 이름 없는 시간을 채웠어" verbatim 3회 (chorus 1, 2, final). 브리프 "-었다" → "-었어" 통일.
- Universal-imagery audit: "텅 빈 의자 / 시계 / 복도 / 화면" 보편 시어. "보내지 못한 한 줄" = unsent message 의 추상 처리. 운명/영원/없으면 못 살아 0건.
- Phase 2 register: 그리움이지만 melodrama 회피. "이 마음에 이름을 붙이지 않고" 로 early longing 만 표현, devastation 회피.
