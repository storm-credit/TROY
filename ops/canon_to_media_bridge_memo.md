# Canon to Media Bridge Memo

## 1. Purpose

이 문서는 `TROY` 소설의 잠긴 캐논을 music / image / MV media lane으로 옮길 때 흔들리기 쉬운 의미를 짧게 고정한다.

목표:

- media가 소설의 분위기를 키우더라도 의미를 바꾸지 않게 한다
- 감정의 크기와 장르의 강도를 올리면서도 캐논 왜곡을 막는다
- downstream 제작팀이 같은 금지선과 같은 강점을 공유하게 한다

## 3. Non-Negotiables

## 4. Emotional Promise

- 이 작품의 핵심 감정은 `과장된 폭발`보다 `정직한 잔향`이다
- 가장 강한 장면도 spectacle보다 intimacy가 우선이다
- 사랑의 증명은 소유나 영원 맹세가 아니라 `묻기`, `멈추기`, `물러서기`, `다시 말하기`다

## 5. Media Translation Rule

### Music

- refrain은 줄거리 요약보다 감정 잔향을 반복한다
- 후렴은 `영원`, `운명`, `완전한 회복`으로 미끄러지지 않는다
- 가장 상업적인 트랙도 감정 윤리를 훼손하지 않는다

### Image

- 캐릭터는 과장된 판타지 로맨스 아우라보다 현실적 거리감과 숨을 가진다
- 캠퍼스와 한강, 카페 달 같은 공간은 `기억의 공기`를 유지한다
- 감정은 표정 과잉보다 posture, gaze, spacing으로 표현한다

### MV

- MV는 재회 판타지나 능력 서사를 과장하지 않는다
- 컷의 힘은 접촉보다 `못 닿는 거리`, `같이 걷는 박자`, `멈춰 묻는 순간`에서 온다
- ending-side MV는 sequel bait보다 여운과 승인으로 닫는다

## 6. Fast Checks

media 초안에서 바로 확인할 질문:

- 이 표현이 관계를 더 정직하게 만들었는가, 더 판타지처럼 만들었는가?
- 이 장면이 멋있어졌지만 의미는 얕아지지 않았는가?
- 이 곡/이미지/MV가 소설보다 더 큰 약속을 하고 있지 않은가?

## 7. Director Note

- media는 소설을 더 크게 들리게 할 수는 있어도 다른 이야기로 바꾸면 안 된다
- 가장 좋은 media는 소설의 의미를 확대하지, 교체하지 않는다
