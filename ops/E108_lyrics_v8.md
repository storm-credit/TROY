## Rationale (post-script, not pasted to Suno)

### First-line hook design
"다 알지 못한 채로 옆에 있어 보기로 했어" — Phase 6 livable uncertainty 의 정의 자체를 첫 줄에 압축. "다 알지 못한 채로"가 즉시 livable uncertainty 를 발화하고 "옆에 있어 보기로 했어" 가 결심형으로 닫지 않은 채 진행형 ("-보기로 했어") 으로 열어 둠. v6 의 "오늘은 확신이 없는 채로" 보다 한 단계 더 보편적이고 한 단계 더 인간적 — "확신/없는" 의 추상 명사 어휘 대신 "다 알지 못한 채로" 의 일상 화법으로 환원. 워밍업 0.

### Chorus emotion line placement
Locked phrase "들리지 않아도 알아 가는 마음" 은 chorus 1·2·final 모두 1행에 verbatim 3회. 변형 1회 lift 는 final chorus 마지막 행 "나는 오늘 손을 빼지 않고 걸았어" 로 사용 — 1·2 chorus 의 "확신이 줄어든 만큼 / 태도는 더 분명해졌어" 정리 진술을 final 에서 "손을 빼지 않은 쪽이 먼저 도착하는 사람이라 / 나는 오늘 손을 빼지 않고 걸었어" 의 행위 진술로 lift, 결심을 결과로 닫음.

### Five standout lines (album-track polish)
1. **첫 행** "다 알지 못한 채로 옆에 있어 보기로 했어" — livable uncertainty 의 정의 한 줄 발화.
2. **verse 1 후반** "확실한 사람이 되려고 애쓰는 일은 / 어쩌면 가장 멀리 가는 일이었어" — Phase 6 의 미션 진술. 확신 추구 = 거리 만드는 일이라는 역설.
3. **pre-chorus 후행** "불안하지 않은 게 사랑이 아니라 / 불안한 채로도 손을 빼지 않는 일이라" — 보편 사랑 정의 재정의, 청자가 자기 관계로 환원 가능.
4. **bridge 첫 두 행** "나는 이해받고 싶었던 게 아니라 / 이해하지 못해도 같이 있고 싶었던 거였어" — bridge subversion 의 핵심. 화자의 욕망 자체가 뒤집힘.
5. **final chorus 끝 행** "손을 빼지 않은 쪽이 먼저 도착하는 사람이라 / 나는 오늘 손을 빼지 않고 걸었어" — 행위로 결산. 결심 → 결과 lift.

### Bridge subversion
v6 bridge 는 "모르는 게 부끄러운 일이 아니라는 걸 / 같이 배우는 중이야" 의 학습 진술 — 정확하지만 평이. v8 bridge 는 화자 자신의 구조를 뒤집음: 이해받는 쪽이 아니라 이해하지 못한 채로 같이 있는 쪽으로. "모르는 채로 옆에 있는 일이 / 모른 척하는 일과 다르다는 걸 / 오늘에야 천천히 구분할 수 있었어" 가 핵심 — '모르는' 과 '모른 척' 의 윤리적 구분이 처음으로 가능해진 순간. 이는 단순한 학습이 아니라 화자의 도덕 구조 자체가 갱신된 것.

### Universal-imagery audit
- 작중 보행로/가을길 직접 묘사 0건 ("같은 길" 한 단어로만 추상화)
- "그게 정상이에요" 작중 대사 직접 인용 회피, "정상이라고 말해 준 그 한 마디" 한 단계 추상화
- miracle / certainty / hearing-regained 어휘 0건
- 공감각 / sensory origin / 청취 능력 0건 — "들리지 않아도" 는 chorus locked phrase 안에서만 사용, 그 외 라인은 "다 보이지 않아도", "다 알지 못한 채로", "다 말하지 않아도" 의 인지적 부재 어휘로 분산해 sensory-origin 인접 위험 분산.
- closure-by-reunion 0건 — final chorus 는 "걸었어" 의 진행 동사로 닫음, 도착/안착 어휘 회피.
- triumph 0건 — "분명해졌어" / "받쳐 줬어" / "걸었어" 의 미세한 안도 톤만, lift 없음.
- 클리셰 (비/별/빈자리/그림자/계절) 0건.

### Korean QA pass
- 자음 군집 위험 0건. 빠른 된소리 hook 없음.
- "확신/태도/이해" 단단한 받침이 chamber-pop patient walking pace 에 자연스럽게 정착. BPM 75 fit.
- "들리지 않아도 / 알아 가는 마음" 한 호흡 안 안정, "다 보이지 않아도 / 옆에 있는 일" 도 동일 음절 구조로 hooking 강화 (chorus 1·2행이 동일 박자 패턴).
- 영어/숫자/외래어/jamo emoticon 0건. 화법 일관 (반말 자기독백, 종결어미 "-했어/-였어/-었어" 통일).
- "이해받고 / 이해하지 못해도" 디귿계열 발음 안정, "구분할 수 있었어" 어미 명료.
- "들리지 않아도" — sensory 직역 위험 회피 위해 chorus 안 "알아 가는 마음" 과 결합되어 등장. "다 보이지 않아도 옆에 있는 일" 가 같은 행 옆에 들어가서 청취 일변도 해석 차단 (시각/존재 어휘 병치).

### Removed / changed from v6
- v6 첫 줄 "오늘은 확신이 없는 채로 옆에 있어 보기로 했어" → v8 "다 알지 못한 채로 옆에 있어 보기로 했어" (추상 명사 → 일상 화법, 더 보편).
- v6 verse 1 후반 7-8행이 다소 진술적·교훈적 ("틀린 게 끝이 아니라는 걸 이제는 알아") → v8 "확실한 사람이 되려고 애쓰는 일은 / 어쩌면 가장 멀리 가는 일이었어" 로 역설형 standout 라인 삽입.
- v6 chorus 2행 "다 알지 못한 채로 같이 걷는 일" → v8 "다 보이지 않아도 옆에 있는 일" 로 감각 어휘 분산 (heard/said motif 균형).
- v6 bridge 학습 진술 톤 → v8 화자 욕망 구조의 뒤집기 (subversion 강화).
- v6 final chorus 끝 "나는 오늘 손을 빼지 않았어" → v8 "나는 오늘 손을 빼지 않고 걸었어" — "걸었어" 진행 동사로 닫아 closure 회피, "걸었어"가 보행 모티프 echo 까지 살림.
