## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오늘 누가 너의 이름을 아무렇지 않게 불렀어" — 한국 청자가 즉시 자기 경험으로 환원할 수 있는 보편 트리거. 시작부터 "감정 사건"이 이미 발생한 상태, "아무렇지 않게"라는 한 단어가 표면 평온과 내면 균열을 동시에 작동시킴 (E001 v8 의 "잘 지내냐는 말에 잘 지낸다고" 패턴 — 평범한 발화 안의 정서 균열). 워밍업 0. 작품 캐릭터 명시 없이 "너"만 둔 이름-부재 hook.

### Chorus emotion line placement
Locked phrase "이름 하나가 / 다른 무게로 떨어졌어" 는 chorus 1·2 행에 verbatim 2회 (chorus 1·2). final chorus 의 마지막 lift 라인은 자연 변형 "이름의 무게로 한 번 더 들어 보았어" 로 종결 — 같은 동사("들다") 어간을 무게 쪽으로 기울여 음악적 안착 (restraint 안에서의 lift). 변형 1회만 허용된 룰 안에서 사용. 핵심 동사 "떨어지다 / 들다 / 받아 들다" 의 손-감각 라인이 chorus 전체를 연결.

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 송유빈/지아린/캠퍼스/카페/창가 → 0건. 이름 자체가 보편 단어로만 존재.
- v6 "테이블/컵/입술/발끝" 직접 도구 → 제거. "손바닥" 한 번만 (chorus 의 받아 든 손 — 추상 손).
- 가을/계절 직접 선언 → 제거 (Phase 5 그레인은 BPM/lane 으로 위임).
- 클리셰 (비/새벽/우산/너의 향기/빈자리) → 0건.
- 이상화/폄하 risk → "너"는 이름의 주인일 뿐, 평가 없음. 비교 상대 부재.
- borderline: "작은 금이 하나 생겼어" — 「금」은 일반 시어, 클리셰 아님, 정서 작동 OK.
- v6 standout "편안한 자리는 도망의 다른 이름이 될 수도 있고" → bridge 1행에 verbatim 보존. 두 번째 행 ("다시 듣는 이름은 처음의 다른 이름이 될 수도 있어서")로 thesis 와 연결되도록 subversion.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 빠른 된소리 hook 없음.
- "다른 무게 / 다른 이름 / 다르게" — ㄷ 계열 의도적 motif. 발음 명확.
- "받아 들었어 / 받아 든 / 들어 보았어" — "듣다/들다" 동사 의도 반복. chorus motif 기능.
- "이름 하나가 / 다른 무게로 떨어졌어" — 한 호흡 안에 들어감 확인. 모음 개방형 (아/우/어) 으로 chorus 안착.
- 영어/숫자/외래어 0건. 화법 일관 (반말 자기독백).
- "아무렇지 않게" — 4음절 약간 길지만 BPM 70 lofi-bedroom 에서 안정. 첫 줄 hook 으로 핵심 정서 전달.
- 자모 emoticon (ㅎㅎ/ㅋㅋ/ㅠㅠ) 0건. 스테이지 디렉션 0건.

### Removed from v6
- "오늘 누군가 옆에서 편안한 침묵을 내어 줬어" → 1행 1행이 시적이지만 thesis ("다른 무게") 비집중. "오늘 누가 너의 이름을 아무렇지 않게 불렀어"로 교체 (이름-무게 hook 로 직진).
- "컵 위의 손이 거의 움직이지 않는 오후였고" → 도구(컵) 직접 묘사. "받아 든 이름이 손바닥 위에서 한 번 기울었어"로 추상 손-감각 치환.
- "사랑이라는 말은 빈 공간을 채우는 단어가 아니어서" → "사랑" 직접 단어 호출. 본 thesis 는 "이름 자체의 무게" 이므로 "사랑"은 chorus 에서 제외, 단어-무게 라인으로 통합.
- v6 chorus "편안하다고 해서 / 그게 끝내 사랑인 것은 아니었어" → thesis 가 "편안함≠사랑" 에서 "재청취된 이름의 다른 무게" 로 이동했으므로 "이름 하나가 / 다른 무게로 떨어졌어"로 교체. 편안함 단어는 bridge 1행에만 보존 (v6 standout 락).
- v6 verse 3 "누군가의 다정함은 그대로 다정함이라 부르고 / 나의 흔들림은 그대로 흔들림이라 부르기로 했어" → thesis 와 직접 연결 약함. "누가 다시 부른다고 해서 다시 안 게 되는 건 아니어서" 로 재청취-인식 분리 라인으로 교체.
- v6 outro "컵을 다 비우지 않은 채로 오후를 비워 두고 / 나는 가만히 자리에서 일어났어" → 컵 도구 직접. final chorus 안 "이름의 무게로 한 번 더 들어 보았어"에 흡수 (50 라인 안에 압축, outro 별행 제거).

## Five standout lines (v8 craft self-audit)

1. "받아 든 이름이 손바닥 위에서 한 번 기울었어" (verse 1) — 이름의 무게를 손-감각으로 이미지화. 추상-구체 brigde.
2. "어떤 단어는 부르는 사람마다 다르게 떨어져" (verse 2) — thesis 의 핵심 일반화. 청자 누구나 자기 이름으로 환원 가능.
3. "다시 듣는 이름이 처음 듣는 이름처럼 낯설어서" (pre-chorus) — Phase 5 correction · re-listening 의 정서 그레인 직격.
4. "들은 것과 안 것 사이에 작은 자리를 비워 두고" (verse 3) — "다시 만났다고 다시 안 건 아니다" 인식의 미사용 자리 — 결정 유보의 시 (詩).
5. "오늘은 이름을 이름으로 부르지 않고 / 이름의 무게로 한 번 더 들어 보았어" (final chorus) — 변형 lift, restraint 안에서의 1-variant 종결.

## Bridge subversion check

v6 standout "편안한 자리는 도망의 다른 이름이 될 수도 있고" 를 bridge 1행에 verbatim 보존, 2행 "다시 듣는 이름은 처음의 다른 이름이 될 수도 있어서"로 thesis 결합. 같은 "다른 이름" 패턴을 두 번 반복하되, 첫 번째는 자기-도망 (회피의 의심), 두 번째는 재청취-갱신 (인식의 가능성) — bridge 안에서 의미 방향이 뒤집히는 subversion. 결정 유보 ("어느 쪽도 빨리 결정하지 않기로 했어") 로 닫지 않고 final chorus 로 떨어트림.

## Final 1-variant lift

final chorus 마지막 행 "이름의 무게로 한 번 더 들어 보았어" — chorus verbatim "이름 하나가 / 다른 무게로 떨어졌어" 의 핵심 단어("이름 / 무게 / 들다")를 보존하면서 동사 시점만 "떨어졌어" (수동 완료) → "들어 보았어" (능동 시도) 로 미세 전환. 정서적으로는 "당하던 무게"에서 "스스로 들어 보는 무게" 로의 lift. Suno 생성 후 lift 약하면 verbatim 3회 ("나는 받아 든 손을 한참 내려놓지 못했어")로 회귀 가능.

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus "이름의 무게로 한 번 더 들어 보았어"** 가 음악적 lift 로 작동하는지는 Suno 생성 후 청취 판단 필요. lift 약하면 verbatim 3회로 회귀.
2. **여성 보컬 적합성** — verse 2 "어떤 단어는 부르는 사람마다 다르게 떨어져" 류 관조 라인은 female intimate alto withdrawn 톤에 정합. 작중 아린-보컬 인-월드 룰 (CLAUDE.md §4) 충족.
3. **Bridge 길이 (5행)** — 8-12행 권고 하한이지만, 의도적으로 짧게 가져가 결정 유보가 마무리 짓지 않은 채 final chorus 로 떨어지게 함. 더 길게 원하시면 2행 추가 가능 ("결정하지 않은 자리에 작은 빛이 한 번 떨어졌고 / 빛은 무게가 없는데도 자리만큼 따뜻했어").
4. **Outro 제거 결정** — v6 의 컵-비우기 outro 는 final chorus 안 "이름의 무게로 한 번 더 들어 보았어"에 의도적으로 흡수 (행위의 부정형 — 더 정직). outro 별행 필요하면 fade 라인 추가 가능.
5. **45-50 라인 카운트** — Suno paste block 본문 47행 (섹션 태그 제외 가사 라인 47). 권고 범위 안.
