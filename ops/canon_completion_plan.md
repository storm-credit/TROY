# Canon Completion Plan

## 1. Director Decision

설정집은 이미 `집필 시작 가능` 수준이다.  
하지만 사용자가 원하는 것은 `최대한 끓여올린 뒤 집필`이므로, 이제는 `끝없는 확장`이 아니라 `집필 직전 최종 잠금 라운드`로 관리한다.

핵심 원칙:
- 새 설정을 계속 늘리지 않는다
- 원고에 직접 영향을 주는 빈칸만 메운다
- 잠금이 끝나면 바로 원고로 넘어간다

## 2. Already Locked

- ending model: beautiful separation
- ending meaning: listening loss -> completion of silence
- total structure: 120 episodes / 24 chapters / 6 movements
- one episode = one song
- base visual mode: cinematic live-action
- public campus: Hyedam University humanities campus
- core couple dynamic: receiver vs sender

## 3. Final-Boil Targets

집필 전 마지막으로 잠가야 할 것은 아래 다섯 가지다.

1. support cast final scope
2. campus and daily-life operation details
3. asset naming and episode packaging rules
4. prose house bible expansion
5. ending-scene physical form

## 4. What Counts As Complete

아래 조건이 충족되면 설정집은 `final simmered` 상태로 본다.

- core/support cast 역할이 겹치지 않는다
- 장소가 story-facing language로 통일된다
- episode id naming rule이 흔들리지 않는다
- 원고 문체 금지선이 분명하다
- 마지막 장면의 장소, 거리감, 대화 톤이 고정된다

## 5. Manuscript Start Gate

원고 시작 게이트는 아래 셋이 모두 yes일 때 열린다.

1. character scope gate: yes
2. place / naming gate: yes
3. prose / ending gate: yes

## 6. Final Recommendation

이 문서 기준으로 마지막 잠금 라운드를 마친 뒤에는, 설정집을 더 늘리지 말고 `E001 manuscript`부터 시작한다.
