# TROY 120 Song Mood Archetypes v1

> Light : Mid : Deep = 4 : 4 : 2 (총 10 개). 사용자 피드백 "OST 라 노래가 다 어두울까 봐 그러함" 직접 대응.
> 모든 archetype 은 Lyrics Director (Songwriter mode) brief 의 craft 기준 통과 — Hook latency 0, lock 라인 ≤10 음절, ㄷ/ㅇ/ㅎ 가족 alliteration, 도메스틱 사물, 자기의문-발견 bridge, 1-variant final lift.
> Reference panel 은 §reference only — Suno prompt 에는 절대 미포함.
> Dark family 는 만들지 않는다. Deep family 도 livable — 결단·침묵·정직한 그리움까지만, 절망은 안 가져간다.

## 사용 룰

1. **Episode 별 1 archetype 어사인** — Conductor 가 Phase 정서 + Lane 어사인 후 이 팔레트에서 archetype 1 개 픽. Lyrics Director (Songwriter mode) brief 에 `archetype: <name>` 한 줄로 전달.
2. **Lane 과 archetype 은 독립 축** — Lane 은 음향, archetype 은 정서 결. 같은 archetype (예: "두고 온 데") 이 `korean-indie-folk` 로도 `lofi-bedroom` 으로도 입혀질 수 있음.
3. **Phase 분포 강제** — 한 Phase (20 곡) 안에 light ≥ 5 곡, deep ≤ 10 곡. Conductor 가 Phase 별 카운터 운영.
4. **Hygiene check 는 archetype 무관 적용** — banned moves 는 archetype 별이지만, 작품 신·고유명·클리셰·영어/숫자 0건 룰은 모든 archetype 공통.
5. **120 곡 전체 분포 (작전 계획)** — Light 32-36 곡 / Mid 56-58 곡 / Deep 28-30 곡 목표. v1 출시 후 60 곡 시점에 분포 audit.

## Family overview

### Light family (4 archetypes, 32-36 곡 cover)
- **A1 아침 햇살 (Soft Morning Warmth)** — 일어났더니 괜찮은 날, 작은 감사
- **A2 산책의 결 (Walking Pulse)** — 다정한 걸음, 길에서 마주친 사소한 발견
- **A3 멋쩍은 미소 (Smile-in-Song)** — 자기 자신이 우스워서 슬며시 웃는 톤
- **A4 작은 자유 (Quiet Freedom)** — 무거운 걸 내려놓은 직후, 가벼운 결단

### Mid family (4 archetypes, 56-58 곡 cover)
- **B1 두고 온 데 (Unnamed Longing)** — 부재한 대상에의 그리움 (E002 v10 의 고향)
- **B2 가까이 본 것 (Close-Mic Noticing)** — 친밀한 관찰, 사소한 것이 커다래지는 순간
- **B3 작은 발견 (Quiet Surprise)** — 자기 안에서 뭔가 처음 들리는 톤
- **B4 다정한 자기 (Self-Tenderness)** — 자기 위로, 다정한 자기독백

### Deep family (2 archetypes, 28-30 곡 cover)
- **C1 닫고 산 마음 (Restrained Ache)** — 절제된 슬픔, 자기기만의 정직한 고백 (E001 의 고향)
- **C2 조용한 결단 (Silent Resolve)** — 침묵을 받아들이는 결의, NOT despair

## Archetype A1 — 아침 햇살 / Soft Morning Warmth

### Family
light

### Mood territory
잘 잤다. 커튼 사이로 빛이 들어오고, 오늘은 어쩐지 괜찮을 것 같은 그 작은 예감.

### Reference tracks (calibration only)
- 시와 (일곱 번째 노래) — paper-thin nylon, 일상의 작은 빛
- 강아솔 (정직한 마음) — fingerpicked nylon, 따뜻한 자기 인정
- 옥상달빛 (없는 게 메리트) — bright domestic, 가볍지만 깊이 있는 톤
- 정밀아 (그리운 그대) — first-phrase intimate warmth
- 백예린 (Bunny) — close-mic alto, 다정한 일상감

### Hook signature
- 진입 방식: action verb (일어났다 / 열었다 / 마셨다 류)
- 음절 범위: 8-12 음절
- 1인칭 위치: 막 일어난 / 부엌에 서 있는 / 창문을 여는 중
- 예시 hook vibes:
  1. "오늘은 조금 늦게 일어났어"
  2. "물을 끓이면서 콧노래를 했어"
  3. "이불을 개는데 햇살이 들었어"

### Chorus shape
- Lock 라인 자리: 1st line
- Lock 라인 음절 범위: 7-9 음절
- Alliteration motif 추천: ㅎ 가족 (햇/하/한/홀)
- Lift mechanic: single-word swap (형용사 한 칸 따뜻한 쪽으로)

### Vocal posture
- 호흡 패턴: walking warm
- 화법: 반말 자기독백
- Dynamic range: light-bright (still restrained, never anthemic)

### Tempo & Lane affinity
- BPM 범위: 72-82
- 1순위 lane: `korean-indie-folk`
- 2-3 순위 lane: `bossa-folk`, `chamber-pop`
- 금지 lane: `ambient-piano`, `folk-electronic`, `lofi-bedroom` (어두운 톤으로 끌림)

### Standout image palette (verse texture seeds)
- 커튼, 머그컵, 식빵, 양말, 슬리퍼, 부엌 창문, 화분

### Bridge subversion model
다정한 자기 발견: "어쩌면 오늘 같은 날이 자주 와도 되겠다" 류 — 좋음을 받아들이는 작은 허락.

### Banned moves
- 어두운 마무리 (final 에서 가라앉지 말 것)
- 슬픔으로 회귀하는 lift
- 운명·기적 어휘
- 추상명사 chorus (빛·결 등)
- "그리움" 단어 직접 등장
- ㅠㅠ 톤의 자기연민
- 새벽·밤 풍경 (이 archetype 은 아침-낮 only)

### Sample subtitle
- "오늘은 조금 늦게"
- "햇살이 든 부엌"
- "콧노래 비슷한 것"

## Archetype A2 — 산책의 결 / Walking Pulse

### Family
light

### Mood territory
신발 끈을 묶고 나간다. 어디로 가는지는 모르지만 걷는다. 길에서 마주친 사소한 것들이 노래가 된다.

### Reference tracks (calibration only)
- 이바다 (산책) — walking-tempo female nylon
- 선우정아 (구애) — walking-but-bittersweet
- 김사월 (스물다섯) — walking-pulse alto
- 정밀아 (그리운 그대) — breath-forward walking
- 강아솔 (정직한 마음) — fingerpicked walking warmth

### Hook signature
- 진입 방식: action verb (걸었다 / 멈췄다 / 봤다)
- 음절 범위: 9-13 음절
- 1인칭 위치: 길 위 / 횡단보도 / 골목 모퉁이
- 예시 hook vibes:
  1. "신발 끈을 두 번 다시 묶었어"
  2. "골목에서 누가 빨래를 널었어"
  3. "한 정거장 전에 그냥 내렸어"

### Chorus shape
- Lock 라인 자리: 1st line
- Lock 라인 음절 범위: 7-10 음절
- Alliteration motif 추천: ㄱ/ㄷ 혼합 (걷/길/그/가)
- Lift mechanic: particle inversion (-도 → -은)

### Vocal posture
- 호흡 패턴: walking warm
- 화법: 반말 자기독백
- Dynamic range: mid (bittersweet-bright)

### Tempo & Lane affinity
- BPM 범위: 78-88
- 1순위 lane: `bossa-folk`
- 2-3 순위 lane: `korean-indie-folk`, `dream-pop` (Var B)
- 금지 lane: `ambient-piano`, `90s-vintage-ballad`

### Standout image palette
- 신발 끈, 횡단보도, 빨래, 골목, 정거장, 우편함, 동전

### Bridge subversion model
미세한 발견: 걷다가 어느 순간 "내가 어딘가 가고 있다는 것" 자체가 충분하다는 깨달음. 목적지 없음이 결함이 아니라 자유로 뒤집힘.

### Banned moves
- 거대한 declarations ("나는 살아 있어" 류)
- 슬픔의 무게 잡기
- 계절 직접 선언
- 비/눈 등 날씨 클리셰
- 외국어 / 외래어
- ballad gloss (이 archetype 으로 절대 ballad lane 안 감)

### Sample subtitle
- "신발 끈 두 번"
- "한 정거장 전"
- "빨래가 길었어"

## Archetype A3 — 멋쩍은 미소 / Smile-in-Song

### Family
light

### Mood territory
자기 자신이 좀 우스워서 슬며시 웃는다. 누가 봐도 사소한 일에 혼자 진지했던 자신을 돌아보며 다정하게 놀린다.

### Reference tracks (calibration only)
- 옥상달빛 (수고했어 오늘도 — light mode) — domestic warmth, 자기 다독임
- 백예린 (Square) — narrow stereo bedroom warmth
- 권진아 (감기) — intimate confession with smile
- 김사월 (농담) — bedroom-quiet female folk, 우스움 포함
- 헤이즈 (And July acoustic) — soft-glide warmth

### Hook signature
- 진입 방식: declarative + 자조적 어미 (-더라 / -었어 / -았네)
- 음절 범위: 10-13 음절
- 1인칭 위치: 거울 앞 / 일기장 앞 / 누군가 떠난 뒤 자기 방
- 예시 hook vibes:
  1. "혼자 한참 화를 내고 있었네"
  2. "거울을 보다가 피식 웃었어"
  3. "그게 뭐라고 그렇게 고민했지"

### Chorus shape
- Lock 라인 자리: 2nd line (1st line 으로 세팅 후 2nd 가 lock)
- Lock 라인 음절 범위: 8-10 음절
- Alliteration motif 추천: ㅎ/ㅁ 가족 (혼자/한참/뭐가/멋쩍/미안)
- Lift mechanic: 어미 시제 미세 이동 (-더라 → -더라고)

### Vocal posture
- 호흡 패턴: intimate withdrawn (still warm, slight smile in voice)
- 화법: 반말 자기독백 (살짝 자조)
- Dynamic range: light-bright

### Tempo & Lane affinity
- BPM 범위: 70-78
- 1순위 lane: `lofi-bedroom`
- 2-3 순위 lane: `korean-indie-folk`, `bossa-folk`
- 금지 lane: `ambient-piano`, `chamber-pop`, `90s-vintage-ballad` (너무 무거워짐)

### Standout image palette
- 거울, 일기장, 머리카락, 양말 짝, 이불, 베개 자국, 안경

### Bridge subversion model
자기 다독임으로 발전: "내가 너무 진지했네" 의 자조가 "근데 그게 또 나야" 로 슬쩍 풀림. 자기 수용의 가벼운 lift.

### Banned moves
- 진지한 declarations
- 슬픔 모드 회귀
- 클리셰 ("나는 나야" 류)
- 영어 / 숫자
- 운명 어휘
- 추상명사 chorus
- 외부 대상에 대한 분노 (이 archetype 은 자기 자신만 대상)

### Sample subtitle
- "혼자 화내던 밤"
- "거울 앞에서 피식"
- "그게 뭐라고"

## Archetype A4 — 작은 자유 / Quiet Freedom

### Family
light

### Mood territory
무거운 걸 하나 내려놓은 직후. 큰 결단 아니다. 그냥 "안 하기로 했어" 의 작은 가벼움. 발이 가볍다.

### Reference tracks (calibration only)
- 새소년 (그대로) — restrained acoustic, 자기 결단 lean
- 권진아 (운이 좋았지) — chamber, 가벼운 인정
- 백예린 (0310) — Rhodes-tinted soulful, 가벼운 lift
- 김사월 (진주) — withdrawn alto, 결단의 결
- 강아솔 (정직한 마음) — fingerpicked, 정직한 가벼움

### Hook signature
- 진입 방식: declarative (-기로 했어 / -지 않기로 했어)
- 음절 범위: 9-12 음절
- 1인칭 위치: 책상 앞 / 문 앞 / 무언가를 닫은 직후
- 예시 hook vibes:
  1. "오늘은 그냥 안 가기로 했어"
  2. "전화기를 뒤집어 놓고 잤어"
  3. "답장은 내일 하기로 했어"

### Chorus shape
- Lock 라인 자리: 1st line
- Lock 라인 음절 범위: 8-10 음절
- Alliteration motif 추천: ㅇ 가족 (안/없/잊/이제)
- Lift mechanic: past-perfect verb shift (-기로 했어 → -기로 했더라)

### Vocal posture
- 호흡 패턴: gentle determined
- 화법: 반말 자기독백
- Dynamic range: mid (light but with quiet weight)

### Tempo & Lane affinity
- BPM 범위: 72-80
- 1순위 lane: `korean-indie-folk`
- 2-3 순위 lane: `k-soul-quiet`, `chamber-pop`
- 금지 lane: `ambient-piano` (너무 가라앉음), `folk-electronic` (긴장 lane)

### Standout image palette
- 책상, 전화기, 의자, 신발장, 문고리, 봉투, 캘린더

### Bridge subversion model
결단의 확장: 작은 안 하기 → "안 해도 괜찮은 것들이 많았네" 의 발견. 자유의 범위가 살짝 넓어지는 자기 의문-발견.

### Banned moves
- 영웅적 declarations ("자유야!" 류 절대 X)
- 분노·복수 톤
- 운명·기적 어휘
- 외국어
- 추상명사 chorus
- 슬픔의 회귀 (이 archetype 은 끝까지 가벼워야 함)
- 누군가를 떠나는 직접 묘사 (이별곡 아님)

### Sample subtitle
- "안 가기로 한 날"
- "전화기를 뒤집고"
- "답장은 내일"

## Archetype B1 — 두고 온 데 / Unnamed Longing

### Family
mid

### Mood territory
이유를 모르는 그리움. 사람도 시절도 정확히 짚을 수 없는, 어딘가에 뭘 두고 온 것 같은 감각. **E002 v10 ("두고 온 데가 있어") 의 home archetype.**

### Reference tracks (calibration only)
- 김사월 (수많은 이야기) — withdrawn alto, 부재의 결
- 권진아 (끝없는 밤) — felt-grand intimate, no-uplift
- 백예린 (Los Angeles) — close-mic alto, 부재한 장소
- 정밀아 (그리운 그대) — first-phrase intimate longing
- 새소년 (집에) — hazy alto, drift toward absent

### Hook signature
- 진입 방식: action verb that reveals absence (펴다 / 더듬다 / 멈추다)
- 음절 범위: 9-13 음절
- 1인칭 위치: 길 위 / 부엌 / 정류장 — 일상 중 멍해진 순간
- 예시 hook vibes:
  1. "주머니를 자꾸 더듬었어"
  2. "국이 식는 줄도 몰랐어"
  3. "버스를 한 대 더 보냈어"

### Chorus shape
- Lock 라인 자리: 1st line
- Lock 라인 음절 범위: 6-9 음절
- Alliteration motif 추천: ㄷ 가족 (두/데/다/닫/닿)
- Lift mechanic: tense/aspect shift (-있어 → -갈래 류 결단 전환)

### Vocal posture
- 호흡 패턴: intimate withdrawn
- 화법: 반말 자기독백
- Dynamic range: mid (restrained)

### Tempo & Lane affinity
- BPM 범위: 68-78
- 1순위 lane: `korean-indie-folk`
- 2-3 순위 lane: `acoustic-ballad`, `lofi-bedroom`
- 금지 lane: `bossa-folk` (너무 walking 화함), `folk-electronic` (긴장 결이 다름)

### Standout image palette
- 우산, 주머니, 국, 숟가락, 정류장, 골목, 봉투

### Bridge subversion model
자기 의문 → 결단: "거긴 없는 데일지도 몰라" 의 인정이 그리움을 죽이지 않고 "그래도 갈래" 로 발전. 부재의 인정이 결단의 lift 로.

### Banned moves
- 누구를 그리워하는지 명시 (대상 익명 유지)
- "보고 싶어" 직접 표현
- 별·실·끈·우주 어휘
- 헤어진 사람 묘사
- ballad gloss
- anthemic final
- 외국어

### Sample subtitle
- "두고 온 데"
- "이름도 모르는 데"
- "주머니를 더듬다"

## Archetype B2 — 가까이 본 것 / Close-Mic Noticing

### Family
mid

### Mood territory
친밀한 관찰. 평소 안 보이던 사소한 것이 갑자기 커다래지는 순간. 그릇의 금, 누군가의 옆모습, 머리카락 한 올.

### Reference tracks (calibration only)
- 백예린 (Square) — close-mic 일상 관찰
- 김사월 (수많은 이야기) — withdrawn observation
- 시와 (일곱 번째 노래) — paper-thin, 작은 것의 큰 무게
- 강아솔 (정직한 마음) — fingerpicked, 정직한 관찰
- 권진아 (감기) — intimate close-mic noticing

### Hook signature
- 진입 방식: observation (보았어 / 알았어 / 들었어)
- 음절 범위: 10-14 음절
- 1인칭 위치: 마주 앉은 / 옆에 선 / 문 너머에서
- 예시 hook vibes:
  1. "컵 가장자리 작은 금을 봤어"
  2. "네 옆모습이 오늘은 좀 달랐어" (※ "너" 는 익명 you 허용 — 작중 인물 매핑 금지)
  3. "머리카락 한 올이 떨어져 있었어"

### Chorus shape
- Lock 라인 자리: paired (1st + 3rd line echo)
- Lock 라인 음절 범위: 8-10 음절
- Alliteration motif 추천: ㄱ/ㅈ 가족 (가깝/가만/작은/잘)
- Lift mechanic: single-word swap (관찰 → 인정)

### Vocal posture
- 호흡 패턴: intimate withdrawn
- 화법: 반말 자기독백
- Dynamic range: restrained

### Tempo & Lane affinity
- BPM 범위: 68-78
- 1순위 lane: `lofi-bedroom`
- 2-3 순위 lane: `korean-indie-folk`, `chamber-pop`

## Archetype C1 — 닫고 산 마음 / Restrained Ache

### Family
deep

### Mood territory
절제된 슬픔. 큰 통곡 아니다. 자기를 닫아 두고 살아온 것을 정직하게 들여다보는 톤. **E001 ("내 귀부터 닫고 살았어") 의 home archetype.**

### Reference tracks (calibration only)
- 김사월 (수많은 이야기) — withdrawn alto, dry close
- 권진아 (끝없는 밤) — felt-grand intimate, no-uplift restraint
- 박혜진 박하잎 (어제 너는 내 생각이 났을까) — felt grand + intimate, no swell
- 새소년 (집에) — hazy alto, restrained ache
- 정밀아 (그리운 그대) — breath-forward restrained

### Hook signature
- 진입 방식: declarative + 자기관찰 (-었어 / -아 살았어)
- 음절 범위: 11-14 음절
- 1인칭 위치: 자기 자신을 들여다보는 / 멈춰 있는
- 예시 hook vibes:
  1. "잘 지내냐는 말에 잘 지낸다고 답했어"
  2. "괜찮다는 말이 입에서 자꾸 굳었어"
  3. "오래 닫아 둔 문 앞에 서 있었어"

### Chorus shape
- Lock 라인 자리: 2nd line (1st 가 setup, 2nd 가 lock)
- Lock 라인 음절 범위: 9-10 음절
- Alliteration motif 추천: ㄷ 가족 (닫/닿/다/들)
- Lift mechanic: past-perfect verb shift (-고 살았어 → -아 두었어)

### Vocal posture
- 호흡 패턴: intimate withdrawn (deepest)
- 화법: 반말 자기독백
- Dynamic range: restrained (절대 anthemic 안 됨)

### Tempo & Lane affinity
- BPM 범위: 65-75
- 1순위 lane: `korean-indie-folk`
- 2-3 순위 lane: `acoustic-ballad`, `90s-vintage-ballad`
- 금지 lane: `bossa-folk`, `dream-pop` (이 톤은 walking·hazy 아님)

### Standout image palette
- 이불, 거울, 창문, 문, 방, 같은 자리, 표정

### Bridge subversion model
자기 의문-발견: "어쩌면 나는 듣고 싶었던 건지도" 의 자기 노출 → "그게 다정인지 도망인지 모르겠어" 의 정직한 양가. 결론 안 내리고 final chorus 로 떨어짐.

### Banned moves
- 거대한 통곡 / power-ballad swell
- 운명·기적·별·실·끈
- 외부 대상 비난
- 영어 / 숫자 / 외래어
- 작품 신 직접 묘사 (이어폰·강의실·캠퍼스·서점)
- anthemic final
- 봄/계절 직접 선언
- "사랑" / "이별" 직접 어휘
- key change

### Sample subtitle

## Archetype C2 — 조용한 결단 / Silent Resolve

### Family
deep

### Mood territory
침묵을 받아들이는 결의. 무언가를 말하지 않기로 한 결단, 또는 답이 안 와도 괜찮다는 자기 정직. **NOT despair** — livable, dignified silence.

### Reference tracks (calibration only)
- 권진아 (끝없는 밤) — felt-grand, dignified silence
- 김사월 (진주) — withdrawn alto, 결단의 결
- 박혜진 박하잎 (어제 너는) — felt grand restraint
- 정밀아 (그리운 그대) — breath-forward dignified
- 시와 (일곱 번째 노래) — paper-thin, 침묵 수용

### Hook signature
- 진입 방식: declarative (-기로 했어 / -지 않기로 / -둘게)
- 음절 범위: 10-13 음절
- 1인칭 위치: 결단 직후 / 닫은 직후 / 멈춰 선
- 예시 hook vibes:
  1. "이제 묻지 않기로 했어"
  2. "답이 안 와도 괜찮을게"
  3. "여기서 멈춰 두기로 했어"

### Chorus shape
- Lock 라인 자리: 1st line
- Lock 라인 음절 범위: 8-10 음절
- Alliteration motif 추천: ㅁ/ㄴ 가족 (멈/말/묻/내/너무)
- Lift mechanic: 청유형 → 단정 (-기로 했어 → -었어)

### Vocal posture
- 호흡 패턴: gentle determined (still warm, dignified)
- 화법: 반말 자기독백
- Dynamic range: restrained

### Tempo & Lane affinity
- BPM 범위: 60-72
- 1순위 lane: `ambient-piano`
- 2-3 순위 lane: `acoustic-ballad`, `90s-vintage-ballad`
- 금지 lane: `bossa-folk`, `folk-electronic`, `dream-pop`

### Standout image palette
- 문, 의자, 책상, 빈 컵, 봉투, 시계, 창문

### Bridge subversion model
결단의 정직한 확장: "묻지 않는다고 그리워하지 않는 건 아니야" 의 자기 정직 → "그래도 묻지 않을게" 의 결의 재확인. 결단이 슬픔을 부인하지 않으면서도 흔들리지 않음.

### Banned moves
- 절망 / 패배 톤 (livable resolve only)
- power-ballad swell
- 운명·기적 어휘
- 외부 대상 원망
- 영어 / 숫자
- 작품 신 직접 묘사
- "다시는" / "영원히" 류 거대 어휘
- 청각 sensory 직접 표현
- key change

### Sample subtitle
