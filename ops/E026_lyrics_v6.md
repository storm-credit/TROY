## Brief rationale (post-script)

- First-line hook: "약속한 시간보다 한참 일찍 도착해 버렸어" — 의도하지 않은 신체 행동이 마음을 누설. 보편 경험 + 사건성 즉시 발동.
- Chorus lock: "아직 말하지 않았는데 / 침묵이 먼저 달라졌어" verbatim 3회. 브리프 "-았다" → "-았어" 통일.
- Universal-imagery audit: "벤치 / 카페" 등 특정 공간 명명 회피. "자리 / 손목 / 손등 / 공기 / 시계" 보편 신체-환경. heartbeat 직접 단어 0건. proposal/forever 0건.
- Phase 2 register: pre-confession 긴장이지만 폭발 회피. final chorus 의 "그 짧은 말이 오래 남을 것 같았어" 가 anticipation 만 남기는 여운 처리.
