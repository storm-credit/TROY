## Rationale (post-script, not pasted to Suno)

### First-line hook design
"여름은 사람을 또렷하게 만드는 계절이라" — 진술형 명제로 시작해 한 줄 안에 lane(dream-pop), 계절(여름), 동사적 thesis(또렷하게 만드는)를 동시에 박는다. 워밍업 0행. 청자가 "어, 무슨 말을 하려는 거지" 가 아니라 "맞아, 그런 계절이지"로 즉시 동의 진입. 이어지는 두 번째 행에서 "어떤 윤곽은 빛 아래에서 처음 보였어"로 thesis를 시각화 → 설명 없이 hook 안착.

### Five standout lines (의도적으로 청취 시 귀에 박히는 라인)
1. "오늘에서야 너의 가장자리를 본 것 같았어" (verse1, line 4) — '가장자리' 단어로 '또렷한 윤곽' 시드를 비유적으로 비틀어 직접 묘사 회피.
2. "그림자가 둘이라 한 사람의 모양 같지 않아" (pre-chorus, line 3) — '둘이 만든 그림자' 시드를 부정문으로 처리 → 쌍/연결의 성립을 역으로 강조.
3. "또렷해서 무서워지는 순간이 있다는 걸 / 오늘에야 알았어" (chorus 5-6) — Phase 3 의 핵심 정서 (행복의 또렷함이 곧 상실 예감을 부른다)를 dream-pop register 로 가볍게 안착.
4. "또렷한 게 좋은데 또렷한 게 무서운 건 / 사라질 때를 미리 알아 버려서" (verse2, line 7-8) — verse 안에서 chorus thesis를 한 번 더 구체화. '사라질 때' 라는 종결 예감을 verse 단계에서 깔아 bridge 진입을 정당화.
5. "사라질 걸 알아도 한 번 더 보고 / 사라질 걸 알아서 한 번 더 부른다" (bridge, line 5-6) — 같은 어구 반복 + 조사 변경(알아도 → 알아서)으로 정서 회전. dream-pop 의 부유감 안에서 결단을 가볍게 내려놓는 효과.

### Bridge subversion
Verse-chorus는 "또렷함이 무섭다" 라는 양가성을 쌓는다. Bridge 는 그 무서움을 그대로 부정하지 않고 **방향을 비튼다** — "사라질 걸 알아서 더 보고 싶다" 로 reframe. 부정→긍정 클리셰 역전이 아니라, 같은 인식을 유지한 채 행동만 한 칸 옮긴다 (관찰 → 부름). 이 micro-pivot 이 Phase 3 의 적정 무게: 순진한 해소가 아니고 과장된 비극도 아닌, "그래서 한 번 더 부른다" 라는 행동적 안착.

### Final chorus 1-variant lift
Chorus 1·2 anchor: "여름 빛 아래서 너의 **윤곽이** 가장 **또렷했어**"
Final chorus lift: "여름 빛 아래서 너의 **윤곽만** **또렷이 남았어**"
- 조사 1개 변경 (이→만): 다른 모든 것의 흐려짐을 함의
- 동사 변형 (또렷했어→또렷이 남았어): 과거 사실 → 잔상으로 종결, dream-pop 의 페이드 정서에 부합
- 5-6행도 미세 변형: "오늘에야 알았어" 는 유지, 그 위 행의 "순간이 있다는 걸" → "하루가 있다는 걸" 로 시간 단위 확장 → 하루 종결의 안착감 부여
- 변형 1회 룰 안에서, anchor 의미는 보존하면서 최종 정서 1단계 lift 달성

### Universal-imagery audit
작품 신 직접 묘사 0건 확인:
- 카페/창가/같은 자리/두 컵 → 0건 (v6 의 cafe register 전부 제거)
- 캠퍼스/교실/캠퍼스 외 특정 장소 → 0건
- 능력 직접 호출 (들리는 결, 공감각, sensory origin) → 0건. "한 박자 늦게 도착하는" 은 청취 일반 비유로 안전.
- 운명/완전한 단둘 세계 → 0건. Bridge 는 명시적으로 유한성 인정 ("이 빛이 지금 여기 있어서 가능한 거였어").
- 클리셰 회피: 비/우산/별빛/꽃잎/네 향기 → 0건. '여름 빛/황금색 오후/그림자/윤곽/가장자리' 만 사용 — genre file 시드와 1:1 정합.
- borderline: "어깨에 잠깐 머물 때" — 신체 묘사지만 일반 시어, 구체 신 아님. OK.
- 화법: 반말 자기독백 일관, 청자 1인칭(나)→2인칭(너) 고정. 영어/숫자/외래어 0건.

### Korean pronunciation pass (dream-pop · BPM 72)
- 자음 군집/된소리 hook 0건. "또렷" 디귿+리을 반복은 의도된 motif (chorus·verse·bridge·final 전반에 분산), BPM 72 안정 호흡 안에서 또렷히 발음됨.
- 한 호흡 안 들어가는 길이 점검: 가장 긴 "다른 어떤 계절에서도 이만큼 보이진 않을 거야" — 16음절, 72BPM dream-pop 에선 2박 분할 가능. OK.
- "황금색 오후가 우리 어깨에 잠깐 머물 때" — '황금색' (ㅎ-ㄱ) 가벼운 마찰, dream-pop reverb 위에서 부드럽게 흐름.
- 잠재 위험: "또렷해서 무서워지는" 의 ㄸ-ㅁ-ㅈ 전이 — 한 박 안에 처리 가능, 클러스터 아님.
- 영어/숫자/외래어 0건. Jamo 이모티콘 0건. Stage direction 0건.

## Removed / pivoted from v6

- v6 thesis ("창가 안쪽은 따뜻했고, 바깥은 조금씩 흐려졌다" / cafe routine / 두 컵 / 같은 자리) → 전면 pivot. 사용자 지시 v8 thesis 가 genre assignment 의 "여름 빛 아래서 서로가 가장 또렷한 순간" 이므로 cafe register 전체 제거, 보편 summer-light register 로 재구축.
- v6 first-line ("한 자리에 두 번 앉으면 그 자리가 우리 자리가 돼") → "여름은 사람을 또렷하게 만드는 계절이라" 로 교체 (즉발 thesis hook).
- v6 chorus anchor → 신 chorus anchor "여름 빛 아래서 너의 윤곽이 가장 또렷했어" 로 교체. v6 의 "안쪽/바깥쪽" 대비 구조는 v8 에서 "또렷함/사라짐" 의 시간적 대비로 재배치.
- v6 의 명시적 "둘만의 세계" 폐쇄성 (좁아진 시야, 같은 페이지) → 제거. dream-pop 안에서 폐쇄성보다는 "지금 이 빛 안의 또렷함" 이라는 순간성으로 정서 톤 재조정.
- 라인 수: v6 약 38행 → v8 47행 (45-50 권고 안). Bridge 6행 유지(자기 의문→행동 micro-pivot). Outro 별행 없음 (final chorus 안 흡수).
