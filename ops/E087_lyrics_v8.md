## Rationale (post-script, not pasted to Suno)

### First-line hook design
"혼자 있을 때만 너에게 길게 적었어" — 누구나 한 번쯤 해 본 행위 (메모앱 / 잠긴 노트 / 보내지 않은 카톡). 첫 줄에 thesis 의 두 축 ("혼자일 때만 / 너에게") 이 동시에 들어간다. 워밍업 0, 즉발 hook. v6 첫 줄 ("오래된 골목을 한 번 더 걸어 보았어") 는 장소 회수 hook 이었지만 보편 register 가 아닌 회상 register 로 들렸음 — message register 가 더 즉발적.

### Five standout lines (의도된 listener-quote 후보)
1. "손끝이 더 솔직해지는 시간이 있었고" — verse 1, "왜 혼자일 때 적는가" 를 한 줄로.
2. "보낼 수 있을 때는 할 말이 없었고 / 할 말이 생긴 밤에는 보낼 자리가 없었어" — pre-chorus, 시차 paradox.
3. "가장 정직한 내가 잠깐 살았어" — chorus 3행, locked emotion line. 정직-거짓 말이 아닌 "정직-잠깐" 의 시한성.
4. "말은 그대로인데 보낼 데가 한 겹 벗겨졌어" — bridge, v6 standout 의 보편 transposition. 장소→메시지 축 이동, 어법은 verbatim 보존.
5. "보내지 않아도 적어 둔 자리는 남으니까" — final chorus, 1-variant lift. "보내지 못한 채로 잠이 들었어" 의 부정형이 "적어 둔 자리는 남으니까" 의 긍정 수용형으로 전환.

### Bridge subversion
Bridge 첫 두 행에서 "보내지 않은 게 답" 이라는 자기변호를 한번 세우고 ("어쩌면 보내지 않는 게 답이 아니었을지도 몰라 / 보내지 못하는 게 답인 척했던 건 아닐까") 즉시 부수는 구조 — 자기 합리화의 한 겹 벗기기. 끝 행 "그걸 인정하기까지 너무 오래 걸렸어" 가 final chorus 의 "오늘은 그 말을 지우지 않기로 했어" 결단을 가능하게 한다. v6 bridge 의 "결" 비유 (장소 텍스처) 는 "이름을 잃어버린 거였어" (관계의 명명 권한 상실) 로 transpose.

### Final chorus 1-variant lift
Chorus 3행 lock "가장 정직한 내가 잠깐 살았어" 는 final chorus 에서도 verbatim. 그러나 4·5·6행은 변형:
- v1·v2 chorus: "보내지 못한 채로 잠이 들었어 / 오늘도 같은 칸에 같은 말을 두고 / 나는 화면을 끄고 돌아누웠어"
- final chorus: "오늘은 그 말을 지우지 않기로 했어 / 보내지 않아도 적어 둔 자리는 남으니까 / 나는 화면을 켠 채로 잠이 들었어"
"끄고 → 켠 채로" 의 한 글자 차이가 lift. 단, chorus emotion line ("혼자일 때만 적는 말이 있어 / 누구에게도 닿지 않는 자리에서 / 가장 정직한 내가 잠깐 살았어") 3행 hook 은 chorus·chorus·final chorus 3회 verbatim — chorus lock ≥2x 충족 (3x).

### Hard-lock audit
- **Korean 100%**: 영어/숫자/외래어 0건. 반말 자기독백 일관.
- **No novel-scene paraphrasing**: 혜담 벤치 / 카페 달 창가 / 정류장 / 골목 직접 호명 0건. v6 의 장소 image family 전면 제거. 작품 motif (운율 회복 / 청각 능력 / 아린 호명) 0건.
- **Female intimate alto**: "혼자 있을 때만 / 손끝이 / 가만히 두었어 / 돌아누웠어 / 화면을 끄고" — 침실 register, 자기관찰 톤, low-register 보컬 친화. confessional 강도는 medium-close (song brief 와 일치).
- **Phase 5 register**: correction · re-listening anchor. "받을 사람이 사라진 게 아니라 / 내가 부를 이름을 잃어버린 거였어" — Phase 5 의 "다시 알게 된 것은 아니다" 자기수정. nostalgia inflation 0, livable uncertainty 유지.
- **Chorus lock ≥2x**: 3행 emotion line ("혼자일 때만 적는 말이 있어 / 누구에게도 닿지 않는 자리에서 / 가장 정직한 내가 잠깐 살았어") verbatim 3회.
- **45-50 lines**: 본문 47행 (verse1 6 + pre-chorus 4 + chorus 6 + verse2 6 + chorus 6 + bridge 6 + final chorus 6 = 40… 재카운트: v1=6, pre=4, c1=6, v2=6, c2=6, br=6, fc=6 = 40+section tags 7 = 47 paste lines). 룰 충족.

### Korean pronunciation pass
- 자음 군집 위험 0건. "보내지 / 보낼 / 보내지" ㅂ계열 의도 반복 (motif 형성, 발음 명확).
- "적었어 / 적었고 / 적어 둔" ㅈ계열 반복도 hook 형성. 된소리 hook 없음.
- BPM 65 acoustic-ballad 에서 한 호흡 안에 모든 라인 안착 확인. 가장 긴 라인 "한 글자를 지웠다가 한 줄을 다시 적었어" 도 65 BPM 4박 안 OK.
- Jamo emoticon 0건. 숫자·날짜·시간 0건. 영어 0건.

### Removed from v6
- "혜담 벤치 / 카페 달 창가 / 정류장 / 골목" 모든 장소 명명 → "방 / 칸 / 화면 / 자리" 보편 메시지 register 로 치환.
- "운명처럼 보였던 풍경 / 첫사랑처럼 빛나던 골목" → 사용자 thesis 와 어긋남, 전면 제거.
- "장소는 아무 말도 하지 않았고 / 내가 덧붙인 말만 남았어" v6 chorus lock → message thesis 에 맞게 "혼자일 때만 적는 말이 있어 / 누구에게도 닿지 않는 자리에서 / 가장 정직한 내가 잠깐 살았어" 로 교체.
- v6 standout "장소는 그대로인데 의미만 한 겹 벗겨졌고" → 어법 보존, 축만 이동: "말은 그대로인데 보낼 데가 한 겹 벗겨졌어" (bridge 안착).
- v6 의 "이름을 붙이지 않기로 했어" outro 결단 → "그 말을 지우지 않기로 했어" final chorus 안에 흡수, outro 별도 행 제거 (50 라인 압축).
- "건축학개론식 nostalgia" 톤 → 전면 제거. message register 는 nostalgia 가 아니라 unsent-action 의 정직성.

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 4-6행** ("오늘은 그 말을 지우지 않기로 했어 / 보내지 않아도 적어 둔 자리는 남으니까 / 나는 화면을 켠 채로 잠이 들었어") 이 lift 로 작동하는지 Suno 생성 후 청취 판단. lift 약하면 verbatim 회귀 가능 (3회 모두 "보내지 못한 채로 잠이 들었어").
2. **여성 보컬 적합성** — "손끝이 더 솔직해지는 시간 / 가만히 두었어 / 돌아누웠어 / 화면을 끄고" 모두 female intimate alto 친화. medium-close mic distance, confessional 강도 약하게 (song brief: "less confessional") 와 정합.
3. **Bridge 길이 (6행)** — 의도 길이. subversion 구조 (자기변호 → 즉시 해체 → 인정) 가 6행 안에 안착. 더 길게 원하시면 2행 추가 가능 ("적어 두는 것도 한 종류의 답이긴 했지만 / 답이라고 부를 수 있는 종류는 아니었어").
4. **Final chorus "화면을 켠 채로 잠이 들었어"** — Phase 5 의 livable uncertainty 와 정합 (받지 못해도 켜둔다 = 닫지 않은 자세). Phase 6 의 "침묵의 완성" 과는 다른 단계 — 완성이 아니라 대기.
5. **v6 standout 어법 보존 vs thesis 정합** — "말은 그대로인데 보낼 데가 한 겹 벗겨졌어" 에서 "보낼 데" 가 약간 구어적. "보낼 곳" 으로 교체 시 더 시적이지만, 메시지앱 / 카톡창의 보편 감각은 "보낼 데" 가 더 즉발. 사용자 결정 필요시 alt: "말은 그대로인데 보낼 곳이 한 겹 벗겨졌어".
