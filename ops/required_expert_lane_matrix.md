# Required Expert Lane Matrix

## 1. Purpose

이 문서는 `TROY`에서 작업 성격별로 반드시 먼저 고정해야 하는 전문가 레인 구성을 잠근다.

핵심 원칙:

- 총괄은 작업 시작 전에 이 문서부터 본다
- 필수 레인이 빠진 상태에서는 `pass`, `lock`, `official` 판정을 하지 않는다
- 글자수/분량 기준은 전문가가 아니라 local hook이 맡는다
- 총괄은 이 최소 세트를 기반으로 필요할 때 추가 전문가를 붙여 총괄한다

## 2. Core Experts

항상 기준점이 되는 핵심 전문가:

- longform narrative craft architect
- continuity / canon architect
- character psychologist
- prose & rhythm director

설명:

- longform narrative craft architect:
  - 엔진 전문가다
  - 장편 작법, 회차 설계, 장면 기능, setup-payoff, 감정 예산, 서사 압력 조절을 맡는다
  - 상황에 맞는 작문법을 고르고 제시한다
  - 예:
    - 지금 장면은 복선 심기가 맞는가
    - 여기서는 reveal delay가 맞는가
    - 이번 화는 감정 압박을 먼저 써야 하는가
    - 지금은 웹소설형 hook를 강하게 걸어야 하는가
    - 이 구간은 설명보다 증상/징후로 보여줘야 하는가
    - 여기선 정치 스릴러 톤인지, 생존 압박 톤인지
  - 한 줄로:
    - `무엇이 맞는 세계인가`가 아니라 `그걸 지금 어떻게 서사로 쓸 것인가`를 다루는 전문가
- continuity / canon architect:
  - 설정 전문가에 가깝다
  - 무엇이 맞는 세계인가를 다룬다
  - 설정 충돌, 관계 단계, 사건 인과, 금지 리스크를 확인한다
- character psychologist:
  - 서준/아린 감정선, 대화 윤리, intimacy ceiling, 과속 여부 점검
- prose & rhythm director:
  - 장면 수, 장면 기능, 문장 밀도, 반복 위험, 재작성 구조 설계

## 3. Task Bundles

### A. Episode Prose Rewrite

적용:

- 회차를 처음부터 다시 쓸 때
- 단순 교열이 아니라 장면 구조를 다시 짤 때

필수 전문가:

- longform narrative craft architect
- continuity / canon architect
- character psychologist
- prose & rhythm director

### B. Event Cluster Rewrite

적용:

- 체육대회, 축제, 공연 주간, 여행처럼 여러 화가 같은 event arc를 공유할 때

필수 전문가:

- longform narrative craft architect
- continuity / canon architect
- character psychologist
- prose & rhythm director
- cluster planner

### C. Length-Gated Rewrite

적용:

- `공백 제외 한글 수` 같은 분량 하한선이 hard gate일 때

필수 전문가:

- longform narrative craft architect
- continuity / canon architect
- character psychologist
- prose & rhythm director

추가 규칙:

- 글자수 측정은 local hook
- `3500`은 hard floor
- `4000`은 safe line
- `3500` 미만은 무조건 `draft`
- `3500-3999`는 hard-gate pass일 수 있어도 safe-line under로 watchlist에 남긴다
- `4000+`이면서 구조/감정선이 정상일 때만 safe-line candidate로 본다
- 분량 채우기 흔적이 보이면 수치와 무관하게 fail
- 전문가는 `숫자 맞추기`가 아니라 `재작성 설계`만 담당

### D. Canon / Structure Change

적용:

- 구조 변경
- 복선 재배치
- phase/movement 재설계

필수 전문가:

- longform narrative craft architect
- continuity / canon architect
- character psychologist
- episode planner

추가 전문가:

- narrative motif specialist

### E. Music / Song Prompt Work

적용:

- 회차별 곡 방향
- Suno brief
- leitmotif / reprise 설계

필수 전문가:

- lyrics director
- music prompt engineer

추가 전문가:

- character psychologist
- narrative motif specialist

### F. MV / Image / Visual Adaptation

적용:

- visual cut list
- storyboard
- key visual continuity

필수 전문가:

- mv storyboard director
- continuity / canon architect

추가 전문가:

- prose & rhythm director
- cluster planner

## 4. Hard Rule

아래는 항상 고정이다.

- 재작성 패스는 먼저 `required expert lane matrix`를 확인한다
- 필수 전문가가 빠졌다면 작업 상태는 `draft`
- local length gate가 있는 작업은 수치 통과 전 `pass` 금지
- 총괄은 전문가 레인을 잊지 않는 책임이 있다
- 총괄은 이 문서를 `최소 필수 구성표`로 쓰고, 작업 리스크가 올라가면 추가 전문가를 배치한다
- 다음 range를 열기 전에는 직전 range handoff check를 먼저 잠근다
