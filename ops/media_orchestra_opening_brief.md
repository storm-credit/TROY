# Media Orchestra Opening Brief

## 1. Purpose

이 문서는 `TROY`가 novel-side lock 이후 media 오케스트라로 넘어갈 때의 첫 진입 순서를 잠근다.

목표:

- 소설을 upstream truth source로 고정한 채 media 작업을 연다
- music / image / MV를 한꺼번에 섞지 않고 lane 순서로 연다
- media 보고를 novel 보고와 분리한다

## 2. Entry Condition

전제:

- `ops/novel_post_lock_expert_orchestra_audit.md`: pass
- `ops/manuscript_migration_checklist.md`: post-lock expert audit complete
- `manuscript/` truth source locked
- `export/` delivery copies available

## 3. Director Decision

media 오케스트라 오픈 판정:

- approved

이유:

- 소설 쪽 blocking issue 없음
- 남은 novel 작업은 packaging 성격이지 story repair가 아님
- 따라서 media는 downstream lane으로 시작 가능

## 4. Lane Order

권장 시작 순서:

1. `music`
2. `image`
3. `MV`

설명:

- music이 감정 곡선과 refrain 구조를 가장 먼저 고정하기 좋다
- image는 캐릭터/공간/정조를 시각 기준으로 정리한다
- MV는 music과 image truth가 잠긴 뒤 컷 흐름으로 여는 것이 가장 안정적이다

## 5. First Media Questions

- 120화 전체 song allocation을 그대로 production-ready로 밀 것인지
- 대표 파일럿 구간만 먼저 만들 것인지
- music first를 `E001`, `E040`, `E078`, `E110`, `E120` 앵커 중심으로 열 것인지

## 6. Lock Note

- media report is separated from novel report
- novel changes do not reopen automatically during media production
- if media reveals canon conflict, it escalates back to director review
