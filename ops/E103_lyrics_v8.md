## Rationale (post-script, not pasted to Suno)

### First-line hook design
"오랜만이라는 말이 둘 사이에 짧게 놓였어" — 즉발 정서. v6 의 "오랜만이라는 말이 짧게 겹쳤어" 가 좋은 hook 이었지만 "둘 사이에 / 놓였어" 로 객체화하여 더 시각적인 첫 호흡 구성. 워밍업 0행, 첫 줄에서 이미 두 사람이 마주 앉은 정황이 그려짐. 보편 인지 (오랜만의 첫 마디) 만으로 작동, 카페·메시지 직접 호명 없음.

### Five standout lines
1. "오랜만이라는 말이 둘 사이에 짧게 놓였어" (verse 1 첫 줄, hook)
2. "변하지 않은 자리를 먼저 발견했어" (verse 1, 보편 재회 인지)
3. "같은 한 줄을 두 번 읽어 가는 자리에서" (chorus 3행, v6 keeper)
4. "이 오후가 길어도 좋았고 / 짧아도 괜찮았어" (chorus tail, 보편 안정)
5. "어쩌면 가장 오래 가는 사이는 / 서로를 다 설명하지 않는 사이였는지도 몰라" (bridge subversion 라인)

### Chorus emotion line placement
Locked phrase "오래된 친구처럼 마주 앉아 있었어" 는 chorus 1행에 verbatim 3회 (chorus 1·2·final). final chorus 의 마지막 lift 라인은 자연 변형 "오래된 친구처럼 그냥 앉아 있었어" — "마주" → "그냥" 한 단어 교체로 결말의 무목적성 (오후의 잔잔함, 어디로도 데려가지 않는) 표현. 변형 1회만 허용된 룰 안에서 사용.

### Bridge subversion
v6 의 bridge 는 "다시 만났다는 말이 아니야 / 다시 묻기 시작했다는 말이야" 로 '대화 시작' 을 명시했으나, v8 thesis 는 phase 6 의 universal afternoon-meeting (대화 시작이 아니라 이미 자리잡은 거리). 따라서 "다시 가까워졌다 → 거리가 어색하지 않다" 로 비틀고, 마지막 2행 "가장 오래 가는 사이는 / 서로를 다 설명하지 않는 사이였는지도" 로 '말의 양' 이 아닌 '말의 부재' 로 재정의 — 일반 청자에게 보편 격언처럼 박힐 라인.

### Universal-imagery audit
- 카페 달 (Cafe Dal) 직접 호명 0건. "카페" 단어 자체도 빠짐. "오후 빛", "테이블 끝", "컵" 의 보편 객체로 처리.
- 짧은 메시지 화면 직접 묘사 0건. "오랜만이라는 말" 의 발화로만 처리.
- back-together / perfect timing / destiny restart / hearing-truth 0건.
- 클리셰 (운명 / 다시 시작 / 새로운 우리 / 추억의 향기) 0건.
- 계절 직접 선언 (봄/가을) 없음. "오후 빛" 만 시간 신호로 사용.
- 공감각·청취 능력·sensory origin 직접 호출 0건. "서로의 침묵을 가만히 들어 보았어" 는 보편 청취 (특수 능력 아님).
- 인-월드 캐릭터 호명 (서준/아린) 0건. POV 1인칭 자기 화법, 상대는 "서로 / 우리" 만.
- borderline: "같은 음악을 두 번 들으며" — 음악 motif 가 등장하지만 BGM 일반 묘사. 작품 motif "아린의 노래" 호출 아님.

### Korean pronunciation pass
- 자음 군집 위험 라인 0건. 90s-vintage-ballad BPM 71 에 모든 라인 한 호흡 안.
- "마주 / 그냥 / 자리" 의도적 모음 명료. "닿 / 닫" 계열 디귿 군집 회피 (E001 v8 와 의도적으로 다른 phonetic palette).
- "오래된 친구처럼" — ㄹ-ㅇ-ㄴ 부드러운 흐름, 90s ballad 호흡에 안정.
- "두 번 읽어 가는" — v6 keeper, 이미 검증된 호흡.
- 영어/숫자/외래어 0건. 화법 일관 (반말 회상조).
- 자모 emoticon (ㅎㅎ/ㅋㅋ) 0건.

### Removed from v6
- "더 말해야 닿을 얼굴이 됐어" → v6 chorus lock. 이 문장은 '대화 재시작' 의 능동성 강조 (E102~103 전환점) 인데, v8 thesis 는 phase 6 의 안정된 거리 (대화의 양이 아니라 함께 앉아 있는 자체) 로 이동. "오래된 친구처럼 마주 앉아 있었어" 로 교체.
- "짐작으로 통하던 시절이 끝났어 / 짐작이 끝난 자리에 말이 들어오기 시작했어" → 시제 전환 격렬 (전후 대비). v8 은 균질한 오후 톤 유지를 위해 "가까이 가는 일이 더는 급하지 않았어 / 멀리 있다는 말도 더는 무겁지 않았어" 로 단조 평형 화법 채택.
- "한 박자 늦은 사람이 됐고 / 늦은 박자에 같이 머물러 있었어" (v6 final chorus) → '늦음' 서사가 v6 의 핵심이었으나 v8 thesis 는 시간성 자체가 풀린 자리. "이 오후가 우리를 어디로도 데려가지 않아서" 로 교체 — 시간 진행 없이 '머무는 자리' 자체가 lift.
- v6 verse 2 "오래된 친구처럼 보이지만 친구는 아니어서 / 오래된 연인처럼 보이지만 연인도 아니어서" → 관계 정의 부정 구조. v8 은 그 부정을 chorus 1행 "오래된 친구처럼 마주 앉아" 의 직유 ("처럼") 로 흡수, verse 2 를 '컵의 온기 / 지난 이야기' 의 시각 구체로 풀어 보편화.

## Unanswered for Conductor

1. **변형 라인 1회 허용 사용 — final chorus 마지막 "오래된 친구처럼 그냥 앉아 있었어"** 가 음악적 lift 로 작동하는지는 Suno 생성 후 청취 판단. lift 가 약하면 verbatim 3회 ("오래된 친구처럼 마주 앉아 있었어") 로 회귀 가능. 90s ballad 의 final chorus 는 일반적으로 한 단어 교체만으로도 정서 전환이 잘 작동.
2. **여성 보컬 적합성** — "컵의 온기 / 오후 빛 / 같은 음악" 의 자기관찰 톤은 female intimate alto 에 자연 적합. POV 가 두 사람 모두에게 열려 있어 (1인칭이지만 상대 호명 없음) 어떤 보컬 성별이어도 작동, female lock 유지 권장.
3. **Bridge 길이 (6행)** — 8-12행 권고 하한이지만, phase 6 의 잔잔한 톤 보존 위해 의도적으로 짧게. 더 길게 원하시면 "가장 오래 가는 사이는 / 서로를 다 설명하지 않는 사이였는지도 몰라" 뒤에 2행 추가 가능 ("그래서 오늘은 / 말 대신 같은 빛을 보고 있었어").
4. **chorus tail 처리** — "이 오후가 길어도 좋았고 / 짧아도 괜찮았어" 의 짧은 2행이 chorus closure 로 음악적 무게가 충분한지 청취 판단. 약하면 chorus 5·6행을 "이 오후가 길어도 좋았고 / 짧아도 괜찮은 자리에 앉아 있었어" 로 단행 통합 가능.
5. **'오후' 시간대 lock** — v8 의 universal register 가 '오후' (낮 빛) 에 묶여 있어 v6 의 '저녁 카페 달' 보다 밝음. director lock note (낮은 조명 유지) 와 살짝 어긋날 수 있어, MV 시각화 시 조명 톤 (warm-low) 으로 보정하면 가사-비주얼 정합 OK.
