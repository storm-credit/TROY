# Character Canon

## 1. Core Dynamic

이 작품의 핵심 커플은 "수신하는 남자"와 "발신하는 여자"다.

- 서준은 사랑을 해석하려 한다
- 아린은 사랑을 표현하려 한다
- 둘은 좋아함의 크기보다 소통 방식이 다르기 때문에 부딪힌다

## 2. Yoon Seojun

- age: 22
- department: Sungkyunkwan University, business
- image keywords: gray, quiet, receiver, earphones, distance
- visible trait: 항상 이어폰을 끼고 있음

### surface

- 무뚝뚝함
- 내성적임
- 타인과 일정 거리를 유지함

### core

- 관찰력이 예민함
- 상처받기 쉬워서 먼저 차단함
- 감정이 없는 게 아니라 감정 교류가 두려움

### emotional arc

회피 -> 시도 -> 실패 -> 성찰 -> 성숙

### narrative rule

서준은 "많이 느끼는 사람"이 아니라 "많이 듣고도 제대로 묻지 못하는 사람"으로 써야 한다.

## 3. Ji Arin

- age: 21
- department: Sungkyunkwan University, Korean literature
- image keywords: spring, clear, piano, sender, warmth
- visible trait: 밴드 동아리 보컬

### surface

- 밝고 사교적임
- 감정 표현이 솔직함
- 호기심이 많음

### core

- 복잡한 내면을 가짐
- 상처에 민감함
- 표현하지 않으면 더 크게 다침

### musical signature

- 기본 운율: 말랑한 피아노 도-미-솔
- 기쁨: 경쾌한 스타카토
- 사랑: 따뜻한 어쿠스틱과 피아노
- 혼란: 복잡한 불협화음
- 닫힘: 완전한 무음

### emotional arc

호기심 -> 끌림 -> 사랑 -> 의심 -> 정리

### narrative rule

아린은 단순한 치유형 여주가 아니다.  
그녀는 서준의 해석이 틀릴 수 있다는 현실을 들고 오는 인물이어야 한다.

## 4. Key Support

| Name | Function |
|---|---|
| 최이든 | 룸메이트, 현실적 완충재, 서준의 유일한 친구 |
| 강도현 | 연적, 다른 방식의 표현성, 관계 압박 장치 |
| 송유빈 | 서준의 초기 상처와 세계관 기원 |
| 배소나 | 아린의 현실 검증 장치이자 감정 통역자 |
| 김은지 | 후반부 캐논 분기 가능성의 핵심 인물 |

## 5. Character Design Rule

모든 주요 인물은 음악적 인상어를 가져야 한다.

예시:
- 서준: 회색 노이즈, 억눌린 저음, 잔향
- 아린: 맑은 피아노, 봄빛, 정직한 호흡
- 이든: 재즈 드럼, 리듬감, 농담
- 도현: 록 밴드, 직진성, 압박감

## 6. Character Canon Warning

후반부 관계선은 아직 최종 잠금 전이다.

특히 다음은 반드시 확정이 필요하다.
- 아린과의 최종 결말
- 김은지의 최종 위치
- 서준의 운율 상실 이후 연애 윤리
