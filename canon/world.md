# World Canon

## 1. The Law of Rhyme

모든 감정은 고유한 진동수와 패턴을 가진다.  
윤서준만이 그것을 음악으로 듣는 수신기다.

이 능력의 성격:
- 초능력이라기보다 후천적 공감각
- 축복이 아니라 소음 과부하에 가까운 저주
- 감정을 언어가 아닌 운율로 받아들이는 방식

## 2. Absolute Rules

### self is inaudible

- 서준은 자기 자신의 운율을 영원히 들을 수 없다.

### live only

- 전화, 녹음, 영상으로는 운율이 전달되지 않는다.
- 실제 공간과 실제 존재가 필요하다.

### range

- 기본 감지 범위는 약 3~5미터다.
- 관계의 거리와 감지 거리의 일치가 상징적으로 중요하다.

### resonance conditions

| Trigger | Meaning | Clarity |
|---|---|---|
| contact | 대화, 스킨십, 감정 교류 | high |
| observation | 의도적 응시와 집중 | low |
| emotional burst | 울음, 분노, 환희 | auto |
| synchronization | 감정 공명 | auto |

## 3. Deep Sea Metaphor

서준의 기본 세계는 심해다.

- deep sea: 타인의 감정으로부터 스스로를 차단한 상태
- earphones: 심해로 잠수하기 위한 생존 장비
- surfacing: 운율을 듣기 위해 다시 세상으로 올라오는 행위

이 메타포는 작품 전체의 정서 언어로 유지한다.

## 4. Origin Trauma

- 16세 첫사랑 송유빈의 "사랑해" 뒤에서 지루함의 불협화음을 들음
- 말과 마음의 괴리를 처음 체험함
- 이후 서준은 관계를 피하고, 감정 차단을 생존 방식으로 삼음

## 5. Arin Exception

지아린은 현재 캐논상 "유일하게 계속 듣고 싶은 선율"이다.

현재 잠정 정의:
- 선명하고 정직한 감정 표현
- 음악적 재능과 보컬 감수성
- 거짓과 자기기만보다 솔직함이 강한 인물

주의:
- 왜 아린이 예외인지에 대한 최종 원인은 아직 잠금 전이다
- 심리적 예외인지, 체질적 예외인지, 상호 공명인지는 `open_questions.md`에서 확정한다

## 6. Fading Arc

운율은 영원한 능력이 아니다.

| Stage | Description |
|---|---|
| 1 | 감정적 차단 |
| 2 | 아린의 소리부터 선택적으로 소실 |
| 3 | 주변인까지 희미해짐 |
| 4 | 기억 속 소리만 남는 환청화 |
| 5 | 완전한 침묵 |

## 7. Key Spaces

| Space | Narrative function |
|---|---|
| 성균관대 | 일상과 관계의 기본 무대 |
| 카페 달 | 침묵과 대화가 교차하는 감정 거점 |
| 밴드 연습실 | 아린의 발신성이 가장 선명해지는 공간 |
| 반포 한강공원 | 설렘, 고백, 회상, 재회의 장소 |
| 서준의 자취방 | 차단과 고립의 실내 세계 |

## 8. World Rule For Expansion

세계관 규칙은 로맨스를 꾸미는 장식이 아니다.

- 규칙은 갈등을 만들어야 한다
- 규칙은 대사보다 감정 오해를 증폭해야 한다
- 규칙은 음악과 영상으로도 반복 번역 가능해야 한다
