# Arin Album Vol.1 First Execution Wave Sheet

## use rule

- 이 문서는 첫 실행 웨이브 5곡의 preflight와 attempt tracking을 한 장에 묶는다
- 실제 결과가 나오기 전에는 `verdict`를 비워 둔다
- 결과가 나오면 `master_session_log`에도 같은 내용을 옮긴다

## fixed run order

1. `E054 / 밝은 창`
2. `E113 / 나를 잃지 않는 일`
3. `E050 / 접힌 문장`
4. `E011 / 남은 소리`
5. `E118 / 같은 말을 하던 저녁`

## preflight checklist

- 금지 패턴:
  - `운명 / 능력 회복 / 재결합 / 해피엔드`
- 공통 체크:
  - 아린 authored voice가 전면인가
  - oversized climax가 없는가
  - 실제 유명곡 모사 감이 없는가
  - 트랙 역할이 다른 곡과 겹치지 않는가
