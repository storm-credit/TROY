# Arin Album Vol.1 Pilot Visual/MV Copy Cards

## purpose

- 이 폴더는 파일럿 3곡의 still image와 short MV test를 위한 copy card 묶음이다
- music attempt가 `pass` 또는 `strong revise`를 받은 뒤에만 사용한다

## order

1. `01_E054_VISUAL_MV_COPY_CARD.md`
2. `02_E113_VISUAL_MV_COPY_CARD.md`
3. `03_E050_VISUAL_MV_COPY_CARD.md`

## operating rule

- 먼저 priority still만 생성한다
- priority still에서 얼굴, 빛, 장소, 소품 연속성이 맞으면 전체 cut set으로 확장한다
- MV는 still-image-first 방식으로 30-45초 테스트만 만든다
- 실제 asset이 생기면 파일 경로와 verdict를 master session log에 남긴다

## global negative lock

- no real campus identifiers
- no logos
- no readable private text
- no idol-stage gloss
- no melodramatic closure
