# SORA_STORYBOARD_VIDEO_MODULE

## owner

- parent:
  - 뮤비 감동 지휘
- specialist:
  - Sora 전문가

## job

- storyboard 기반 shot 또는 short scene을 만든다
- recut/remix/shot-level iteration에 적합한 영상 후보를 만든다

## input slots

- storyboard beat
- first frame or reference still if available
- subject continuity instruction
- camera and framing
- action timing
- emotional objective
- negative guardrails

## output

- storyboard clip
- shot variation
- recut/remix candidate

## best use

- multi-shot emotional test
- scene bridge
- lyrical storyboard block
- alternate cinematic interpretation

## pass gate

- shot objective clear
- face/character continuity acceptable
- clip can be cut into 30-45 sec MV
- no story drift

## reject if

- Sora invents new plot beats
- character identity changes
- scene becomes too cinematic for the song
- emotional tone contradicts accepted direction note
