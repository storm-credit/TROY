# VEO_CINEMATIC_VIDEO_MODULE

## owner

- parent:
  - 뮤비 감동 지휘
- specialist:
  - Veo 전문가

## job

- cinematic short video 후보를 만든다
- accepted still 또는 reference image를 활용해 분위기와 카메라를 테스트한다

## input slots

- text prompt or reference image
- subject / scene description
- camera direction
- motion direction
- lighting and atmosphere
- duration target
- audio/environment note if supported
- negative guardrails

## output

- cinematic clip candidate
- atmosphere / camera test

## best use

- lyrical establishing shot
- atmospheric room/campus motion
- cinematic transition candidate
- emotional environment shot

## pass gate

- cinematic quality serves the track
- face lock remains acceptable if character appears
- no overproduced trailer feel
- no canon/emotional drift

## reject if

- character identity becomes unstable
- camera turns the MV into trailer spectacle
- romance/finale coding appears
- environment invents wrong setting
