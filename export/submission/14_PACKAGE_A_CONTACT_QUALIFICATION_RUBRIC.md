# TROY Package A Contact Qualification Rubric

## 1. purpose

이 문서는 `package A first wave`의 실제 대상자를 `A-primary-01`, `B-primary-01`, `D-primary-01` 슬롯에 넣기 전에 빠르게 거르는 기준표다.

원칙:

- 수를 채우기보다 응답 질을 본다
- 같은 독자 성향만 반복해서 넣지 않는다
- first wave는 `가볍게 봐주고 짧게라도 답을 줄 가능성`이 높은 사람을 우선한다

## 2. hard pass conditions

- 최근 3개월 안에 장문 읽기나 피드백 경험이 있었다
- 링크나 문서를 받아 보는 데 부담이 크지 않다
- 답이 늦어도 최소한 `봤다 / 어렵다 / 나중에 보겠다` 정도의 회신 가능성이 있다
- 해당 reader type 역할을 실제로 수행할 수 있다

## 3. hard fail conditions

- 장문 읽기를 거의 하지 않는다
- 부탁을 자주 잊거나 회신이 거의 없다
- 작품 톤보다 자극성만 요구하는 경향이 강하다
- reader type 구분 없이 모든 작품에 같은 반응만 주는 편이다

## 4. quick scoring

각 항목을 `0 / 1 / 2`로 본다.

- response reliability:
  - 답을 주는 편인가
- reading stamina:
  - 샘플 묶음을 실제로 볼 수 있는가
- fit to reader type:
  - type A / B / D 역할이 분명한가
- tone compatibility:
  - 이 작품의 낮은 감정선과 윤리 축을 읽어 줄 수 있는가

판정:

- `7-8`:
  - primary candidate
- `5-6`:
  - secondary candidate
- `0-4`:
  - first wave 제외

## 5. slot mapping guide

- `A-primary-01`:
  - 문장 결, 잔향, ending temperature를 읽어 줄 사람
- `B-primary-01`:
  - 초반 진입감, 다음 화 끌림, 읽기 속도를 말해 줄 사람
- `D-primary-01`:
  - 일반 독자 기준 첫인상과 추천 가능성을 말해 줄 사람

## 6. orchestra note

- first wave에서 가장 중요한 것은 `좋은 평`이 아니라 `비교 가능한 첫 반응`이다
- 관계상 미안해서 좋은 말만 할 사람보다는, 짧아도 실제 반응을 줄 사람을 우선한다
