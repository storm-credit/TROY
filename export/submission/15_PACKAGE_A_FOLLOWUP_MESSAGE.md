# TROY Package A Follow-Up Message

## 1. purpose

이 문서는 `package A first wave` 발송 후 `day 7` 전후에 한 번만 보낼 가벼운 follow-up 문구를 정리한다.

원칙:

- 재촉하지 않는다
- 안 봐도 괜찮다는 톤을 유지한다
- 짧은 반응만 받아도 된다는 걸 다시 분명히 한다

## 2. default follow-up

안녕하세요. 지난번에 보내드린 샘플 묶음은 편하실 때 보셔도 괜찮습니다.

혹시 이미 조금이라도 보셨다면 아주 짧게라도

- 첫인상이 어땠는지
- 계속 읽고 싶은 느낌이 드는지
- 남는 장면이 하나라도 있었는지

정도만 알려주셔도 큰 도움이 됩니다.

부담되시면 이번에는 그냥 넘기셔도 괜찮습니다.

## 3. shorter version

안녕하세요. 지난번 자료는 편하실 때 봐주셔도 괜찮습니다.

혹시 조금이라도 보셨다면 첫인상 한 줄만 알려주셔도 큰 도움이 됩니다.

## 4. use rule

- follow-up은 한 번만 보낸다
- 답이 없으면 `no response` 또는 `closed`로 정리한다
- follow-up 이후에도 장문 피드백을 요구하지 않는다
