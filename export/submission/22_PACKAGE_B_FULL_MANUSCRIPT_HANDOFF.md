# TROY Package B Full Manuscript Handoff

## 1. purpose

이 문서는 `package A` 이후 `package B`로 넘어갈 때 무엇을 보내고 무엇을 요청할지 한 장으로 정리한 handoff 문서다.

## 2. use when

- `package A` 반응이 긍정적이다
- 상대가 전체 원고를 읽을 의사가 있다
- 장편 구조와 결말 설득력을 보고 싶다

## 3. send bundle

- `00_START_HERE.md`
- `01_EVALUATOR_NOTE.md`
- `showcase_index.md`
- `너라는운율_submission_draft_v1.md`

## 4. what to ask

- 초반 진입감
- 중반부 반복/이탈 위험
- 후반 결말 설득력
- 완독 후 남는 감정

## 5. message stance

- 자유 감상을 먼저 받는다
- 질문은 2-4개로 좁힌다
- 피드백 양식은 강제하지 않는다

## 6. must-avoid

- package A 반응이 충분하지 않은데 바로 전체 확장
- full manuscript와 alpha feedback set을 동시에 강하게 요구
- 수정 포인트를 먼저 유도하는 질문 과다

## 7. next step

- 회신이 오면:
  - `24_PACKAGE_B_FULL_READ_LOG.md`에 핵심을 남긴다
- 상세 수정 의사가 분명하면:
  - `package C` 후보로 분기한다
