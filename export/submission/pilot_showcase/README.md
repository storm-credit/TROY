# TROY Pilot Showcase Pack

이 폴더는 `TROY`의 대표 package pilot 3종을 외부 검토나 쇼케이스용으로 빠르게 보여 주기 위한 submission-facing copy를 담는다.

구성:

1. `01_TROY_PILOT_ENDING_E118_packet.md`
2. `02_TROY_PILOT_OPENING_STAGE_E011_packet.md`
3. `03_TROY_PILOT_PEAK_LIGHT_E054_packet.md`

권장 읽기 순서:

- `E118`
  - 작품의 윤리와 ending grammar를 가장 선명하게 보여 준다
- `E011`
  - 초반 인상과 human-scale performance rule을 보여 준다
- `E054`
  - 관계 정점의 밝기와 그림자 균형을 보여 준다

주의:

- 이 폴더는 showcase packet이며 full manuscript submission을 대체하지 않는다
- full manuscript는 `export/submission/너라는운율_submission_draft_v1.md`를 기준으로 본다
- 여기의 packet들은 story-song-MV bundle reference를 보여 주기 위한 대표 샘플이다
