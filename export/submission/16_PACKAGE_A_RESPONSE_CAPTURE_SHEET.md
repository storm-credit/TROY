# TROY Package A Response Capture Sheet

## 1. how to use

이 문서는 `package A first wave`에 대한 실제 회신을 짧고 비교 가능하게 기록하는 내부용 시트다.

사용 순서:

1. `Tracker Slot`을 적는다
2. 실제 받은 반응을 3-5줄 안으로 요약한다
3. 아래 capture tags를 체크한다
4. `next move suggestion`을 적는다

## 2. capture tags

- `strong continue intent`
- `tone immediately clear`
- `memorable scene present`
- `ending curiosity`
- `slow opening concern`
- `middle drag concern`
- `tone mismatch`
- `needs full manuscript`
- `willing to give detailed feedback`
- `no clear signal`

## 4. shorthand interpretation

- `strong continue intent` + `needs full manuscript`:
  - `package B` 후보
- `willing to give detailed feedback`:
  - `package C` 후보
- `slow opening concern` or `middle drag concern`:
  - 반복 발견 여부를 watch
- `tone mismatch`:
  - 작품 적합 독자 타입인지 재검토

## 5. orchestra note

- 여기서는 긴 감상문을 예쁘게 옮기는 것보다, 서로 다른 reader type의 반응을 같은 형식으로 비교 가능하게 남기는 것이 더 중요하다
