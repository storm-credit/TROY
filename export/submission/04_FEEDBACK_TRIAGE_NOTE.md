# TROY Feedback Triage Note

피드백은 아래처럼 보면 가장 도움이 됩니다.

## 1. 바로 고칠 수 있는 것

- 안내문
- 읽기 순서
- 파일 진입 동선
- 외부용 표현

## 2. 바로 고치지 않는 것이 좋은 것

- 한 명만 느낀 취향 차이
- 작품의 기본 호흡 자체를 뒤집는 요구
- 결말 온도를 더 뜨겁게 바꾸자는 요구
- 이미 잠근 관계 윤리나 질문 문법을 약화시키는 요구

## 3. 특히 가치 있는 피드백

- 여러 사람이 비슷하게 말한 지점
- 특정 회차나 특정 장면을 정확히 짚은 의견
- `왜 그렇게 읽혔는지`가 설명된 의견
- `좋았다/아쉬웠다`가 아니라 실제 읽기 경험을 적어 준 의견
