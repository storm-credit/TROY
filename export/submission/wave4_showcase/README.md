# TROY Wave4 Showcase Pack

이 폴더는 `wave4 packet family` 5종을 submission-facing showcase copy로 묶는다.
읽는 사람이 `이 작품이 사람을 어떻게 잘못 읽고, 다시 어떻게 고쳐 읽는가`를 빠르게 파악하도록 배열했다.

구성:

1. `01_TROY_WAVE4_SELF_STORY_ERROR_E097_packet.md`
2. `02_TROY_WAVE4_CONTACT_DANGER_E018_packet.md`
3. `03_TROY_WAVE4_READING_DISCOMFORT_E047_packet.md`
4. `04_TROY_WAVE4_QUESTION_SPACE_E028_packet.md`
5. `05_TROY_WAVE4_PRELOSS_INSTABILITY_E064_packet.md`

권장 읽기 구조:

- core three:
  - `E097`
  - `E018`
  - `E047`
- support two:
  - `E028`
  - `E064`

읽기 포인트:

- `E097`
  - 자기서사 수정의 윤리
- `E018`
  - 몸-접촉이 위험 신호로 먼저 읽히는 방식
- `E047`
  - 말보다 해석이 먼저 도착할 때의 불편
