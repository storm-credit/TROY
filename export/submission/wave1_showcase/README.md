# TROY Wave1 Showcase Pack

읽는 사람이 작품의 가장 넓은 감정 진폭과 대표 turning point를 빠르게 파악하도록 배열했다.

구성:

1. `01_TROY_WAVE1_THRESHOLD_E040_packet.md`
2. `02_TROY_WAVE1_RUPTURE_E078_packet.md`
3. `03_TROY_WAVE1_ENDING_E120_packet.md`
4. `04_TROY_WAVE1_CONFESSION_E029_packet.md`
5. `05_TROY_WAVE1_RECONNECTION_E110_packet.md`
6. `06_TROY_WAVE1_OPENING_E001_packet.md`

권장 읽기 구조:

- core three:
  - `E040`
  - `E078`
  - `E120`
- support three:
  - `E029`
  - `E110`
  - `E001`

의미:

- core three는 threshold / rupture / ending dignity를 보여 준다
- support three는 confession / reconnection / opening identity를 보완한다
