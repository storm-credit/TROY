# TROY Alpha Reader Target Matrix

## 1. purpose

이 문서는 10점 업그레이드를 위한 외부 검증 대상을 정리한다.

핵심:

- 누구에게 읽힐지
- 무엇을 물을지
- 어떤 피드백은 반영하지 않을지

를 미리 잠근다.

## 2. target reader types

### type A

이름:

- 문학형 로맨스 정독 독자

보고 싶은 것:

- 문장 결
- 낮은 종지의 설득력
- 정서 잔향

꼭 물을 질문:

- 가장 오래 남는 장면 3개는 무엇인가요
- 결말은 약했나요, 아니면 낮고 선명했나요
- 문장이나 이미지가 과반복으로 느껴진 구간이 있었나요

### type B

이름:

- 웹소설 플랫폼 습관 독자

보고 싶은 것:

- 초반 진입감
- 회차 말 끌림
- 중반 이탈 위험

꼭 물을 질문:

- 1-3화에서 계속 읽게 만든 핵심은 무엇이었나요
- 가장 늘어진 구간은 어디였고 왜 그렇게 읽혔나요
- 회차 말미에서 다음 화를 누르고 싶어진 순간이 언제였나요

### type C

이름:

- 관계 윤리 / 심리 민감 독자

권장 레인:

- 기본은 `package C`
- first wave 기본 lane은 아님

보고 싶은 것:

- 서준의 오독
- 아린의 경계
- 결별과 재회의 윤리

꼭 물을 질문:

- 서준의 변화가 말이 아니라 태도로 읽혔나요
- 아린의 경계와 선택은 설득력 있었나요
- 결별과 재회가 건강하게 읽혔나요

### type D

이름:

- 완독형 일반 독자

보고 싶은 것:

- 끝까지 읽히는지
- 감정 보상감이 있는지

꼭 물을 질문:

- 끝까지 읽은 뒤 가장 먼저 남는 감정은 무엇인가요
- 추천하고 싶은 작품인가요
- 계속 읽기 버거웠던 구간이 있었다면 어디인가요

### type E

이름:

- 여성 주인공 입체성 민감 독자

보고 싶은 것:

- 아린의 독립성
- 대상화 위험
- 창작 주체성

꼭 물을 질문:

- 아린은 언제부터 입체적 인물로 느껴졌나요
- 아린이 기능적 인물이나 치유 오브젝트처럼 읽힌 순간이 있었나요
- 아린의 음악, 일기, 선택은 독립 축으로 읽혔나요

## 3. universal questions

- 초반 1-3화에서 버거웠던 지점은 어디였나요
- 중반부에서 가장 반복적으로 느껴진 구간은 어디였나요
- 결말은 `심심하다`와 `정확하다` 중 어느 쪽에 더 가까웠나요
- 삭제/축약이 필요해 보이는 회차가 있다면 어디인가요
- 가장 오래 남는 회차 3개는 무엇인가요

## 4. ignore list

아래 피드백은 원칙적으로 반영하지 않는다.

- 더 자극적으로 바꾸라는 요구
- 더 빨리 사귀게 하라는 요구
- 결말을 더 뜨겁게 바꾸라는 요구
- 아린의 경계와 selfhood를 약화시키는 요구
- 작품 포지셔닝을 무시한 플랫폼 도파민 문법 강요
- 위치와 이유가 없는 `그냥 답답했다`형 총평

## 5. pass condition

10점 업그레이드 검증 통과 조건:

- 서로 다른 reader type 3종 이상이 공통으로
  - 후반 보상감
  - 관계 윤리의 설득력
  - 문장/분위기 몰입감
  을 인정한다
- `초반 진입`, `중반 반복`, `결말 온도`에서 같은 병목이 2명 이상에게 반복 확인될 때만 수정 후보로 올린다

## 6. orchestra note

- first wave 기본 mix는 `type A / type B / type D`
- `type C`는 상세 수정 의사가 확인된 뒤 `package C`에서 더 유효하다
