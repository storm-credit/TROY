wave2는 사적인 밀도, 기억 수정, 묻기의 윤리처럼 더 조용한 감정 축을 대표하는 묶음이다.

1. `TROY_WAVE2_PRIVATE_CLOSENESS_E012`
2. `TROY_WAVE2_DIARY_SEED_E050`

4. `TROY_WAVE2_ASKING_ETHICS_E094`
5. `TROY_WAVE2_MATURE_SILENCE_E119`
