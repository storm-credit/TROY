# TROY Wave4 Packet Export

이 폴더는 `wave4 packet family` 5종의 export-facing copy를 담는다.
wave4는 큰 사건보다 안쪽 작동 규칙, 오해, 자기 수정의 순간을 대표하는 묶음이다.

현재 패킷:

1. `TROY_WAVE4_CONTACT_DANGER_E018`
2. `TROY_WAVE4_QUESTION_SPACE_E028`
3. `TROY_WAVE4_READING_DISCOMFORT_E047`
4. `TROY_WAVE4_PRELOSS_INSTABILITY_E064`
5. `TROY_WAVE4_SELF_STORY_ERROR_E097`

의미:

- `E018`
  - contact as danger seed
- `E028`
  - question-space before confession
- `E047`
  - reading discomfort and over-interpretation
- `E064`
  - pre-loss instability and misread
- `E097`
  - self-story error correction
