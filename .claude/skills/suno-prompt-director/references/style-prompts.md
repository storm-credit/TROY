# Style Prompt Patterns

## Target

Write compact, original, production-directed Style prompts. Use enough detail for arrangement and vocal intent, but avoid bloated adjectives and duplicated mood words.

## 7-Block Pattern

Use this order for full paste packs:

1. **Genre identity**: original Korean K-pop, K-R&B, K-indie, ballad, dance pop, synth pop, hip-hop, or hybrid.
2. **Sound palette**: tempo feel, drums, bass, harmony instrument, texture, room/space, production cleanliness.
3. **Vocal direction**: gender/role, tone, range, delivery, harmony density. Avoid real artist names.
4. **Emotional thesis**: one sentence describing the emotional job.
5. **Section arc**: verse restraint, pre-chorus tension, chorus release, bridge contrast, final chorus lift.
6. **Main imagery**: 3-5 concrete images that anchor the lyric mood.
7. **Negative constraints and technical target**: what to avoid, BPM/key if useful, language clarity, target length as a goal.

## Compact Template

```text
Original Korean [genre] song with [emotional job]. [Tempo/BPM] [key if useful].
Palette: [drums], [bass], [harmony instrument], [lead texture], [space/FX].
Vocals: [role/tone/delivery], clear Korean diction, [harmony plan].
Arc: verses [feel], pre-chorus [function], chorus [hook behavior], bridge [contrast], final chorus [lift].
Imagery: [image 1], [image 2], [image 3].
Avoid [artist imitation], [tone risk], [arrangement risk], [lyric risk].
```

## Length Policy

- Default: concise enough to paste cleanly and remain focused.
- House target: about 600-850 characters for high-control Korean pop.
- If the user needs broad experimentation, loosen detail instead of adding more adjectives.
- If the user needs strict replication across runs, keep the same Style and vary only one planned parameter.

## Compression Rules

- Remove duplicate adjectives.
- Convert long clauses into short noun phrases.
- Keep only one phrase for time/place/mood when several say the same thing.
- Prefer concrete instrumentation over generic intensity words.
- Move lyric concepts to Lyrics; keep Style about sound, form, and delivery.

## Safety Wording

Do not write:

```text
Like [artist], in the style of [song], with [celebrity] vocals.
```

Write:

```text
Contemporary Korean R&B-pop with intimate male tenor delivery, dry close vocal, soft falsetto edges, syncopated topline, muted Rhodes, clean 808 bass, and understated late-night tension.
```
