# Arrangement And Run Strategy

## BPM And Key Guide

Use this as a starting point, not a promise:

| Genre | BPM | Keys |
|---|---:|---|
| K-ballad | 60-75 | C minor, E-flat minor, D major |
| Sad K-R&B pop | 75-90 | D minor, A minor, F-sharp minor |
| Warm K-R&B pop | 85-100 | G major, C major |
| K-indie pop | 85-110 | D major, A major, E minor |
| K-dance pop | 110-130 | F major, C major |
| K-hip-hop | 80-100 | D minor, F minor |
| K-synth pop | 95-115 | B minor, E minor |

## Lyric Volume Planning

Duration is not guaranteed by line count, but lyric volume affects structure.

| Vocal lines | Planning use |
|---:|---|
| 20-25 | Short sketch or 2-minute idea |
| 30-40 | Compact pop draft |
| 45-55 | Full pop/MV target |
| 55-65 | Extended narrative or section-rich arrangement |

For 3:30+ targets, plan:

- Verse 1 and Verse 2 with 6-8 singable lines each.
- Pre-chorus twice if the hook needs lift.
- Chorus repeated at least twice.
- Bridge with a clear contrast.
- Final chorus with either harmony, ad-libs, or lyric variation.
- Outro that can fade cleanly.

## Run A/B Strategy

Change only one or two variables.

Good variables:

- Tempo: BPM plus or minus 5-10.
- Vocal plan: solo vs duet.
- Arrangement: electronic vs acoustic.
- Section emphasis: bigger first chorus vs bigger bridge.
- Key or mode: D minor vs E minor, major warmth vs minor tension.

**Locking rule (변수 폭증 방지):**
- If Run B changes vocal config (solo→duet, mono→harmony, female→male lead), **BPM and key must be locked to Run A's values**. Vocal change alone counts as 1 variable; adding tempo/key shift makes comparison meaningless.
- If Run B changes BPM/key, vocal config must be locked.
- Section structure (bridge length, instrumental break) counts as a variable — don't stack with vocal/tempo changes.

Bad variables:

- Changing genre, tempo, vocal, lyrics, and hook all at once.
- Rewriting the chorus so comparison becomes impossible.
- Using different emotional theses for the same song test.
- Changing vocal config AND tempo AND section structure in the same Run B (KR-03 borderline case — Run B added duet + Bridge 2 + 4-bar instrumental change, which is 3 variables; should have picked 2).

## Extend Planning

Use Extend when:

- The first generation has a strong core but weak ending.
- The song needs a post-bridge final chorus.
- The user wants a longer MV arrangement.
- The first half works and only the second half needs expansion.

Prepare Extend material as:

```text
[Part 2 - Extend from bridge]
[Bridge]
...
[Final Chorus]
...
[Outro]
...
```

Keep new lyrics compatible with the existing hook and do not introduce a new song premise.
