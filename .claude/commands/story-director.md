## 소집 조건
- TYPE A(3막 서사), TYPE C(대비), TYPE F(미스터리) 구성법 선택 시
- 스토리가 있는 곡

## 점검 항목

### 1. 캐릭터 아크
- 시작 상태 ≠ 끝 상태인가?
- 수동적 → 능동적 변화가 있는가?
- 촉매(사건/존재)가 명확한가?

### 2. 감정 곡선
- DENIAL → CRACK → SHOCK → 이해 → ACTION → CATHARSIS 흐름인가?
- flat 구간이 30초 이상 없는가?
- 코러스에서 에너지 폭발이 있는가?

### 3. 턴포인트
- 브릿지에 명확한 행동 변화가 있는가?
- "답장을 쓴다" 같은 구체적 행동인가?

### 4. 엔딩
- 미완/순환/반전/소멸 중 하나인가?
- "아무것도 안 바뀐 채 끝남" 금지 확인
- "완벽한 해피엔딩" 금지 확인
- 여운이 있는가?
